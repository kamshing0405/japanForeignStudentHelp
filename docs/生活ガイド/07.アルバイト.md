留学生が日本でアルバイトをするには、「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">守らなければならない法律のルール</span>」があります。これを知らないと、ビザの更新ができなくなったり、帰国させられたりすることもあるので注意が必要です。

#### <div class="custom-title">働くための「許可」をとる</div>

留学生のビザ（在留資格：留学）は、勉強するためのもので、働くためのものではありません。アルバイトをする前に必ず以下を<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">確認</span>してください（資格外活動許可欄）。

<br>

<div class='flex justify-center'>
<img src='/images/work28hour.png' class='rounded-2xl w-100'>
</div>

<br>

| 項目           | 内容                                                                                                                 |
| -------------- | -------------------------------------------------------------------------------------------------------------------- |
| 資格外活動許可 | パスポートや在留カードの`裏`に、スタンプまたは記載があるか確認してください。ない場合は、入国管理局で申請が必要です。 |
| 勉強が優先     | 学校を`休みすぎる`と、資格外活動許可があっても、ビザの更新が難しくなります。                                         |

<br>

#### <div class="custom-title">「28 時間の壁」を絶対に守る</div>

日本で留学生が働ける時間には、法律で<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">厳しい制限</span>があります。

<br>

| 項目           | 内容                                                                                              |
| -------------- | ------------------------------------------------------------------------------------------------- |
| 週 28 時間以内 | すべてのアルバイトを合計して、`1週間に28時間`までです。                                           |
| 掛け持ち       | `2つ`以上の仕事をしていても、合計で`28時間`以内です。                                             |
| 違反のリスク   | `1分`でも超えると法律違反（オーバーワーク）となり、将来、日本で就職できなくなるリスクがあります。 |
| 長期休暇中     | 学校が定めた長期休暇期間中に限り、`1日8時間まで（週40時間まで）`働くことができます。              |

<br>

#### <div class="custom-title">働いてはいけない場所（風俗営業の禁止）</div>

留学生は、以下の場所で働くことが法律で<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">禁止</span>されています。たとえ掃除や皿洗いであっても<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">ダメ</span>です。

<br>

`パチンコ店`、`ゲームセンター`、`スナック`、`キャバクラ`、`ホストクラブ`、`バー`（お酒をメインに提供し、接客する場合）、`麻雀店`など

<br>

これらの場所で働くと、資格外活動許可があっても違法となり、 即座に「`強制送還（帰国）`」の対象になるほど<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">厳しく処罰</span>されます。

<br>

#### <div class="custom-title">アルバイトの見つけ方</div>

多くの留学生が以下の方法で仕事を探しています。

<br>

| 探し方          | 内容                                                                                                 |
| --------------- | ---------------------------------------------------------------------------------------------------- |
| 求人アプリ      | `タウンワーク`、`バイトル`、`マイナビバイト`などが有名です。<br>「留学生歓迎」の条件で検索できます。 |
| 店舗の貼り紙    | `コンビニ`や`レストラン`の入り口にある「スタッフ募集」のポスターを見て、`直接電話`します。           |
| 学校の紹介      | 大学の`キャリアセンター`や`事務局`に、留学生向けの求人が来ていることがあります。                     |
| SNS・友人の紹介 | `先輩`や`友人`から紹介してもらう方法で、比較的安心です。                                             |

<br>

#### <div class="custom-title">採用されたら確認すること</div>

<br>

| 確認ポイント 　　　 | 内容                                                                     |
| ------------------- | ------------------------------------------------------------------------ |
| 時給                | 自分が住んでいる地域の「`最低賃金`」を下回っていないか確認します。       |
| 交通費              | `電車`や`バス`の`交通費`が支給されるかどうかを確認します。               |
| 給料日              | `毎月何日`に、どの方法で`支払`われるか（銀行振込が一般的）を確認します。 |

「緊急應對」是指針對<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">火災、自然災害、事故、急病</span>或重大商務障礙等威脅生命財產安全、且「刻不容緩」的事態，採取迅速且適當行動的過程。

#### <div class="custom-title">緊急求助電話（24 小時・免費）</div>

首先請務必記住這 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">3 個</span>號碼。

| 號碼  | 內容                     | 使用方法・備註                                   |
| :---- | :----------------------- | :----------------------------------------------- |
| 110   | 警察（刑事案件・事故）   | 遭遇`犯罪`、`遭小偷`、發生`交通事故`等。         |
| 119   | 消防・急救（火災・急病） | 接通後會詢問是`火災 (Kaji)`還是`救急 (Kyukyu)`。 |
| #7119 | 急救諮詢中心             | 猶豫`是否該叫救護車`時可以撥打諮詢。             |

#### <div class="custom-title">災害（地震・颱風）準備</div>

日本是一個<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地震與颱風</span>頻繁的國家。請事先做好以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">3 點</span>準備。

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">確認避難場所</p>
</div>

<br>

<img src='../../../images/hinansyo.jpg' class='rounded-2xl w-136'>

請事先透過防災地圖 (Hazard Map) 確認住家或學校附近的「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">避難所</span>」。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">防災避難包</p>
</div>

<br>

<img src='../../../images/bag.avif' class='rounded-2xl w-136'>

請在後背包內準備好`飲用水`、`乾糧`（3 天份）、`行動電源`、`在留卡影本`、`現金`、`常備藥`等物品。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">資訊收集 APP</p>
</div>

除了準備防災物資，下載防災 APP 以便隨時<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">收集最新資訊</span>也是非常重要的。

| 服務名稱                                                                  | 特徵                                           |
| :------------------------------------------------------------------------ | :--------------------------------------------- |
| [Safety Tips](https://www.jnto.go.jp/safety-tips/eng/app.html)            | 觀光廳監修，提供多語系的地震速報與避難建議通知 |
| [NHK WORLD-JAPAN](https://www3.nhk.or.jp/nhkworld/en/multilingual_links/) | 提供英文等各國語言的災害新聞轉播               |
| [Yahoo! 防災速報](https://emg.yahoo.co.jp)                                | 可透過推播接收地震、颱風、豪雨等災害資訊       |
| [JMA（氣象廳）APP](https://www.data.jma.go.jp/suishin/info/keitai.html)   | 日本氣象廳官方提供，包含地震、海嘯、颱風資訊   |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">災害用留言專線（171）</p>
</div>

當發生大規模災害導致電話難以接通時，可用來向<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">家人或友人</span>告知平安。

- 錄音：撥打 `171` → 按 `1` → 輸入`自己的電話號碼` → 錄製留言。
- 播放：撥打 `171` → 按 `2` → 輸入`對方的電話號碼` → 聽取留言。

<br>

#### <div class="custom-title">夜間・假日的急病處理</div>

若在夜間或假日突發急病，首先請務必記住這 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 個</span>諮詢號碼。

| 號碼  | 諮詢窗口         | 內容・角色                                     |
| :---- | :--------------- | :--------------------------------------------- |
| #7119 | 急救安心中心     | 由護理師針對是否需要就醫及應對方法提供建議。   |
| #8000 | 兒童急救電話諮詢 | 由小兒科醫師或護理師針對兒童突發疾病提供諮詢。 |

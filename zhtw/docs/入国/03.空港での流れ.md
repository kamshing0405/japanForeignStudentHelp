第一次來到日本，感到緊張是正常的。但請放心，日本對於「第一次到訪的外國人」是非常友善的國度。

以下將針對持`留學簽證`入境的學生，詳細解說抵達日本主要機場（成田、羽田、關西、中部等）後的專屬入境步驟。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">機場入境流程</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">視擁擠情況約需 1〜2 小時</span>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">検疫 (Quarantine)</p>
</div>

- 下飛機後，請依照動線指示前進。

- 目前主要以健康狀況詢問及紅外線熱像儀測量體溫為主。

- 若感到`身體不適`，請在此處向`工作人員諮詢`。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入國審查 (Immigration)</p>
</div>

- 請前往「`中長期在留者（留學）`」專用櫃檯。

- 須出示之文件：`護照`、`簽證`、Visit Japan Web 的 `QR Code` 或 `紙本入境卡`。

- 審查手續：進行 `指紋` 採集與 `面部照片` 拍攝。

- 領取在留卡：持有留學簽證者，會當場獲發「`在留卡`」。請務必現場確認卡面上的`姓名`與`出生年月日`是否正確。

<br>

::: warning 申請「資格外活動許可」

計畫工讀（打工）的人，請在此審查時提交申請書。工作人員會當場在在留卡背面蓋章（若日後再申請會非常耗時，在此處辦理是最理想的）。

:::

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">領取行李 (Baggage Claim)</p>
</div>

請確認螢幕顯示的`航班編號`，前往對應的行李轉盤領取您的`託運行李`。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">動植物檢疫(Optional)</p>
</div>

僅限有攜帶肉製品、水果、蔬菜等物品的人員需要前往。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">海關申報 (Customs)</p>
</div>

請使用於[Visit Japan Web](https://services.digital.go.jp/visit-japan-web/) 註冊完成的「海關申報」QR Code。

電子申報閘門：讓機器讀取 QR Code 與護照後，僅需透過臉部辨識即可順暢通過。

即使沒有任何需要申報的物品，此手續也是所有人皆必須完成的。

<br>

    <p class="text-[#00873c] font-bold text-lg pl-0.5">入境後的交通移動</p>
</div>

<br>

| 交通方式 | 特徵               | 優點                   | 缺點               |
| -------- | ------------------ | ---------------------- | ------------------ |
| 電車     | 班次多、運行準時   | 快速／便宜／有英語導引 | 轉乘可能較令人不安 |
| 機場巴士 | 直達主要車站及飯店 | 無須轉乘／行李輕鬆     | 視路況可能較費時   |
| 計程車   | Door to Door       | 不怕迷路／深夜 OK      | 費用昂貴           |
| 學校接送 | 需事先預約         | 最安心／完全不怕迷路   | 僅限部分學校提供   |

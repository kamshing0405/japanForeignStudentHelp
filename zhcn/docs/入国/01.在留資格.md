若要前往日本留学，必须持有由本国政府签发的护照，以及由日本政府（大使馆、领事馆）签发的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">签证（Visa）</span>。 留学生取得的在留资格通常为“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">留学</span>”。与旅游签证不同，申请留学签证必须先取得“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">在留资格认定证明书（COE）</span>”。

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">在留资格</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">总计约需 2 个月〜4 个月</span>

申请留学签证的一般流程如下：

| 流程步骤     | 内容                                                                       |
| ------------ | -------------------------------------------------------------------------- |
| 取得入学许可 | 从日本的教育机构收到`录取通知`（入学许可书）                               |
| 申请 COE     | 由学校`代理`向出入国在留管理局申请在留资格认定证明书（审查约需 1〜3 个月） |
| 核发 COE     | 审查通过后，`COE` 原件将寄送给申请人                                       |
| 申请签证     | 准备好 COE 等必要文件，至`驻外使领馆`（如：日本驻华大使馆/领事馆）申请签证 |
| 入境日本     | 取得签证后，需在 COE `有效期限`内入境日本                                  |

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>COE（在留资格认定证明书）的重要性</p>

| 项目     | 内容                                                                                        |
| -------- | ------------------------------------------------------------------------------------------- |
| 作用     | 证明申请人符合`学习计划`、`经济能力`等入境条件的文件。提交 COE 申请签证可大幅缩短审查时间。 |
| 有效期间 | 签证签发后，原则上必须在 `3 个月内`入境。逾期可能导致 COE 失效。                            |

<br>

#### <div class="custom-title">核发在留卡</div>

| 项目     | 内容                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 入境许可 | 在`机场`进行入境审查时，护照上会贴上“`上陆许可`”贴纸。                       |
| 在留卡   | 在新千岁、成田、羽田、中部、关西、广岛、福冈等主要机场，入境时会`当场核发`。 |
| 重要事项 | 在留卡是您在日本证明身份的`唯一公认证件`，法律规定必须随身携带。             |

<br>

#### <div class="custom-title">入境后必要的申报</div>

| 项目                  | 内容                                                                                           |
| --------------------- | ---------------------------------------------------------------------------------------------- |
| 住所登记              | 确定住处后 `14 日内`，须持在留卡前往居住地的市、区役所（区公所）办理地址登记。                 |
| 个人编号（My Number） | 完成地址登记后，日后将收到“`个人编号通知书`”。办理`银行开户`或工作的`税务手续`时均为必需文件。 |

<br>

#### <div class="custom-title">资格外活动许可（兼职・打工许可）</div>

| 项目     | 内容                                                                           |
| -------- | ------------------------------------------------------------------------------ |
| 原则     | 持有“`留学`”在留资格，`原则上禁止从事有酬劳的工作`。                           |
| 申请     | 若要进行`勤工俭学（兼职/打工）`，必须申请`资格外活动许可`。                    |
| 时间限制 | 每周 `28 小时`以内；`暑假`等长假期间，为`每日 8 小时以内`。                    |
| 要点     | 入境时在机场审查台提交申请书，`即可当场获得许可`，可省去日后前往入管局的麻烦。 |

<h1 style="text-align: center;">學校選擇</h1>

為了讓留學取得成功，事前的計畫制定非常重要。你是否已經為自己的留學計畫做好充分準備了呢？

首先最需要考慮的是<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">教育機關的類別選擇</span>。日本的高等教育機關根據「想學什麼」以及「未來想朝什麼樣的職涯發展」，其角色與特徵有很大的不同。

因此，在明確留學目的後，選擇適合自己的教育機關至關重要。

由於日本的入學考試制度與簽證（在留資格）申請手續非常耗時，建議從<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">一年前</span>就開始行動，這樣可以增加選擇空間並防止失敗。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">大學與研究所 (大學院)</div>

#### <div class="custom-title">大學（學部）・短期大學</div>

日本的大學不僅是習得高度<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">專業知識</span>的地方，更是培養自主研究能力與廣泛<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">通識教養</span>的高等教育核心機構。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基本概要與修業年限</p>
</div>

| 項目               | 內容                                                            |
| ------------------ | --------------------------------------------------------------- |
| 標準修業年限       | 大多數學部、學科為 `4 年制`                                     |
| 修業年限較長的學部 | 醫學、齒學、獸醫學，以及部分藥學部（藥劑師養成課程）為 `6 年制` |
| 學位               | 畢業後將授予`學士`學位                                          |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入學資格</p>
</div>

要進入日本的大學就讀，必須符合以下任一項<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">學歷條件</span>。

| 項目         | 內容                                                                                                 |
| ------------ | ---------------------------------------------------------------------------------------------------- |
| 標準學歷     | 在海外完成 `12 年正規學校教育`（高中畢業或具同等學歷）                                               |
| 學力認定考試 | 通過被認定與完成 `12 年教育同等學力的考試`（如大學預科課程等）                                       |
| 國際資格     | 擁有國際學士學位（IB）、德國大學入學資格（Abitur）、法國大學入學資格（Baccalauréat）等國際`認可資格` |
| 補充說明     | 若本國學校教育年限`未滿 12 年`，原則上需修滿指定的「`準備教育課程`」                                 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学選抜・試験方法</p>
</div>

日本的大學提供多樣化的選拔方式，且針對<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">留學生的制度</span>也十分完善。

| 項目                     | 內容                                                                             |
| ------------------------ | -------------------------------------------------------------------------------- |
| 日本留學試驗（EJU）      | 許多大學要求留學生提交 `EJU 成績`，用以評估日語能力及基礎學力                    |
| 日語能力試驗（JLPT）     | 視大學或學部而定，有時可以提交 `N1 或 N2` 合格證書來代替 EJU 的日語科目          |
| 全英文學位課程（SGU 等） | 提交 `TOEFL・IELTS` 分數、`SAT・ACT`・高考成績及志願理由書等，無需在日本參加筆試 |
| 校內選考                 | 參加各大學自行辦理的`筆試`（小論文・專業科目）或`面試`                           |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">畢業要求與學分制度</p>
</div>

| 項目           | 內容                                       |
| -------------- | ------------------------------------------ |
| 學分制         | 學生在學期間必須修滿大學規定的`必要學分數` |
| 一般學部       | 4 年制大學通常需要修滿 `124 學分`以上      |
| 醫學・獸醫學系 | 6 年制課程通常要求修滿 `182 學分`以上      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">短期大學</p>
</div>

對於想要在短時間內習得<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">專業知識</span>與技能，並<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">希望盡快步入社會的人</span>來說，短期大學是一個有力的選擇。

| 項目           | 內容                                                                       |
| -------------- | -------------------------------------------------------------------------- |
| 修業年限與目的 | 一般為 `2 年制`（部分醫療領域為 3 年制），重視培養與職業直接相關的實踐能力 |
| 學位           | 畢業後將授予`短期大學士`學位                                               |
| 升學之路       | 畢業後除了就業，也可以透過「編入學考試」進入 `4 年制`大學的 `3 年級`就讀   |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">6</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">如何選擇大學</p>
</div>

| 項目         | 內容                                                                       |
| ------------ | -------------------------------------------------------------------------- |
| 觀察大學性質 | 國公立大學學費相對便宜且研究環境充實；私立大學則學部與學科的`選擇非常豐富` |
| 考慮授課語言 | 衡量要用日語學習，或是挑戰`全英文授課課程`                                 |
| 確認數據資料 | 可利用 `JPSS` 搜尋網站確認過去的`合格基準`                                 |

#### <div class="custom-title">研究所 (大學院)</div>

日本的研究所是進行<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">高度學術研究</span>並培養高專業人才的教育機關。根據研究目的與取得學位的不同，分為碩士課程、博士課程及專業職學位課程。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">課程內容與修業年限</p>
</div>

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>碩士課程</p>

| 項目     | 內容                                             |
| -------- | ------------------------------------------------ |
| 修業年限 | 通常為 `2 年`                                    |
| 學位     | 修滿學分並通過碩士論文審查後，授予「`碩士`」學位 |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>博士課程</p>

| 項目     | 內容                                               |
| -------- | -------------------------------------------------- |
| 修業年限 | 通常為 `5 年`                                      |
| 構成     | 一般分為前期課程（`2 年`）與後期課程（`3 年`）     |
| 補充     | 醫學・齒學・獸醫學等 6 年制學部畢業後通常為 `4 年` |
| 學位     | 修滿課程後授予「`博士`」學位                       |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>專業職學位課程</p>

| 項目     | 內容                                                       |
| -------- | ---------------------------------------------------------- |
| 目的     | 培養如法科大學院、教職大學院、MBA 等具備高`實務能力的人才` |
| 修業年限 | 原則上為 `2 年`（依領域不同可能有所變動）                  |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入學資格</p>
</div>

要報考日本的研究所，通常需要符合以下條件：

| 對象課程             | 出願要件                                                                   |
| -------------------- | -------------------------------------------------------------------------- |
| 碩士・專業職學位課程 | 在外國完成 `16 年學校教育`並取得學士學位                                   |
| 博士課程（後期）     | 取得`碩士`學位，或被認定具備同等學力                                       |
| 特殊情況             | 若學制為 `15 年`等情況，可能需要透過個別入學資格審查，或先以研究生身份在校 |
| 語言能力證明         | 提交 `JLPT`、`TOEFL` 或 `IELTS` 等語言考試成績                             |
| 推薦信               | 提交 1 ～ 2 封由指導教授或院長等撰寫的`推薦信`（依大學、研究科而異）       |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">關於「研究生」制度</p>
</div>

日本所稱的「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">研究生</span>」與中文裡的「碩士班學生」意義不同。日本大學或研究所的「研究生」是指不以取得學位（學士或碩士等）為目的的「非正規生」身份。

| 項目     | 內容                                                         |
| -------- | ------------------------------------------------------------ |
| 定義     | 不取得學位、`不屬於正規課程`的學生制度                       |
| 目的     | 作為進入碩、博士課程前的準備期，主目的通常是應對`入學考試`等 |
| 選考方法 | 多數情況僅透過`書面審查`或與指導教授`面談`即可獲得許可       |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">修了要求與取得學分</p>
</div>

| 課程區分       | 修了條件                                                       |
| -------------- | -------------------------------------------------------------- |
| 碩士課程       | 需修滿 `30 學分`以上，完成碩士論文並`通過最終審查與考試`       |
| 專業職學位課程 | 通常需修滿 `30 學分`以上，依領域不同可能要求提交實踐性課題報告 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">授業言語</p>
</div>

| 授業形態             | 內容                                                       |
| -------------------- | ---------------------------------------------------------- |
| `日語`授課           | 許多研究科要求具備 N1 等級的日語能力，且需具備學術寫作能力 |
| `英語`授課（SGU 等） | 在理工、經濟、國際關係等領域設有全英文學位學程             |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">專門學校與專修學校</div>

#### <div class="custom-title">高等專門學校 (高專)</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">對象為國中畢業生</span>

高等專門學校（簡稱「高專」）是日本<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">獨特</span>的高等教育機關。其目的是從<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">早期階段</span>開始進行專業教育，培養出能應對<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">尖端技術</span>的實踐型技術人才。
<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基本概要與定位</p>
</div>

| 項目     | 內容                                                  |
| -------- | ----------------------------------------------------- |
| 教育目標 | 以`工業領域`為中心，重視`實驗・實習・實作`的專業教育  |
| 教育特色 | 採行 `5 年`一貫制教育課程，兼顧`專業知識`與`通識`教養 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">修業年限與留學生升學路徑</p>
</div>

| 項目         | 內容                                                       |
| ------------ | ---------------------------------------------------------- |
| 標準修業年限 | 對象為國中畢業生，修業年限通常為 `5 年`（商船系為 5.5 年） |
| 留學生編入學 | 留學生通常以第 `3 學年編入`方式入學，修業年限通常為 `3 年` |
| 專攻科       | 本科畢業後，可進入為期 `2 年`的專攻科進行高度應用研究      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">授予稱號與進路</p>
</div>

| 項目     | 內容                                                                |
| -------- | ------------------------------------------------------------------- |
| 準學士   | 完成 `5 年`課程（或留學生 3 年課程）後，授予準學士稱號              |
| 學士     | 再完成 `2 年`專攻科課程並通過審查，可取得學士學位                   |
| 主要進路 | 就業評價極高且就業率近 100%，亦可透過編入學進入 ~4 年制大學 3 年級~ |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入學資格（編入學）</p>
</div>

| 項目     | 內容                                                        |
| -------- | ----------------------------------------------------------- |
| 學歷條件 | 在外國完成相當於 ~11 年學校教育~的課程（針對第 3 學年編入） |
| 年齡條件 | 原則上報名時需`滿 17 歲`以上                                |
| 日語能力 | 為了修習專業科目，需要具備充分的`日語能力`                  |

<br>

#### <div class="custom-title">專修學校 (專門學校)</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">對象為高中畢業生</span>

專修學校在日本高等教育制度中具有<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">顯著特徵</span>。在專門學校可以學到與<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">職業及實際生活</span>直接掛鉤的知識與技能，是培養「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">職人 (專家)</span>」及高技術人才的場所。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>

根據修業年限與教育內容，結業後取得的稱號會有所不同。

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>専門士</p>

| 項目     | 內容                                 |
| -------- | ------------------------------------ |
| 修業年限 | 通常為 `2～3` 年                     |
| 條件     | 總授課時數達 `1,700 小時以上`        |
| 進路     | 取得編入 `4 年制`大學 `3 年級`的資格 |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>高度専門士</p>

| 項目     | 內容                                           |
| -------- | ---------------------------------------------- |

| 條件     | 總授課時數達 `3,400 小時以上`                  |
| 進路     | 待遇等同`大學畢業生`，可報考研究所（碩士課程） |

<div class="flex items-center text-center gap-2 h-6 mt-6">

報考專門學校通常需完成正規教育的 12 年課程。此外，還需符合以下任一項<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日語能力</span>要求：

| 日語能力要求   | 需符合以下條件中的其中之一                       |
| -------------- | ------------------------------------------------ |
| `日語學校經歷` | 在法務大臣告示的日語教育機關學習 `6 個月以上`    |
| `JLPT`         | 通過 `N1` 或 `N2`                                |
| `日本就學經歷` | 在學校教育法第 1 條規定的學校接受 `1 年以上`教育 |
| `EJU`          | 日本留學試驗日語科目取得 `200` 點以上            |
| `BJT`          | 商務日語能力考試取得 `400` 點以上                |

#### <div class="custom-title">日語教育機關</div>

對於母語<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">非日語</span>的留學生，在進入日本的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">大學或專門學校</span>前，先在日語教育機關提升日語能力是很常見的做法。日本的日語教育機關主要分為以下 3 類：
<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">日本語学校</p>
</div>

| 項目         | 內容                                           |
| ------------ | ---------------------------------------------- |
| 定位         | 由學校法人或企業等設立，`最常見`的日語教育機關 |
| 課程特徵     | 提供`升學`、`商務`、`文化體驗`等多樣化課程     |
| 學生在留資格 | 在日本以「`留學`」在留資格就讀                 |
| 優點         | 升學輔導`體制完善`，如 EJU 對策與面試指導      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">大學留學生別科</p>
</div>

| 項目     | 內容                                                         |
| -------- | ------------------------------------------------------------ |
| 定位     | 附設於私立大學或短期大學內的`日語預備教育課程`               |
| 課程特徵 | 以進入該大學或研究所為目標，教育內容以`日語及基礎科目`為中心 |
| 優點     | 可使用`大學設施`，有時設有推薦入學制度                       |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">準備教育課程</p>
</div>

| 項目     | 內容                                             |
| -------- | ------------------------------------------------ |
| 對象者   | 本國學校教育年限`未滿 12 年`的學生               |
| 主要角色 | 文部科學省`指定課程`，結業後可取得`大學報考資格` |
| 授課內容 | 除了`日語`外，還學習大學升學所需的`基礎科目`     |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">學校選擇查檢重點</p>
</div>

建議在學校官網等處確認以下幾點：

| 項目         | 內容                                 |
| ------------ | ------------------------------------ |
| 是否為告示校 | 確認是否為`法務省公告`的日語教育機關 |
| 課程內容     | 是否設有`符合自己目標`的升學課程     |
| 所在地       | 考慮城市或地方的`生活費差異`         |
| 畢業生進路   | 確認留學生的`升學與就業實績`         |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入學資格</p>

| 項目     | 內容                                                           |
| -------- | -------------------------------------------------------------- |
| 學歷     | 原則上需完成 `12 年正規學校教育`                               |
| 日語能力 | 視學校而定，通常要求 `JLPT N5`程度或具備一定時數的日語學習經歷 |

#### <div class="custom-title">短期課程</div>

「短期留學」是指不以取得學位為目的，以「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">非正規生</span>」身份進行大約<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">數週至一個學年</span>的留學。

學程形式多樣，有些設有專為短期留學生開設的特別班級，有些則是修習一般學部或學科內的課程。

短期留學依其特徵可分為以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">5 類</span>：

| 區分                   | 概要                                                     | 學費支付對象                         |
| ---------------------- | -------------------------------------------------------- | ------------------------------------ |
| 交換留學               | 在簽署學生交流協定的大學間留學的制度                     | 原所屬學校（留學大學的學費通常免除） |
| 「交換留學」以外的學程 | 交流協定外的留學制度，也有個別接收學生的學程             | 留學大學                             |
| 研究生                 | 對象為大學畢業生，以研究生身份在校進行一定期間研究的制度 | 留學大學                             |
| 聽講生                 | 以「聽講」課程科目為目的的制度                           | 留學大學                             |
| 科目等履修生           | 修習課程科目並以取得學分為目的的制度                     | 留學大學                             |

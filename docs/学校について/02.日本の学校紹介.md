<h1 style="text-align: center;">学校選択</h1>

留学を成功させるためには、事前の計画作りが非常に重要です。あなたは、自分の留学計画をしっかり立てていますか。

まず最初に考えるべきなのが、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">教育機関の種別選択</span>です。日本の高等教育機関は、「何を学ぶのか」「将来どのような進路に進みたいのか」によって、その役割や特徴が大きく異なります。

そのため、自分の留学目的を明確にしたうえで、それに合った適切な教育機関を選択することが重要です。

日本の入試制度やビザ（在留資格）の申請手続きは非常に時間がかかるため、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">1 年前</span>から動くことで選択肢が広がり、失敗を防ぐことができます。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">大学と大学院</div>

#### <div class="custom-title">大学（学部）・短期大学</div>

日本の大学は、高度な<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">専門知識</span>の習得だけでなく、自主的な研究能力や幅広い<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">教養</span>を養う高等教育の中心的な機関です。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基本的な概要と修業年限</p>
</div>

| 項目               | 内容                                                                |
| ------------------ | ------------------------------------------------------------------- |
| 標準修業年限       | ほとんどの学部・学科は `4 年制`                                     |
| 修業年限が長い学部 | 医学・歯学・獣医学、および一部の薬学部（薬剤師養成課程）は `6 年制` |
| 学位               | 卒業すると `学士` の学位が授与される                                |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学資格</p>
</div>

日本の大学に入学するためには、以下のいずれかの<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">学歴条件</span>を満たす必要があります。

| 項目         | 内容                                                                                                     |
| ------------ | -------------------------------------------------------------------------------------------------------- |
| 標準的な学歴 | 海外において `12年間の正規学校教育` を修了していること（高等学校卒業、または同等の学歴）                 |
| 学力認定試験 | `12年教育修了と同等` と認められる学力認定試験に合格していること（大学予備課程など）                      |
| 国際資格     | `国際バカロレア（IB）`、`アビトゥア（Abitur）`、`バカロレア（Baccalauréat）`など、国際的に認められた資格 |
| 補足説明     | 自国の学校教育年限が `12年に満たない場合`、原則として指定された「準備教育課程」を修了する必要がある      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学選抜・試験方法</p>
</div>

日本の大学では、多様な入学選抜方法が用意されており、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">留学生向けの制度</span>も整えられています。

| 項目                           | 内容                                                                                                                 |
| ------------------------------ | -------------------------------------------------------------------------------------------------------------------- |
| 日本留学試験<br>（EJU）        | 多くの大学で留学生に `EJU の成績提出` が求められ、日本語能力や基礎学力の評価に用いられる                             |
| 日本語能力試験<br>（JLPT）     | 大学・学部によっては、EJU 日本語科目の代わりに `N1 または N2 の合格証明書` を認める場合がある                        |
| 英語のみで学べる課程（SGU 等） | `TOEFL・IELTS` のスコア、`SAT・ACT・大学入学共通試験（高考）` の成績、志望理由書などを提出し、日本での筆記試験は不要 |
| 学内選考                       | 各大学が実施する `筆記試験（小論文・専門科目）` や `面接` が行われる                                                 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">卒業要件と単位制度</p>
</div>

| 項目           | 内容                                                                   |
| -------------- | ---------------------------------------------------------------------- |
| 単位制         | 学生は在学期間中に、大学が定める `必要単位数` を修得しなければならない |
| 一般的な学部   | `4 年制`大学では、通常 `124単位以上` の修得が必要                      |
| 医学・獣医学系 | `6 年制`課程では、通常 `182単位以上` の修得が求められる                |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">短期大学</p>
</div>

短期間で<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">専門的な知識</span>や技能を身につけ、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">早く社会に出たい人</span>にとって、短期大学は有力な選択肢です。

| 項目           | 内容                                                                                              |
| -------------- | ------------------------------------------------------------------------------------------------- |
| 修業年限と目的 | 一般的に `2年制`（一部の医療系分野では 3 年制）で、職業に直結する実践的な能力の育成を重視している |
| 学位           | 卒業後は `短期大学士` の学位が授与される                                                          |
| 進学の道       | 卒業後は就職のほか、「編入学試験」を受けて `4 年制大学`の `3年次` に編入することも可能            |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">6</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">大学の選び方</p>
</div>

| 項目             | 内容                                                                                      |
| ---------------- | ----------------------------------------------------------------------------------------- |
| 大学の性格を見る | 国公立大学は学費が比較的安く研究環境が充実しており、私立大学は `学部・学科の選択肢が豊富` |
| 授業言語を考える | 日本語で学ぶか、`英語のみで学べる課程` に挑戦するかを検討する                             |
| データを確認する | `JPSS 検索サイト`を利用し、`過去の合格基準や倍率` を確認できる                            |

#### <div class="custom-title">大学院</div>

日本の大学院は、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">高度な学術研究</span>を行い、専門性の高い人材を育成する教育機関です。研究目的や取得する学位の違いによって、修士課程・博士課程・専門職学位課程に分けられます。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">課程内容と修業年限</p>
</div>

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>修士課程</p>

| 項目     | 内容                                                           |
| -------- | -------------------------------------------------------------- |
| 修業年限 | 通常 `2年`                                                     |
| 学位     | 修了し、修士論文の審査に合格すると「`修士`」の学位が授与される |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>博士課程</p>

| 項目     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 修業年限 | 通常 `5年`                                                   |
| 構成     | 一般的に前期課程（`2 年`）と後期課程（`3 年`）に分かれている |
| 補足     | 医学・歯学・獣医学など 6 年制学部卒業後は通常 `4年`          |
| 学位     | 修了すると「`博士`」の学位が授与される                       |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>専門職学位課程</p>

| 項目     | 内容                                                              |
| -------- | ----------------------------------------------------------------- |
| 目的     | 法科大学院・教職大学院・MBA など、`実務能力の高い人材` を育成する |
| 修業年限 | 原則として `2年`（分野により異なる場合がある）                    |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学資格</p>
</div>

日本の大学院に出願するためには、一般的に次の条件を満たす必要があります。

| 対象課程                 | 出願要件                                                                               |
| ------------------------ | -------------------------------------------------------------------------------------- |
| 修士課程・専門職学位課程 | 外国で `16年間の学校教育` を修了し、`学士`の学位を取得していること                     |
| 博士課程（後期）         | `修士`の学位、または `同等の学力` があると認められること                               |
| 特別な場合               | 学制が `15 年`の場合などは、`個別の入学資格審査` や研究生在籍が必要な場合がある        |
| 語学能力証明書           | `JLPT`や`TOEFL`や`IELTS`などの語学試験成績を提出すること                               |
| 推薦書                   | `指導教員`や`学部長`等による推薦書を 1 ～ 2 通提出すること（大学・研究科により異なる） |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">「研究生」制度について</p>
</div>

日本でいう「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">研究生</span>」は、中国語でいう<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">修士課程</span>の学生とは異なります。日本の大学や大学院における「研究生」は、学位（学士や修士など）の取得を目的としない「非正規生」という身分を指します。

| 項目     | 内容                                                                  |
| -------- | --------------------------------------------------------------------- |
| 定義     | 学位を取得しない、`正規課程ではない` 学生制度                         |
| 目的     | 修士・博士課程進学前の準備として利用され、`入学試験対策` などが主目的 |
| 選考方法 | `書類審査や指導教員との面接` のみで許可される場合が多い               |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">修了要件と取得単位</p>
</div>

| 課程区分       | 修了要件                                                                          |
| -------------- | --------------------------------------------------------------------------------- |
| 修士課程       | `30単位以上` を修得し、`修士論文`を完成させ、最終審査・試験に`合格`する必要がある |
| 専門職学位課程 | 一般的に `30単位以上` が必要で、分野により実践的課題の提出が求められる            |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">授業言語</p>
</div>

| 授業形態                   | 内容                                                                   |
| -------------------------- | ---------------------------------------------------------------------- |
| `日本語`での授業           | 多くの研究科で `N1レベル` の日本語力が求められ、学術的文章作成力も必要 |
| `英語`による授業（SGU 等） | 理工系・経済・国際関係などで `英語のみで学べるプログラム` がある       |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">専門学校と専修学校</div>

#### <div class="custom-title">高等専門学校</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">中学校卒業者を対象とする</span>

高等専門学校（通称「高専」）は、日本<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">独自</span>の高等教育機関で、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">早い段階</span>から専門教育を行い、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">最先端の技術</span>に対応できる実践的な技術者を育成することを目的としています。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基本概要と位置づけ</p>
</div>

| 項目       | 内容                                                                 |
| ---------- | -------------------------------------------------------------------- |
| 教育目標   | `工業分野`を中心に、`実験・実習・実技` を重視した教育で専門性を養う  |
| 教育の特色 | `5年一貫制` の教育課程を採用し、`専門知識`と`教養`の両立を図っている |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">修業年限と留学生の進学ルート</p>
</div>

| 項目             | 内容                                                                |
| ---------------- | ------------------------------------------------------------------- |
| 標準的な修業年限 | `中学校卒業者`を対象とし、修業年限は通常 `5年`（商船系は `5.5` 年） |
| 留学生の編入学   | 多くの場合 `第3学年編入` として入学し、修業年限は通常 `3` 年        |
| 専攻科           | 本科修了後、さらに `2年間` の専攻科で高度な応用研究が可能           |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">授与される称号・学位と進路</p>
</div>

| 項目     | 内容                                                                                           |
| -------- | ---------------------------------------------------------------------------------------------- |
| 準学士   | `5 年`課程（または留学生の `3 年`課程）を修了すると、`準学士` の称号が与えられる               |
| 学士     | 専攻科を `2 年`間修了し、所定の審査に合格すれば `学士` の学位を取得できる                      |
| 主な進路 | `就職`では高い評価を受け就職率はほぼ 100％、または `編入学` により 4 年制大学 3 年次へ進学可能 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基入学資格（編入学）</p>
</div>

| 項目       | 内容                                                                      |
| ---------- | ------------------------------------------------------------------------- |
| 学歴条件   | 外国で `11年の学校教育` に相当する課程を修了していること（第 3 学年編入） |
| 年齢条件   | 出願時に原則として `17歳以上` であること                                  |
| 日本語能力 | 専門科目履修のため、`十分な日本語力` が求められる                         |

#### <div class="custom-title">専修学校</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">高校卒業者を対象とする</span>

専修学校は、日本の高等教育制度の中でも<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">大きな特徴</span>を持つ教育機関です。専門学校では、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">職業や実際の生活</span>に直結した知識・技能を学ぶことができ、「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">職人</span>」や高度な技術人材を育成する場となっています。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>

| 主な分野           | 主な専攻例                                      |
| ------------------ | ----------------------------------------------- |

| 医療分野           | `看護・理学療法・放射線技術` など               |
| 衛生分野           | `栄養・調理・美容・理容` など                   |

| 商業実務分野       | `ホテル・観光・会計・秘書` など                 |
| 服飾・家政分野     | `ファッションデザイン・インテリア` など         |
| 文化・教養分野     | `アニメ、ゲーム制作、イラスト、翻訳、音楽` など |

修業年限や教育内容により、修了後に取得できる称号が異なります。

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>専門士</p>

| 項目     | 内容                                                  |
| -------- | ----------------------------------------------------- |
| 修業年限 | 通常 `2～3年`                                         |
| 条件     | 総授業時間が `1,700時間以上`                          |
| 進路     | `4 年制`大学の `3 年次`への `編入学資格` を取得できる |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>高度専門士</p>

| 項目     | 内容                                                            |
| -------- | --------------------------------------------------------------- |
| 修業年限 | 通常 `4年`                                                      |
| 条件     | 総授業時間が `3,400時間以上`                                    |
| 進路     | 大学卒業者と同等扱いとなり、`大学院（修士課程）` への出願が可能 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学資格</p>

専門学校への出願には、一般的に正規の学校教育における 12 年課程を修了していることが必要です。
また、次の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日本語能力</span>要件のいずれかを満たす必要があります。

| 日本語能力要件    | 以下の条件のうち `いずれか1つ` を満たす必要がある              |
| ----------------- | -------------------------------------------------------------- |
| `日本語学校履修`  | 法務大臣告示の日本語教育機関で `6か月以上` 学習していること    |
| `JLPT`            | `N1またはN2` に合格していること                                |
| `日本の学校歴   ` | 学校教育法第 1 条に定める学校で `1年以上` 教育を受けていること |
| `EJU `            | 日本留学試験の日本語科目で `200点以上` を取得していること      |
| `BJT  `           | ビジネス日本語能力テストで `400点以上` を取得していること      |

#### <div class="custom-title">日本語教育機関</div>

日本語を<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">母語としない</span>留学生にとって、日本の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">大学や専門学校へ</span>進学する前に、日本語教育機関で日本語能力を高めることは一般的です。日本の日本語教育機関は、主に次の 3 種類に分けられます。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">日本語学校</p>
</div>

| 項目               | 内容                                                         |
| ------------------ | ------------------------------------------------------------ |
| 位置づけ           | 学校法人や企業等が設立する、`最も一般的な日本語教育機関`     |
| カリキュラムの特徴 | 進学・ビジネス・文化体験など `多様なコース` が用意されている |
| 学生の在留資格     | 日本では `「留学」` の在留資格で在学する                     |
| メリット           | EJU 対策や面接指導など、`進学指導体制が充実`                 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">大学留学生別科</p>
</div>

| 項目               | 内容                                                              |
| ------------------ | ----------------------------------------------------------------- |
| 位置づけ           | 私立大学・短期大学に併設された `日本語の予備教育課程`             |
| カリキュラムの特徴 | 当該大学・大学院進学を目指し、`日本語および基礎科目` を中心に教育 |
| メリット           | 大学施設の利用や、`推薦入学制度` が設けられている場合がある       |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">準備教育課程</p>
</div>

| 項目     | 内容                                                     |
| -------- | -------------------------------------------------------- |
| 対象者   | 母国の学校教育年限が `12年未満` の学生                   |
| 主な役割 | 修了すると `大学出願資格` を取得できる文部科学省指定課程 |
| 授業内容 | 日本語に加え、`大学進学に必要な基礎科目` を学習する      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">学校選びのチェックポイント</p>

| 項目           | 内容                                                   |
| -------------- | ------------------------------------------------------ |
| 告示校かどうか | 法務省により告示された `日本語教育機関` であるかを確認 |
| コース内容     | 自分の目標に合った `進学コース` が用意されているか     |
| 所在地         | 都市部か地方かなど、`生活費の違い` も考慮する          |
| 卒業生の進路   | 留学生の`進学・就職実績` を確認する                    |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">

#### <div class="custom-title">短期プログラム</div>

「短期留学」とは、学位取得を目的としない「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">非正規生</span>」として、おおむね<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">数週間から 1 学年間</span>の留学をすることです。

短期留学生のための特別クラスを設けている場合や、一般の学部・学科内の授業を履修する場合等、プログラムはさまざまです。

短期留学は、特徴から以下の <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">5 つ</span>に分けられます。

| 区分                         | 概要                                                               | 学費の支払先                           |
| ---------------------------- | ------------------------------------------------------------------ | -------------------------------------- |
| 交換留学                     | 学生交流協定を結んでいる大学に留学する制度                         | 派遣、在籍校（留学先大学の学費は免除） |
| 「交換留学」以外のプログラム | 学生交流協定以外の留学制度。学生を個別に受け入れるプログラムもある | 派遣、留学先校                         |
| 研究生                       | 大学を卒業した人を対象に、研究生として一定期間在籍し研究する制度   | 留学先校                               |
| 聴講生                       | 授業科目の「聴講」を目的とした制度                                 | 留学先校                               |
| 科目等履修生                 | 授業科目を履修し、単位取得を目的とした制度                         | 留学先校                               |

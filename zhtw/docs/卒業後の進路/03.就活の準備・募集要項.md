<h2 class=' text-center'> 求職準備</h2>

求職準備時期對大學生來說是至關重要的階段。一般而言，就活（就職活動）多從<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">大學三年級的春季</span>開始，但理想的情況是比這更早便展開準備。

在此介紹當您開始意識到需要準備求職時，應該著手進行的事項。

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">自我分析</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：理想是儘早開始</span>

「自我分析」的目的是客觀地理解自己的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">性格、價值觀、優勢與劣勢</span>。

透過這項工作，能<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">明確化適合自己的工作與職場環境</span>，並為思考未來的發展方向奠定基礎。

此外，在求職活動中，為了讓<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">應徵動機與自我推薦（Self PR）</span>的內容更具體且具說服力，自我分析是不可或缺的。透過以自身經驗作為根據來陳述，才能精準地向企業傳達<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自己的魅力</span>。

<br>

 <div class="custom-title">自我分析的方向</div>

| 方向                     | 內容範例・說明               | 分析方法                       |
| ------------------------ | ---------------------------- | ------------------------------ |
| `長處`                   | 自己的優勢、擅長的事         | 個人 SWOT 分析（Strength）     |
| `短處`                   | 自己的弱點、想改善的點       | 個人 SWOT 分析（Weakness）     |
| `喜歡的事物・討厭的事物` | 喜歡的事、不擅長的事         | 過去的「自我史」・MBTI         |
| `專業能力`               | 專業知識、技能、語言能力     | 過去的「自我史」・RIASEC       |
| `興趣 `                  | 感興趣的領域或主題           | MBTI                           |
| `未來夢想`               | 未來的目標或理想模樣         | Ikigai（生存意義）・未來自我史 |
| `價值觀`                 | 工作或生活中重視的思想與信念 | Ikigai・未來自我史             |
| `人生觀`                 | 對人生或工作方式的想法與方向 | Ikigai・未來自我史             |

<br>

::: tip 自我分析的方法

- [SWOT](https://offerbox.jp/columns/42821.html)

（為了個人成長，掌握自身的優勢、劣勢、機會與威脅的方法）

- [MBTI](https://www.mbti.or.jp/what/)

（基於卡爾·榮格心理學理論，用以理解性格的自我分析工具）

- [RIASEC](https://www.sifiji.org/riasec自己分析ツールを徹底解説就活生が必ずやる/)

（「人格（性格）與職業（環境）的 6 種類型」分類理論，用以了解自己的興趣與優勢，尋找適合職涯的工具）

- [過去的自我史](https://job.rikunabi.com/contents/howto/18838/)

（回顧至今為止的人生，像年表一樣依時間順序寫下的記錄）

- [生きがい（Ikigai）](https://www.keiro.org/jp/fact-sheet/ikigai-j)

（指生存的喜悅、動力、人生的意義或目的之日式概念）

- [未來自我史](https://woman.nikkei.com/atcl/doors/column/19/040300173/040300003/)

（想像將來想變成什麼樣子、過什麼樣的人生而寫下的自我史） 自分史です）

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">產業研究（業界研究）</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學三年級春季</span>

挑選心儀企業的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">第一步</span>就是「產業研究」。

產業研究是指理解各種產業的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">特徵與趨勢</span>，進而縮小志願產業範圍的過程。

首先應掌握產業有哪些種類的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">全貌</span>，並在<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">透過自我分析明確化興趣與關心</span>後，再調查哪種產業<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">適合自己</span>。

<br>

 <div class="custom-title">產業分類</div>
 
| 產業 | 主要內容 | 角色 |
|------|----------|------|
| `製造業` | 食品、汽車、電子設備、化學、醫藥、建設 | 製作物品的工作 |
| `商社` | 綜合商社、專業商社 | 在國內外串聯物品、服務、資訊，促成交易的工作 |
| `金融` | 銀行、證券、保險、信用卡、信用合作社 | 籌集、貸款、增加、流動及保護資金的工作 |
| `服務業` | 旅遊、飯店、餐飲、不動產、顧問、運輸 | 交易後不留下實體，提供效用或滿意度等無形服務的工作 |
| `流通・零售` | 百貨公司、超市、便利商店、EC（電商） | 將物品送達消費者手中的工作 |
| `IT・資訊・通訊` | 軟體、通訊營運商、Web 服務 | 支撐資訊基礎建設的工作 |
| `廣告・出版・媒體` | 電視、報紙、出版、廣告 | 傳達資訊的工作 |
| `官公廳・團體` | 公務員、非營利團體、基礎建設（電、瓦斯） | 支撐社會基盤的工作 |

<br>

::: tip 有助於產業研究的網站

1. [業種・職種・業態分析](https://www.theport.jp/portcareer/article/79266/#⑤広告・出版・マスコミ)
2. [產業研究的做法](https://www.onecareer.jp/articles/790)

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">企業研究</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學三年級春季</span>

針對企業的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">事業內容、策略、優勢、公司風氣</span>等進行調查，是為了明確化自己與企業的契合度及應徵理由的分析。

 <div class="custom-title">研究視角</div>

| 視角       | 項目           | 主要內容                                            |
| ---------- | -------------- | --------------------------------------------------- |
| `經營視角` | 經營・理念     | 企業理念、願景、使命、經營方針、社會角色            |
|            | 事業策略・業績 | 主力事業、成長策略、市場定位、營收・利潤推移        |
|            | 經營資源       | 人才、資金力、技術力、品牌力、海外展開、IT・DX 基盤 |
| `職場視角` | 人才・組織     | 人才培育制度、研修內容、組織體制、社風、企業文化    |
|            | 工作內容       | 新員工的業務內容、分發、業務流程、裁量權大小        |
|            | 評鑑・待遇     | 評鑑制度、晉升・晉級、薪資・福利、工作彈性          |

<br>

::: tip 進行企業研究的方法

1. 調查公司的`官方 SNS` 與`新聞網站`

2. 確認與公司相關的`新聞報導`

3. 確認在產業內的`定位`（與同業他社比較）

4. 參加`公司說明會`

5. 透過 OB・OG 訪問 蒐集`第一手資訊`

6. 閱讀`徵才頁面`與`員工訪談`

7. 參加`實習`（Internship）

:::

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">職種研究</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學三年級春季</span>

這項工作並非聚焦於產業或企業等「場所」，而是專注於您每天實際執行的「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">工作內容</span>」。

<br>

<div class="custom-title">職種紹介</div>

| 職種                   | 主要工作內容                               | 要求的技能・適性                              |
| ---------------------- | ------------------------------------------ | --------------------------------------------- |
| `事務・管理類`         | 文件製作、資料輸入、會計、人事、總務       | 精確性、事務處理能力、溝通能力、進度管理能力  |
| `企劃類`               | 商品企劃、行銷、經營企劃、營業企劃         | 分析力、構思力、簡報力、問題解決能力          |
| `技術・研究類`         | 研究開發、產品設計、實驗・分析、技術支援   | 專業知識、邏輯思考力、問題解決力、持續學習力  |
| `營業・銷售・服務類`   | 法人營業、個人營業、銷售、接待、客戶支援   | 溝通能力、提案力、談判力、靈活性              |
| `醫療・福祉類`         | 醫師、護理師、照護、復健、福祉服務         | 專業知識、對人技能、責任感、協調性            |
| `語言類`               | 翻譯、口譯、語言教師、編輯・寫作           | 語言能力、寫作能力、精確性、異文化理解        |
| `金融類`               | 銀行業務、證券、保險、資產運用、財務分析   | 數字能力、分析力、風險管理能力、溝通能力      |
| `創意類`               | 設計、影像製作、廣告、Web 製作、編輯       | 構思力、表現力、設計軟體技能、企劃力          |
| `資訊處理・通訊技術類` | 程式開發、系統開發、網路管理、AI・數據分析 | IT 技能、邏輯思考力、問題解決能力、團隊協作力 |

<br>

::: tip 進行職種研究的方法

1. [職種研究的做法](https://www.stephouse-recruit.com/article/job-research)
2. [職種研究 徹底指南](https://job.mynavi.jp/conts/2027/complete_guide/shokushu/)

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">實習（Internship）</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學三年級起（長假期間）</span>

這是讓學生或求職者在企業或團體中，於一定期間內<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">體驗實際工作</span>，對求職準備非常有幫助的計畫。

目的在於讓學生評估自己是否具備從事該工作所需的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">能力</span>，並進行與自身專業、興趣或未來職涯相關的就業體驗活動。

<br>

::: tip 參加實習的優點

1. 獲得`實務經驗`與`技能`

2. 理解社會人士的`禮儀`與`職場環境`

3. 建立`人脈`（與員工或其他學生交流）

4. 作為自我推薦與應徵動機的`素材`
   :::

<br>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>實習的種類</p>

| 類型       | 期間            | 主要內容                                                       |
| ---------- | --------------- | -------------------------------------------------------------- |
| `短期`     | 約 1 至 5 天    | 公司介紹、小組討論、職場參觀等。                               |
| `中期`     | 1 至 2 週       | 體驗部分實際業務，或針對課題提出解決方案。                     |
| `長期`     | 數月至 1 年以上 | 以接近員工的身分執行實務（通常有給薪）。在創業公司中較為常見。 |
| ------     | ------          | ----------                                                     |
| `夏`季實習 | 8 月 〜 9 月    | 參加者最多。產業研究的第一步。                                 |
| `秋`冬實習 | 10 月 〜 2 月   | 縮小志願企業範圍的時期。較容易與正式選考掛鉤。                 |
| `春`季實習 | 2 月 〜 3 月    | 正式選考前的最終確認。多為短期集中型。                         |

在計畫日本留學時，建議將費用分為「年度的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">學費</span>（含入學金）」與「每月的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">生活費</span>」這兩部分來考量。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">費用</div>

#### <div class="custom-title">年度學費（平均學費）</div>

在日本，年度學費主要由<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">入學金（僅限首年）、學費、設施設備費</span>等構成。 以下是根據學校種類劃分的初年度學費平均預估值。

| 學校種類             | 入學金（平均）              | 學費（年額）                 | 設施費・其他      | 初年度繳納金合計（平均）     |
| :------------------- | :-------------------------- | :--------------------------- | :---------------- | :--------------------------- |
| 國立大學             | `282,000 日圓`              | `535,800 日圓`               | ―                 | `817,800 日圓`               |
| 公立大學             | `392,391 日圓`              | `538,294 日圓`               | `7,540 日圓`      | `938,225 日圓`               |
| 私立大學（文科）     | `225,386 日圓`              | `813,244 日圓`               | `150,361 日圓`    | `1,188,991 日圓`             |
| 私立大學（理科）     | `251,029 日圓`              | `1,116,880 日圓`             | `198,353 日圓`    | `1,566,262 日圓`             |
| 私立大學（醫・齒學） | `1,073,083 日圓`            | `2,865,067 日圓`             | `952,389 日圓`    | `4,890,539 日圓`             |
| 專門學校             | `約 160,000 日圓`           | `約 710,000 日圓`            | `約 370,000 日圓` | `約 1,240,000 日圓`          |
| 日語教育機構         | `約 50,000 ～ 100,000 日圓` | `約 600,000 ～ 750,000 日圓` | ―                 | `約 700,000 ～ 850,000 日圓` |

#### <div class="custom-title">每月生活費</div>

根據最新調查，在日本生活的留學生月平均支出約為 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">148,000 日圓</span>。

| 項目                                 | 月額基準          |
| :----------------------------------- | :---------------- |
| 食費                                 | `約 28,000 日圓`  |
| 住宿費（學生宿舍・分租房）           | `約 38,000 日圓`  |
| 交通費                               | `約 5,000 日圓`   |
| 水電・瓦斯費                         | `約 7,000 日圓`   |
| 保險・醫療費                         | `約 3,000 日圓`   |
| 娛樂費・雜費（圖書・衣物・通訊費等） | `約 67,000 日圓`  |
| **合計**                             | `約 148,000 日圓` |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>地區導致的費用差異</p>

生活費會因城市規模而有顯著差異。東京及關東圈最<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">高</span>，東北與九州地區則相對較<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">便宜</span>。

| 地區           | 月平均生活費                 |
| :------------- | :--------------------------- |
| 東京周邊       | `約 160,000 日圓` 以上       |
| 九州・東北地區 | `約 110,000 ～ 130,000 日圓` |

<br>

#### <div class="custom-title">收入來源：打工</div>

許多留學生透過<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">打工</span>來補貼部分生活費。

| 項目         | 內容                                                                    |
| :----------- | :---------------------------------------------------------------------- |
| 許可申請     | 必須在出入國在留管理局取得「`資格外活動許可`」                          |
| 勞動時間限制 | `每週 28 小時` 以內（長假期間每日 8 小時以內）                          |
| 預估收入     | 時薪約 `900 ～ 1,200 日圓`（例：時薪 1,100 日圓 → 每月約 123,200 日圓） |
| 注意事項     | 打工僅為補貼生活費。因全額負擔學費非常`困難`，不可過度依賴              |

<br>

#### <div class="custom-title">節約要點</div>

| 節約・支援方法 | 內容                                                            |
| :------------- | :-------------------------------------------------------------- |
| 學費減免制度   | 國立大學與部分私立大學提供 `30％、50％` 甚至 `100％` 的學費免除 |
| 超市折扣       | 傍晚至夜間時段，便當或熟食會有「`半價`」折扣，對節省食費很有效  |
| 學生優惠       | 出示學生證，可購買 `學生定期券`、`電影優惠票` 等折扣項目        |

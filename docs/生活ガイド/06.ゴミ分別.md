日本のゴミ出しルールは、世界的に見ても<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">非常に細かく</span>、地域によって全くルールが異なります。

間違った出し方をすると、近所の人や大家さんとトラブルになったり、ゴミが回収されずに残されたりすることがあるので<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">注意</span>しましょう。

#### <div class="custom-title">基本的なゴミの分類</div>

一般的には、次の <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">4〜6 種類</span>に分けられます。ただし、地域によって分類方法が異なることがあります。

<br>

| 分類                       | 主な例                         | 出し方・注意点                 |
| -------------------------- | ------------------------------ | ------------------------------ |
| `燃えるゴミ`<br>（可燃）   | 生ごみ、汚れた紙、衣類、木製品 | 水分を切る                     |
| `燃えないゴミ`<br>（不燃） | 金属、ガラス、陶器、小型家電   | 割れ物は包んで「キケン」と表示 |
| `資源ごみ`                 | ペットボトル、缶、びん、古紙   | 洗って乾かす                   |
| `プラスチックごみ`         | 食品トレー、包装袋、カップ容器 | プラマーク確認、軽く洗う       |
| `粗大ごみ`                 | 家具、自転車、布団             | 事前申込・有料が多い           |
| `有害・危険ごみ`           | 電池、蛍光灯、スプレー缶       | 指定回収を利用                 |

#### <div class="custom-title">ゴミ出しの「3 つの鉄」</div>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>「指定の袋」を使う</p>

<img src='/images/i-img1200x620-1678874508903ha1dl8.jpg' class=' rounded-2xl'>

多くの自治体では、スーパーやコンビニで売っている <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">有料の指定ゴミ袋</span>を使わないと回収してくれません。レジ袋で出して良い地域とダメな地域があります。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>「曜日」と「時間」を守る</p>

<br>

<img src='/images/gomisuteba.jpeg' class='rounded-2xl w-90 mx-auto'>

<br>

ゴミの種類ごとに<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">収集曜日</span>が決まっています。各カテゴリーごとに「月曜日は燃えるゴミ」のように決まっています。

<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">当日の朝 8 時</span>ごろまでに出すのが一般的です。前日の夜に出すと、カラスに荒らされたり火災の原因になったりするため<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">禁止</span>されている場所が多いです。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>「正しい場所」に出す</p>

<img src='/images/20190423-114141.png' class='rounded-2xl'>

<br>

指定された「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">集積所</span>」に出す アパート専用のゴミ置き場か、近所の住民が共同で使っている場所（ゴミステーション）に出します。「隣のマンションの方が近い」といって他人の敷地に捨てるのは犯罪（<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">不法投棄</span>）です。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>罰則とトラブル</p>

<br>

<img src='/images/51PRZY3lCEL._AC_SX679_.jpg' class='rounded-2xl w-90 mx-auto'>

<br>

不法投棄には罰則があります。山林や道路、他人の敷地にゴミを捨てることは法律で禁止されており、「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">5 年以下の懲役</span>」または「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">1,000 万円以下の罰金</span>」が科されるなど、非常に重い罰則が定められています。

また、アパートの契約トラブルとして、ゴミ出しのルールを何度も守らない場合、近隣住民からの苦情を受けて、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">アパートの契約を解除され</span>、退去を求められるケースもあります。

<br>

#### <div class="custom-title">正しい捨て方（重要）</div>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>資源ゴミの「3R」ルール</p>

資源ゴミ（ボトル・缶など）は、そのまま捨てては<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">いけません</span>。以下のステップが必要です。

<br>

| Action              | 説明                                                                                           |
| ------------------- | ---------------------------------------------------------------------------------------------- |
| Rinse<br>(すすぐ)   | 中身を飲み干し、水で軽くすすぎます（`汚れ`や`臭い`を防ぐため）。                               |
| Remove<br>(外す)    | ペットボトルの`キャップ`と`ラベル`を外します。<br>プラスチック資源として分別します。           |
| Recycle<br>(分ける) | `ボトル本体`・`キャップ`・`ラベル`をそれぞれ指定されたカゴや袋に入れます。                     |
| 注意                | `汚れが落ちない`プラスチック（油だらけの容器など）は、「燃えるゴミ」として出すのが一般的です。 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>危険物の出し方</p>

ゴミ収集車や処理施設での<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">火災・怪我を防ぐ</span>ための非常に重要なルールです。

<br>

| 種類                       | 出し方・注意点                                                                                                     |
| -------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| スプレー缶・カセットボンベ | 必ず`中身を使い切って`から出します。振って音がしないか確認してください。                                           |
| 電池・バッテリー           | 普通のゴミに混ぜるのは`絶対 NG` です。火災の原因になります。`専用の回収箱`（家電量販店やスーパーなど）に出します。 |
| 刃物・割れたガラス         | 厚紙や新聞紙でしっかり`包み`、袋に大きく`キケン`と書いて出します。                                                 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>紙類のまとめ方</p>

雑誌、新聞、段ボールは、袋に入れず「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">ひもで十字に縛って</span>」出すのが基本です。

<br>

| 種類               | 出し方・注意点                                                                                         |
| ------------------ | ------------------------------------------------------------------------------------------------------ |
| 段ボール           | 必ず`平ら`に潰してから、ひもで縛って出します。                                                         |
| 小さい紙（雑がみ） | `封筒`や`チラシ`などは、`紙袋`にまとめて入れるか、雑誌の間に挟んで一緒に縛ると、バラバラになりません。 |

<br>

#### <div class="custom-title">粗大ゴミ（家具・家電）はどうする？</div>

家具や家電は、ふだんのゴミ捨て場に置いておくだけでは絶対に<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">回収されません</span>。 日本では、大きなゴミを捨てるには「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">予約</span>」と「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">料金（手数料）</span>」が必要です。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>家具や小さな家電</p>

一辺が <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">30cm〜50cm</span> を超えるもの（布団、机、椅子、自転車、電子レンジなど）が対象です。

| 手順             | 内容                                                                         |
| ---------------- | ---------------------------------------------------------------------------- |
| 申し込み（予約） | 自治体の「`粗大ゴミ受付センター`」に電話またはインターネットで申し込みます。 |
| 内容の伝達       | 捨てるものの`種類`・`サイズ`・`個数`を伝えます。                             |
| 確認             | `収集日`と`手数料`（金額）を案内されます。                                   |
| 処理券の購入     | コンビニやスーパーで指定金額の「`粗大ゴミ処理券`（シール）」を購入します。   |
| シール貼付       | シールに`氏名`や`受付番号`を書き、ゴミの目立つ場所に貼ります。               |
| 排出             | `指定日`の朝、玄関先や指定場所に出します（家の中までは回収不可）。           |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>捨てられない家電</p>

以下の <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">4 種類</span>の家電は、自治体の粗大ゴミとして捨てることはできません。 法律でリサイクルが義務付けられています。

`冷蔵庫・冷凍庫`、`洗濯機・衣類乾燥機`、`テレビ`、`エアコン`

<br>

| ケース           | 方法・料金                                                                     |
| ---------------- | ------------------------------------------------------------------------------ |
| 買い替える場合   | 新しい製品を購入する店舗に、古い家電の`引き取り`を依頼します。                 |
| 捨てるだけの場合 | 購入した店舗、またはお住まいの自治体の「家電リサイクル協力店」に`依頼`します。 |
| 料金             | 「リサイクル料金」＋「運搬費」で、合計 `3,000 円〜10,000 円`程度が一般的です。 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>無料で手放す方法</p>

| 方法               | 内容                                                                 |
| ------------------ | -------------------------------------------------------------------- |
| 学校の掲示板       | 「帰国するので冷蔵庫を`譲ります`」などの掲示を出す。                 |
| リサイクルショップ | 状態が良ければ買い取ってもらえる場合あり（`出張買取`も可）。         |
| ジモティー         | 日本で有名な「あげます・譲ります」`アプリ`。無料で譲れば処分費不要。 |

在计划日本留学时，建议将费用分为“年度的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">学费</span>（含入学金）”与“每月的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">生活费</span>”这两部分来考量。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">费用</div>

#### <div class="custom-title">年度学费（平均学费）</div>

在日本，年度学费主要由<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">入学金（仅限首年）、学费、设施设备费</span>等构成。 以下是根据学校种类划分的初年度学费平均预估值。

| 学校种类             | 入学金（平均）              | 学费（年额）                 | 设施费・其他      | 初年度缴纳金合计（平均）     |
| :------------------- | :-------------------------- | :--------------------------- | :---------------- | :--------------------------- |
| 国立大学             | `282,000 日元`              | `535,800 日元`               | ―                 | `817,800 日元`               |
| 公立大学             | `392,391 日元`              | `538,294 日元`               | `7,540 日元`      | `938,225 日元`               |
| 私立大学（文科）     | `225,386 日元`              | `813,244 日元`               | `150,361 日元`    | `1,188,991 日元`             |
| 私立大学（理科）     | `251,029 日元`              | `1,116,880 日元`             | `198,353 日元`    | `1,566,262 日元`             |
| 私立大学（医・齿学） | `1,073,083 日元`            | `2,865,067 日元`             | `952,389 日元`    | `4,890,539 日元`             |
| 专门学校             | `约 160,000 日元`           | `约 710,000 日元`            | `约 370,000 日元` | `约 1,240,000 日元`          |
| 日语教育机构         | `约 50,000 ～ 100,000 日元` | `约 600,000 ～ 750,000 日元` | ―                 | `约 700,000 ～ 850,000 日元` |

#### <div class="custom-title">每月生活费</div>

根据最新调查，在日本生活的留学生月平均支出约为 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">148,000 日元</span>。

| 项目                                 | 月额基准          |
| :----------------------------------- | :---------------- |
| 食费                                 | `约 28,000 日元`  |
| 住宿费（学生宿舍・分租房）           | `约 38,000 日元`  |
| 交通费                               | `约 5,000 日元`   |
| 水电・瓦斯费                         | `约 7,000 日元`   |
| 保险・医疗费                         | `约 3,000 日元`   |
| 娱乐费・杂费（图书・衣物・通讯费等） | `约 67,000 日元`  |
| **合计**                             | `约 148,000 日元` |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>地区导致的热用差异</p>

生活费会因城市规模而有显著差异。东京及关东圈最<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">高</span>，东北与九州地区则相对较<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">便宜</span>。

| 地区           | 月平均生活费                 |
| :------------- | :--------------------------- |
| 东京周边       | `约 160,000 日元` 以上       |
| 九州・东北地区 | `约 110,000 ～ 130,000 日元` |

<br>

#### <div class="custom-title">收入来源：打工</div>

许多留学生通过<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">打工</span>来补贴部分生活费。

| 项目         | 内容                                                                    |
| :----------- | :---------------------------------------------------------------------- |
| 许可申请     | 必须在出入国在留管理局取得“`资格外活动许可`”                            |
| 劳动时间限制 | `每周 28 小时` 以内（长假期间每日 8 小时以内）                          |
| 预估收入     | 时薪约 `900 ～ 1,200 日元`（例：时薪 1,100 日元 → 每月约 123,200 日元） |
| 注意事项     | 打工仅为补贴生活费。因全额负担学费非常`困难`，不可过度依赖              |

<br>

#### <div class="custom-title">节约要点</div>

| 节约・支援方法 | 内容                                                            |
| :------------- | :-------------------------------------------------------------- |
| 学费减免制度   | 国立大学与部分私立大学提供 `30％、50％` 甚至 `100％` 的学费免除 |
| 超市折扣       | 傍晚至夜间时段，便当或熟食会有“`半价`”折扣，对节省食费很有效    |
| 学生优惠       | 出示学生证，可购买 `学生定期券`、`电影优惠票` 等折扣项目        |

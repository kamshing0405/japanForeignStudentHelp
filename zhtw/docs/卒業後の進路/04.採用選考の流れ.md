採用選考是企業從應徵候選人中找出適合該職務人才的過程。透過結合<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">書類選考（文件審查）、筆記試驗（筆試）、面試、小組討論（Group Work）、適性檢查</span>等多元手法，評估應徵者的能力、適性與意願。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">Entry（意願登記/報名）</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學四年級春季</span>

「Entry」是指向企業表達「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">我有興趣</span>」的意願表示。

若要參加企業的選考，通常<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">必須</span>先完成 Entry。

就職活動中的 Entry 分為「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">Pre-Entry（初步登記）</span>」與「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">Entry（正式報名）</span>」兩個階段，請務必確認兩者的差異。

<div class="flex relative">

<img src='../../../images/text_entrance.png' class='absolute w-46 right-25 top-5 opacity-20'>

::: tip Pre-Entry（初步登記）

- 在求職網站或企業招募頁面註冊`姓名`與`大學名稱`。
- 企業會為您建立專屬的「`My Page（個人主頁）`」，並發送說明會通知與選考時程表。

<b>Entry（正式報名）</b>

- 為了`實際參加採用選考`（面試等）而進行的申請。

- 須提交`報名表`（Entry Sheet, ES）、`參加適性檢查`（如 SPI 等）、上傳`履歷表`。

- 完成後才能進入`書類選考`（文件審查）的階段。

:::

</div>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">公司說明會</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學四年級春季</span>

<div class="flex justify-center">

<br>

公司說明會是親自感受「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">公司內部氛圍</span>」與「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">具體工作內容</span>」的寶貴機會，這些是僅看企業官網或手冊無法得知的細節。

4. 絕對不可遲到

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">募集要項（招募簡章）</div>

募集要項是企業在進行徵才時，所提示的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">招募職缺內容、工作條件、待遇</span>等必要資訊的總結。

<div class='flex flex-col justify-center items-center '>

| ---------------- | ------------------------------------ |
| 雇用形態         | 正社員、契約社員、派遣社員等工作類型 |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">製作提交文件</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學四年級春季</span>

<div class="flex justify-center">
  <img src='../../../images/cv.png' class='rounded-2xl'>
</div>

<br>

`Entry Sheet (ES)` 與 `履歷表` 等應徵文件，是企業用來掌握應徵者特質與個人資訊的文件。

<br>

<div class="custom-title">Entry Sheet (ES) 與履歷表的差異</div>

|          | Entry Sheet（ES）                                      | 履歷表                            |
| -------- | ------------------------------------------------------ | --------------------------------- |
| 目的     | 判斷選考`合格與否`                                     | 公職`身份證明`、`事實確認`        |
|          | 確認`匹配度`（Matching）                               | 註冊至`人事資料庫`用              |
| 填寫項目 | 住所、姓名等`基本資訊`                                 | 住所、姓名等`基本資訊  `          |
|          | `自我推薦`（Self PR）、應徵動機、大學事蹟（Gakuchika） | 自`我推薦`與`應徵動機`            |
|          | 回答企業獨自設計的`問題`                               |                                   |
| 形式     | 企業`自定題`項較多                                     | `定型化`（如 JIS 規格等通用格式） |

<br>

履歷表可以說是職場上的「身份證」。企業首先確認的並非你本身的能力，而是「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">資訊的正確性</span>」與「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">誠實度</span>」。
<br>

相對地，Entry Sheet (ES) 是一份傳達「如何發揮我的個人能力」的「提案書」。僅憑履歷表無法看出的「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">思考過程</span>」與「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">熱情的溫度</span>」將會是評估對象。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">筆記試驗（筆試）</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學四年級春季</span>

筆試是求職過程中用來測量應徵者<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">基礎學力（語文・數理邏輯）、一般常識、邏輯思考能力、專業知識、性格（適性）</span>等的考試。

近年來，為了提高學生便利性與評估的客觀性，「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">線上受驗</span>」與「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">適性檢查（性格診斷+能力檢查）</span>」已成為主流。這些測試可在<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自家</span>或<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">測試中心 (Test Center)</span> 等多種地點進行。此外，部分企業也會在公司或<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">指定場所</span>使用紙本試卷進行適性檢查。
<br>

<div class="custom-title">筆記試驗的種類</div>

| 考試類別     | 內容                                               | 目的                                 |
| ------------ | -------------------------------------------------- | ------------------------------------ |
| 適性檢查     | `語文`（日文）、`數理`邏輯（數學・推理）、`英語`等 | 確認思考速度、準確度與邏輯思考力     |
|              | 有關`價值觀`與`行動模式`的問題                     | 確認對社風的適應性、抗壓性、職務適性 |
| 一般常識     | `時事`問題、`社會`、`理化`、`國文`、`商務禮儀`     | 確認作為社會人士的基本知識與關注廣度 |
| 小論文・作文 | 針對給予的`主題`撰寫自己的見解                     | 考察文章構成力、邏輯力與價值觀深度   |

<br>

<div class="custom-title">代表性的外部測試</div>

| 檢查名稱                                                          | 特徵                                                 | 問題構成                   |
| ----------------------------------------------------------------- | ---------------------------------------------------- | -------------------------- |
| [SPI3](https://www.spi.recruit.co.jp)                             | 日本`普及率最高`。平衡測量性格與能力（語文・數理）   | 語文・數理・性格檢查       |
| [玉手箱 Ⅲ](https://www.shl.co.jp/service/assessment/tamatebako3/) | 每題限制`時間極短`。重視速度與準確性。               | 　計算・語文・英語・OPQ    |
| [TG-WEB](https://tg-web.humanage.co.jp)                           | 與其他測試相比`難度較高`，考驗語文與數理的邏輯思考。 | 語文・計數・英語・性格檢查 |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">面試</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大學四年級春、夏季</span>

面試是企業與求職者深化相互理解、確認技能、熱情與適性的場合。

在日本，面試通常會進行<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">兩次以上</span>。這是為了讓不同階層的人觀察應徵者的性格與思維，判斷其是否為適合該企業的人才。

此外，現在「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">線上與實體併用</span>」已成為常態，面試已從單純的「問答」進化為「企業與學生互相確認匹配度的對話」。

<br>

<div class="custom-title">面試的種類</div>

| 種類           | 特徵                         | 優點                                                 |
| -------------- | ---------------------------- | ---------------------------------------------------- |
| 個人面試       | 面試官 1〜3 名對`學生 1 名`  | 最常見。能深度挖掘個人特質，確認真心話與價值觀       |
| 集體面試       | 面試官 1〜3 名對`學生多名`   | 能一次評估多名學生。可看出與他人的比較及簡要表達能力 |
| 小組討論（GD） | 針對特定主題，`學生彼此討論` | 可看見領導力、協調性、邏輯力等「在群體中的行為」     |

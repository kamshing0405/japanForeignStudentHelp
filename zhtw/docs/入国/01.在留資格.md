若要前往日本留學，必須持有由本國政府簽發的護照，以及由日本政府（大使館、領事館）發出的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">簽證（Visa）</span>。 留學生取得的在留資格通常為「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">留學</span>」。與觀光簽證不同，申請留學簽證必須先取得「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">在留資格認定證明書（COE）</span>」。

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">在留資格</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">總計約需 2 個月〜4 個月</span>

申請留學簽證的一般流程如下：

| 流程步驟     | 內容                                                                  |
| ------------ | --------------------------------------------------------------------- |
| 取得入學許可 | 從日本的教育機構收到`錄取通知`（入學許可書）                          |
| 申請 COE     | 由學校`代理`向入國管理局申請在留資格認定證明書（審查約需 1〜3 個月）  |
| 核發 COE     | 審查通過後，`COE` 正本將寄送給申請人                                  |
| 申請簽證     | 準備好 COE 等必要文件，至`駐外使領館`（如：日本台灣交流協會）申請簽證 |
| 入境日本     | 取得簽證後，需於 COE `有效期限`內入境日本                             |

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>COE（在留資格認定證明書）的重要性</p>

| 項目     | 內容                                                                                        |
| -------- | ------------------------------------------------------------------------------------------- |
| 作用     | 證明申請人符合`學習計畫`、`經濟能力`等入境條件的文件。提交 COE 申請簽證可大幅縮短審查時間。 |
| 有效期間 | 簽證核發後，原則上必須於 `3 個月內`入境。逾期可能導致 COE 失效。                            |

<br>

#### <div class="custom-title">核發在留卡</div>

| 項目     | 內容                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| 入境許可 | 在`機場`進行入境審查時，護照上會貼上「`上陸許可`」貼紙。                     |
| 在留卡   | 在新千歲、成田、羽田、中部、關西、廣島、福岡等主要機場，入境時會`當場核發`。 |
| 重要事項 | 在留卡是您在日本證明身分的`唯一公認證件`，法律規定必須隨身攜帶。             |

<br>

#### <div class="custom-title">入境後必要的申報</div>

| 項目                  | 內容                                                                                             |
| --------------------- | ------------------------------------------------------------------------------------------------ |
| 住所登記              | 確定住處後 `14 日內`，須持在留卡前往居住地的市、區役所辦理地址登記。                             |
| 個人編號（My Number） | 完成地址登記後，日後將收到「`個人編號通知書`」。辦理`銀行開戶`或工作的`稅務手續`時均為必需文件。 |

<br>

#### <div class="custom-title">資格外活動許可（工讀・工作許可）</div>

| 項目     | 內容                                                                           |
| -------- | ------------------------------------------------------------------------------ |
| 原則     | 持有「`留學`」在留資格，`原則上禁止從事有酬勞的工作`。                         |
| 申請     | 若要進行`工讀（兼職）`，必須申請`資格外活動許可`。                             |
| 時間限制 | 每週 `28 小時`以內；`暑假`等長假期間，為`每日 8 小時以內`。                    |
| 要點     | 入境時在機場審查台提交申請書，`即可當場獲得許可`，可省去日後前往入管局的麻煩。 |

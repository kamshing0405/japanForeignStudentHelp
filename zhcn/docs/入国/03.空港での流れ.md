第一次来到日本，感到紧张是正常的。但请放心，日本对于“第一次到访的外国人”是非常友好的国度。

以下将针对持`留学签证`入境的学生，详细解说抵达日本主要机场（成田、羽田、关西、中部等）后的专属入境步骤。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">机场入境流程</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">视拥挤情况约需 1〜2 小时</span>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">检疫 (Quarantine)</p>
</div>

- 下飞机后，请依照动线指示前进。

- 目前主要以健康状况询问及红外线热像仪测量体温为主。

- 若感到`身体不适`，请在此处向`工作人员咨询`。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入境审查 (Immigration)</p>
</div>

- 请前往“`中长期在留者（留学）`”专用柜台。

- 须出示的文件：`护照`、`签证`、Visit Japan Web 的 `QR Code` 或 `纸本入境卡`。

- 审查手续：进行 `指纹` 采集与 `面部照片` 拍摄。

- 领取在留卡：持有留学签证者，会当场获发“`在留卡`”。请务必现场确认卡面上的`姓名`与`出生年月日`是否正确。

<br>

::: warning 申请“资格外活动许可”

计划勤工俭学（打工）的人，请在此审查时提交申请书。工作人员会当场在在留卡背面盖章（若日后通过学校或自行申请会非常耗时，在此处办理是最理想的）。

:::

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">领取行李 (Baggage Claim)</p>
</div>

请确认屏幕显示的`航班编号`，前往对应的行李转盘领取您的`托运行李`。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">动植物检疫 (Optional)</p>
</div>

仅限有携带肉制品、水果、蔬菜等物品的人员需要前往。

<div class="flex w-auto min-h-15 rounded-md bg-blue-800 text-white items-center p-2">

    </div>

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">海关申报 (Customs)</p>
</div>

请使用于 [Visit Japan Web](https://services.digital.go.jp/visit-japan-web/) 注册完成的“海关申报”QR Code。

电子申报闸门：让机器读取 QR Code 与护照后，仅需通过人脸识别即可顺畅通过。

即使没有任何需要申报的物品，此手续也是所有人必须完成的。

<br>

    <p class="text-[#00873c] font-bold text-lg pl-0.5">入境后的交通移动</p>
</div>

<br>

| 交通方式 | 特征               | 优点                   | 缺点               |
| -------- | ------------------ | ---------------------- | ------------------ |
| 电车     | 班次多、运行准时   | 快速／便宜／有英语引导 | 换乘可能较令人不安 |
| 机场巴士 | 直达主要车站及酒店 | 无须换乘／行李轻松     | 视路况可能较费时   |
| 出租车   | Door to Door       | 不怕迷路／深夜 OK      | 费用昂贵           |
| 学校接送 | 需事先预约         | 最安心／完全不怕迷路   | 仅限部分学校提供   |

<h1 style="text-align: center;">公司雇用制度</h1>

<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70"></span>

在求職活動中，與薪資及業務內容同樣重要、甚至更值得關注的就是「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">公司雇用制度</span>」。這不僅僅是公司內部的規則，更是形塑勞動者`生活穩定`、`技能成長`及`未來選擇`的基石。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">雇用形態</div>

雇用形態是指企業與員工之間簽署的雇用契約種類。

- 雇用期間依雇用形態而異。
- 在薪資、福利待遇等方面存在差異。
- 工作時間與工作方式也有所不同。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 text-center'>簡單說明</p>

| 雇用形態   | 雇用期間          | 社會信用・穩定性 | 薪資・待遇               | 責任範圍       |
| ---------- | ----------------- | ---------------- | ------------------------ | -------------- |
| 正社員     | 無期（至退休）    | ✓✓✓✓✓            | 高（有加薪・獎金）       | 廣（責任較重） |
| 契約社員   | 有期（最長 3 年） | ✓✓✓              | 中等（多無獎金）         | 限定・明確     |
| 派遣社員   | 有期（最長 3 年） | ✓✓               | 時薪制（由派遣公司支付） | 指示範圍內     |
| 兼職／打工 | 有期／無期        | ✓                | 時薪制（排班制）         | 輔助・限定     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>正規雇用（正社員）</p>

| 項目               | 內容                       |
| ------------------ | -------------------------- |
| 雇用形態           | `正社員`（正規雇用）       |
| 雇用期間           | 無`特定期限`（無期）       |
| 工作時間           | 一般為`全職`（Full-time）  |
| 薪資・待遇         | 薪資、獎金、加薪`制度完善` |
| 社會保險・福利厚生 | 非常`充實`                 |
| 穩定性             | `高`                       |
| 責任・業務範圍     | 相對`較廣`                 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>契約社員</p>

| 項目               | 內容                                   |
| ------------------ | -------------------------------------- |
| 雇用形態           | `契約社員`（有期雇用）                 |
| 雇用期間           | `有期`（原則最長 `3 年`，不保證續約）  |
| 工作內容・職務範圍 | `明確規定`                             |
| 薪資・待遇         | 通常比正社員較為`受限`                 |
| 轉正機會           | 根據`實績`與`評鑑`，有`可能`轉為正社員 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>派遣社員</p>

| 項目               | 內容                                   |
| ------------------ | -------------------------------------- |
| 雇用形態           | `派遣社員`                             |
| 雇用契約           | 與`派遣公司`簽署                       |
| 工作地點           | 實際工作地點為`派遣先企業`（要派企業） |
| 雇用期間           | 原則最長 `3 年`                        |
| 工作時間・業務內容 | 聽從`派遣先企業`的指示                 |
| 薪資               | 由`派遣公司`支付                       |
| 擔當業務           | 多為`專業性`或`輔助性`業務             |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>兼職 / 打工 (Part-time)</p>

| 項目               | 內容                               |
| ------------------ | ---------------------------------- |
| 雇用形態           | 兼職、打工（非正規雇用）           |
| 工作形態           | `短時間`工作、每週勞動天數較少     |
| 工作目的           | 多為`兼顧`家庭或學業               |
| 雇用契約           | 多為`有期契約`                     |
| 福利厚生・社會保險 | 依`工作時數`或`天數`有適用條件限制 |
| 責任・業務範圍     | 相對較為`限定`                     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>雇用形態的福利、責任、加薪與穩定性對照</p>

| 項目       | 正社員                     | 契約社員             | 派遣社員                     | 兼職員工                   |
| ---------- | -------------------------- | -------------------- | ---------------------------- | -------------------------- |
| 福利       | 社會保險・福利厚生`充實`   | 有`部分`福利厚生     | `視派遣公司而定`，多為`受限` | 依~條件~部分適用           |
| 責任       | 對整體業務與成果的`責任大` | 對擔當`業務負責`     | 業務範圍`受限`，責任`較輕`   | 業務範圍`受限`，責任`較輕` |
| 晉升・加薪 | 制度`完善`                 | `有限`，有轉正可能性 | 基本`無`                     | 基本`無`                   |
| 雇用穩定性 | `高`                       | `中等`               | `低`                         | `低`                       |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">採用形態（招募類別）</div>

採用形態是指招募時考量職種及未來職涯路徑的招募框架。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>総合職</p>

| 項目     | 內容                                                     |
| -------- | -------------------------------------------------------- |
| 招募前提 | 以未來擔任`管理職`或經歷`廣泛職種`為前提進行招募         |
| 分發單位 | 入職時`不限定`單位，通常會經歷業務、企劃、管理等多個部門 |
| 職涯路徑 | 較易成為升遷、晉級對象，職涯發展路徑`明確`               |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>一般職</p>

| 項目           | 內容                                     |
| -------------- | ---------------------------------------- |
| 招募前提       | 以專門從事`特定業務`或職種為前提進行招募 |
| 分發・業務範圍 | `相對固定`，轉調或部門異動的可能性較少   |
| 職涯路徑       | 升遷、晉級範圍較為`受限`，不如綜合職廣泛 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>技術職</p>

| 項目           | 內容                                                 |
| -------------- | ---------------------------------------------------- |
| 招募前提       | 發揮`專業知識`或`技能`進行業務的招募形態             |
| 分發・業務範圍 | 多限定於`研究`、`開發`、`設計`、`IT・工程`等特定領域 |
| 升遷・加薪     | 多根據`技術能力`或成果進行評定                       |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">研修（員工培訓）</div>

研修是指企業為了讓從新進員工到管理職的各階層員工，有系統且高效率地習得業務所需知識與技能，而實施的組織性教育計畫。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>OJT（On the Job Training，在職培訓）</p>

| 項目       | 內容                                         |
| ---------- | -------------------------------------------- |
| 特徵       | 在`職場`透過實際業務進行的實踐性教育訓練     |
| 指導方式   | 由`前輩員工`或`上司`直接指導，習得技能與知識 |
| 目的・效果 | 習得與`實務直接掛鉤`的技能與問題解決能力     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>Off-JT（Off-the-Job Training，離職培訓）</p>

| 項目     | 內容                           |
| -------- | ------------------------------ |
| 特徵     | `離開`工作崗位進行的研修       |
| 學習內容 | 學習`產業知識`或`特定專業技能` |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>集合研修</p>

| 項目     | 內容                                                     |
| -------- | -------------------------------------------------------- |
| 參加者   | `全體新進員工`                                           |
| 學習內容 | 習得作為社會人士所需的`基礎知識`、`商務禮儀`及`企業文化` |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>導師制度（Mentor 制度）</p>

| 項目       | 內容                                                               |
| ---------- | ------------------------------------------------------------------ |
| 角色       | 由`資深員工`（導師）對年輕員工或新進員工（學員）進行`建議`與`指導` |
| 效果・優點 | 建立安心`可諮詢`的環境，`加速對業務的理解`                         |

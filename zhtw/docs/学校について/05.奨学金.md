### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">獎學金</div>

#### <div class="custom-title">獎學金的種類</div>

根據發放機構的不同，針對日本留學生的獎學金主要可分為以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">4 種類型</span>。

| 獎學金種類             | 概要                                                               |
| :--------------------- | :----------------------------------------------------------------- |
| 日本政府 (MEXT) 獎學金 | 由 `文部科學省` 發放。包含學費全免、每月生活費、來回機票。額度最高 |
| JASSO 獎學金           | 針對 `自費外國留學生`。提供每月固定額度的生活費補助                |
| 地方自治體・民間獎學金 | 由 `地方政府`、`企業`、`財團` 實施。可能限定專業領域或出身國家     |
| 大學獨自獎學金・減免   | 包含 `入學金免除`、學費 `30 ～ 100％ 減免`、校內限定獎學金等       |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請時期</p>

獎學金的申請時期各異，主要分為以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 個</span>階段。

| 區分         | 獎學金種類             | 內容・特徵                                            |
| :----------- | :--------------------- | :---------------------------------------------------- |
| 赴日以前申請 | MEXT 獎學金            | 須在赴日前約 `1 年開始準備`。透過大使館或大學推薦申請 |
|              | JASSO 預約制度         | `EJU 成績優秀者` 可在赴日前預約學習獎勵費             |
|              | 民間獎學金 (部分)      | 有些可透過 `入境前書面審查` 即可申請                  |
| 入境以後申請 | 民間・地方自治體獎學金 | 多數在入學後，透過 `就讀大學` 進行申請                |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>選拔方法</p>

獎學金的選拔不單取決於成績，通常會進行以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">審查</span>。

| 選拔方法 | 內容                                                       |
| :------- | :--------------------------------------------------------- |
| 書面審查 | 綜合評估 `成績`、`經濟狀況`、`研究計畫書`、`志願理由書` 等 |
| 筆試     | 部分獎學金 (如 MEXT) 會進行 `基礎學力` 或 `日語測驗`       |
| 面試     | 重視 `溝通能力`、`學習意欲`、`未來目標`                    |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請時的重要關鍵</p>

| 注意事項     | 內容                                               |
| :----------- | :------------------------------------------------- |
| 資訊蒐集時期 | 招募期間 `短` 且名額 `少`，儘早確認至關重要        |
| 日語能力     | 許多民間獎學金要求 `N2` 以上的日語能力             |
| 不可兼領     | MEXT 等高額獎學金多規定 `不可與其他獎學金同時領取` |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本政府獎學金</div>

日本政府（文部科學省）獎學金（通稱 MEXT 獎學金）設立於 1954 年，旨在透過接收<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">優秀的外國學生</span>，深化日本與海外各國的友好關係。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>支給內容</p>

MEXT 獎學金為全額給付型，主要包含以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">3 項</span>支援。

| 支援內容 | 詳細                                       |
| :------- | :----------------------------------------- |
| 學費     | 入學考試費、入學金、學費 `全額免除`        |
| 生活費   | 依課程每月發放約 `117,000 ～ 145,000` 日圓 |
| 機票     | 補助赴日及歸國時的國際 `往復機票`          |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請管道</p>

MEXT 獎學金的主要官方申請管道有以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 個</span>。

| 推薦方式     | 選拔機構                | 招募・申請時期（基準） | 主要選拔內容            | 特徵                       |
| :----------- | :---------------------- | :--------------------- | :---------------------- | :------------------------- |
| `大使館推薦` | 日本駐外公館 (大使館等) | 3 ～ 5 月              | 書面審查・筆試・面試    | 名額較多，最常見的申請方式 |
| `大學推薦`   | 日本大學 → 文部科學省   | 12 ～ 隔年 1 月        | 書面審查・面試 (或免試) | 以大學間交流協定推薦為主   |

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>重要注意事項</p>

| 注意事項   | 內容                                                  |
| :--------- | :---------------------------------------------------- |
| `日語學習` | 許多類別規定在正式入學前需接受約 6 個月的預備日語教育 |
| `不可兼領` | 領取 MEXT 者不可同時領取 JASSO 或其他民間獎學金       |
| `每年審查` | 若成績或出席狀況不佳，獎學金可能會被 `停止發放`       |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本學生支援機構（JASSO）獎學金</div>

與 MEXT 獎學金相比，JASSO 獎學金的受眾<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">範圍更廣</span>，是日本自費留學生利用率最高、最實用的獎學金制度。

JASSO 提供的獎學金主要是針對自費留學生的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">給付型獎學金</span>（不需償還），主要分為以下兩類。

在日本留學試驗（EJU）中取得優秀成績的學生，可以在來日本之前「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">預約</span>」此獎學金。請務必在報名 EJU 時勾選申請選項。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>文部科學省外國留學生學習獎勵費</p>

此獎學金針對學業優秀且需要經濟援助的自費留學生。

| 項目       | 內容                                                                                                 |
| :--------- | :--------------------------------------------------------------------------------------------------- |
| 對象者     | 就讀於 `研究所・大學`・`短期大學`・`高等專門學校`・`專修學校`・`預備教育課程`・`日語教育機構` 的學生 |
| 支給額     | 每月 `48,000 日圓` (日語教育機構學生同額)                                                            |
| 支給期間   | 原則上為 `12 個月`                                                                                   |
| 學業條件   | `學業成績優秀`，GPA 達到規定基準                                                                     |
| 經濟條件 ① | 國外匯款每月平均額在 `120,000 日圓以下` (不含學費入學金)                                             |
| 經濟條件 ② | 在日扶養人（若有）年收入須 `低於 500 萬日圓`                                                         |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>海外留學支援制度（協定受入）獎學金</p>

此獎學金對象為基於大學間交流協定的交換留學生。

| 項目     | 內容                                                    |
| :------- | :------------------------------------------------------ |
| 對象者   | 基於交流協定，進行 `8 天以上 1 年以內` 短期留學的學生   |
| 支給額   | 每月 `80,000 日圓`                                      |
| 申請方式 | 學生 `不可直接申請`。須透過母校向日本的接收大學提出申請 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請相關基本條件</p>

| 應募條件・注意點 | 內容                                                  |
| :--------------- | :---------------------------------------------------- |
| 在留資格         | 必須持有「`留學`」在留資格                            |
| 不可兼領         | `不可同時領取` MEXT 或母國政府獎學金                  |
| 成績基準         | 前一年度成績依 JASSO 換算須達 `2.30 以上`（滿分 3.0） |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">其他獎學金</div>

除了政府與 JASSO 之外，還有許多由<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地方自治體、民間團體、大學</span>獨立設置的獎學金。

這些獎學金種類繁多，特徵在於通常帶有<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地域性、專業性、學校限制</span>等條件。

<br>

#### <div class="custom-title">地方自治體・國際交流協會獎學金</div>

| 項目     | 內容                                                          |
| :------- | :------------------------------------------------------------ |
| 對象者   | 居住於 `該地區` 或就讀於該地區大學的學生                      |
| 實施團體 | 都道府縣・市町村等 `地方政府`、`國際交流協會`                 |
| 特徵 ①   | 有明確的 `地域限制`（例如：僅限東京都居住者）                 |
| 特徵 ②   | 除了 `現金給付`，也有 `住宅補助` 或 `生活用品發放` 等多元方式 |

<br>

#### <div class="custom-title">民間團體獎學金（Private Foundation Scholarships）</div>

| 項目   | 內容                                                       |
| :----- | :--------------------------------------------------------- |
| 對象者 | 由 `民間企業` 或 `獎學財團` 設立，基於企業興趣或創始人理念 |
| 特徵 ① | 多為 `專業領域特定型`（如理工、藝術、特定經濟領域等）      |
| 特徵 ② | 可能有`國籍限制`（如僅限中國、東南亞出身者）               |
| 特徵 ③ | 通常要`求參加交流會或研討會`，有助於建立人脈               |

<br>

#### <div class="custom-title">大學獨自獎學金・學費減免制度</div>

| 項目         | 內容                                                      |
| :----------- | :-------------------------------------------------------- |
| 對象者       | 在 `該大學` 正式就讀且成績優秀的留學生                    |
| 學費減免制度 | 學費 `30％、50％` 甚至 `100％ 免除`（在私立大學尤為常見） |
| 校內獎學金   | 來自大學基金的 `每月生活補助` 或一次性給付                |
| 入學獎勵金   | 針對 `入學考試成績優秀者` 發放的獎勵金                    |

<br>

#### <div class="custom-title">發放額度基準</div>

| 獎學金種類                 | 支給內容・注意點                                |
| :------------------------- | :---------------------------------------------- |
| 地方自治體・民間團體獎學金 | 每月約 `30,000 日圓 ～ 200,000 日圓`            |
| 大學獨自支援               | `金額` 與 `內容` 差異極大。須在志願學校官網確認 |

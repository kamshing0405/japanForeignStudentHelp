為了進入日本的高等教育機關進修，留學生通常需要參加幾項<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">重要考試</span>。這些考試不僅評估<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日語能力</span>，也會評估基礎的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">學力</span>。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本留學試驗（EJU）</div>

#### <div class="custom-title">日本留学試験（EJU）</div>

[日本留学試験](https://www.jasso.go.jp/ryugaku/eju/index.html)是以希望作為外國留學生進入日本大學（學部）等就讀的人為對象，旨在評估其是否具備日本大學等機關所要求的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日語能力及基礎學力</span>。

| 項目     | 內容                                       |
| -------- | ------------------------------------------ |
| 実施時期 | 每年 `6 月・11 月`                         |
| 日語     | `讀解`、`聽解`（聽讀解）、`記述`（作文）   |
| 理科     | 從物理、化學、生物中選擇 `2 科目`          |
| 綜合科目 | `政治、經濟、地理、歷史`                   |
| 數學     | 分為`文科用`（課程 1）與`理科用`（課程 2） |
| 重要性   | 許多大學將其活用為`判定合格與否的重要資料` |

<br>

#### <div class="custom-title">日語能力試驗（JLPT）</div>

這是世界規模最大的[日本語能力認定試験](https://www.jlpt.jp)。分為從 N1 到 N5 的 5 個等級。

| 項目     | 內容                                            |
| -------- | ----------------------------------------------- |
| 實施時期 | 每年 `7 月・12 月`                              |
| 升學用途 | 專門學校通常要求 `N2 以上`，研究所則多要求 `N1` |
| 就業用途 | 許多企業希望應徵者取得 `N1` 等級                |
| 優點     | 證照`沒有有效期限`，是國際通用資格              |

<br>

#### <div class="custom-title">英語能力考試</div>

若要報考著名大學的學部（如早稻田、慶應、舊帝國大學等）或研究所（特別是理工、經濟、醫學系），通常會被要求提交英語考試成績。

| 考試名稱                                        | 特徵                                         |
| ----------------------------------------------- | -------------------------------------------- |
| [TOEFL](https://www.toefl-ibt.jp)               | 知名度最高，幾乎`所有大學皆可採計`           |
| [TOEIC](https://www.iibc-global.org/toeic.html) | 在日本國內評價高，在 `商學、實務領域` 較有效 |
| [IELTS](https://www.eiken.or.jp/ielts/)         | 知名度上升中，但在日本 採計範圍`相對有限`    |

<br>

#### <div class="custom-title">大學校內考（校內考）</div>

通過書面審查或 EJU 選拔後，由各大學獨自實施的考試。

| 項目   | 內容                                         |
| ------ | -------------------------------------------- |
| 筆試   | 透過小論文或`專業科目`評估邏輯能力與表達能力 |
| 面試   | 重視`留學動機`、`學習計畫`及`日語會話能力`   |
| 重要性 | 作為決定錄取與否的`最終選拔`，面試尤其重要   |

<br>

#### <div class="custom-title">其他特殊考試</div>

| 考試名稱         | 內容                                             |
| ---------------- | ------------------------------------------------ |
| 大學入學共通考試 | 多數留學生`無需參加`，但醫學系等部分學科可能需要 |
| 藝術類術科測驗   | 美大、音大會進行`素描`、`演奏`等術科檢定         |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">考試相關注意事項</div>

在日本參加考試（特別是大學、研究所、專門學校）時，有一些日本特有的習慣與規則。我們總結了留學生或海外居住者容易出錯的重點。

#### <div class="custom-title">文件準備</div>

| 注意事項     | 具體內容                                                                       |
| ------------ | ------------------------------------------------------------------------------ |
| 提交正本     | 畢業證書、成績單原則上需提交「`正本`」。多數情況不接受影本                     |
| 附上翻譯本   | 若文件非日文或英文，需附上官方翻譯（`公證`）                                   |
| 確認有效期限 | EJU、TOEFL、IELTS 原則上 `2 年有效`。也要注意需提交發行 `3 個月內的文件之規定` |
| 統一姓名拼音 | 確保護照、證照、成績單上的姓名拼寫`完全一致`                                   |

<br>

#### <div class="custom-title">出願截止・郵寄</div>

| 注意事項       | 具體內容                                                             |
| -------------- | -------------------------------------------------------------------- |
| 必著・消印有效 | `必著`：需於截止日前送達大學；`消印有效`：只要有截止日當天的郵戳即可 |
| 郵寄方法       | 通常指定使用掛號（`書留`）、`Letter Pack` 等可追蹤進度的郵寄方式     |

<br>

#### <div class="custom-title">考試報名時間</div>

| 時期        | 考試・手續  | 補充說明                                  |
| ----------- | ----------- | ----------------------------------------- |
| 3 月與 9 月 | EJU 報名    | 針對 `6 月、11 月`實施之考試              |
| 3 月與 8 月 | JLPT 報名   | 針對 `7 月、12 月`實施之考試              |
| 全年        | TOEFL/TOEIC | 成績單郵寄約`需 1 個月`，建議儘早參加考試 |

<br>

#### <div class="custom-title">EJU 科目選擇</div>

| 注意事項   | 具體內容                                                 |
| ---------- | -------------------------------------------------------- |
| 文理科選擇 | 依照志願學校的要求選擇綜合科目`文科`或`理科`             |
| 出題語言   | 確認選擇`日語`或`英語`（日語學科一般指定必須選日語出題） |

<br>

#### <div class="custom-title">面試禮儀</div>

| 注意事項 | 具體內容                                            |
| -------- | --------------------------------------------------- |
| 服裝     | 穿著`西裝`等具備`整潔感的服裝`。嚴禁牛仔褲、涼鞋    |
| 守時     | 無論現場或線上，基本要求為 `15 分鐘前`抵達          |
| 志願理由 | 準備好能具體且具邏輯性地說明`選擇該大學/學部的理由` |

<br>

#### <div class="custom-title">報名費支付</div>

| 注意事項 | 具體內容                                       |
| -------- | ---------------------------------------------- |
| 支付證明 | 切勿忘記將匯款證明或繳費證明`粘貼於報名文件上` |

<h1 style="text-align: center;">会社雇用制度</h1>

<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70"></span>

就職活動において、給与や業務内容と同じくらい、あるいはそれ以上に注目すべきなのが「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">会社雇用制度</span>」です。これは単なる社内ルールではなく、働く人の`生活の安定`、`スキルの成長`、`将来の選択肢`」を形作る土台だからです。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">雇用形態</div>

雇用形態とは、企業と社員が結ぶ雇用契約の種類である

- 契約期間は雇用形態によって異なる
- 給与や福利厚生などの待遇に違いがある
- 勤務時間や働き方にも差がある

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 text-center'>簡単な説明</p>

| 雇用形態 | 雇用期間         | 社会的信用・安定性 | 給与・待遇              | 責任の範囲       |
| -------- | ---------------- | ------------------ | ----------------------- | ---------------- |
| 正社員   | 無期 (定年まで)  | ✓✓✓✓✓              | 高い (昇給・賞与あり)   | 広い (責任重め)  |
| 契約社員 | 有期 (最長 3 年) | ✓✓✓                | 中程度 (賞与なしも多い) | 限定的・明確     |
| 派遣社員 | 有期 (最長 3 年) | ✓✓                 | 時給制 (派遣元から支給) | 指示された範囲内 |
| パート   | 有期/無期        | ✓                  | 時給制 (シフト制)       | 補助的・限定的   |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>正規雇用</p>

| 項目               | 内容                               |
| ------------------ | ---------------------------------- |
| 雇用形態           | `正社員`（正規雇用）               |
| 雇用期間           | 定めなし（`無期`）                 |
| 勤務時間           | `フルタイム`が一般的               |
| 給与・待遇         | 給与・賞与・昇給制度が`整っている` |
| 社会保険・福利厚生 | `充実`している                     |
| 安定性             | `高い`                             |
| 責任・業務範囲     | 比較的`広い`                       |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>契約社員</p>

| 項目               | 内容                                       |
| ------------------ | ------------------------------------------ |
| 雇用形態           | `契約社員`（有期雇用）                     |
| 雇用期間           | `有期`（原則最長 `3 年`、更新の保証なし）  |
| 勤務内容・職務範囲 | `明確に`決められている                     |
| 給与・待遇         | 正社員より`限定的`なことが多い             |
| 正社員登用         | 実績や評価によって`可能性がある`場合もある |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>派遣社員</p>

| 項目               | 内容                           |
| ------------------ | ------------------------------ |
| 雇用形態           | 派遣社員                       |
| 雇用契約           | `派遣会社`と結ぶ               |
| 勤務先             | 実際の勤務先は`派遣先企業`     |
| 雇用期間           | 原則`最大 3 年`                |
| 勤務時間・業務内容 | `派遣先の指示`に従う           |
| 給与               | `派遣会社`から支払われる       |
| 担当業務           | `専門的`・`補助的`な業務が多い |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>パート</p>

| 項目               | 内容                                   |
| ------------------ | -------------------------------------- |
| 雇用形態           | パート・アルバイト（`非正規雇用`）     |
| 勤務形態           | `短時間`勤務・週の労働日数少なめ       |
| 働く目的           | `家庭`や`学業`との両立が多い           |
| 雇用契約           | `有期`の場合が多い                     |
| 福利厚生・社会保険 | 勤務`時間`や`日数`によって適用条件あり |
| 責任・業務範囲     | `比較的限定`される                     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>それぞれ雇用形態の福利・責任・昇給・安定性</p>

| 項目       | 正社員                               | 契約社員                         | 派遣社員                             | パート社員                         |
| ---------- | ------------------------------------ | -------------------------------- | ------------------------------------ | ---------------------------------- |
| 福利       | 社会保険・福利厚生が`充実`           | 一部福利厚生`あり`               | 派遣会社による、`限定的`な場合が多い | `条件`により一部適用               |
| 責任       | 業務全体・成果に対する責任が`大きい` | `担当業務`に対する責任           | 業務範囲が`限定`され、責任は`軽い`   | 業務範囲が`限定`され、責任は`軽い` |
| 昇進・昇給 | 制度が`整っている`                   | `限定的`、正社員登用の可能性あり | 基本的に`ない`                       | 基本的に`ない`                     |
| 雇用安定性 | `高い`                               | `中程度`                         | `低い`                               | `低い`                             |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">採用形態</div>

採用形態とは、採用時における職種や将来のキャリアパスを考慮した採用の枠組みを指す。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>総合職</p>

| 項目         | 内容                                                                         |
| ------------ | ---------------------------------------------------------------------------- |
| 採用の前提   | 将来的に`管理職`や`幅広い職種`を経験することを前提に採用                     |
| 配属         | 入社時に`限定されず`、営業・企画・管理部門など複数の部署を経験する場合が多い |
| キャリアパス | 昇進・昇格の対象となりやすく、キャリアパスが`明確`                           |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>一般職</p>

| 項目           | 内容                                                 |
| -------------- | ---------------------------------------------------- |
| 採用の前提     | `特定の業務`や職種に専門的に従事することを前提に採用 |
| 配属・業務範囲 | `比較的固定`され、転勤や部署異動の可能性は少ない     |
| キャリアパス   | 昇進・昇格の範囲は`限定`的で、総合職ほど広くない     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>技術職</p>

| 項目           | 内容                                                                               |
| -------------- | ---------------------------------------------------------------------------------- |
| 採用の前提     | `専門的な知識`や`技能`を活かして業務を行う採用形態                                 |
| 配属・業務範囲 | `研究`・`開発`・`設計`・`IT・エンジニアリング`など、特定分野に限定される場合が多い |
| 昇進・昇給     | `技術力`や成果に基づく場合が多い                                                   |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">研修</div>

研修とは、企業が新入社員から管理職までの従業員に対し、業務に必要な知識やスキルを体系的かつ効率的に習得させるために実施する組織的な教育プログラム。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>OJT（On Job Training）</p>

| 項目       | 内容                                                     |
| ---------- | -------------------------------------------------------- |
| 特徴       | `職場`で実際の業務を通じて行われる実践的な教育訓練       |
| 指導方法   | `先輩社員`や`上司`から直接指導を受けてスキルや知識を習得 |
| 目的・効果 | `実務に直結`したスキルや問題解決能力の習得               |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>Off-JT（Off-the-Job Training）</p>

| 項目     | 内容                             |
| -------- | -------------------------------- |
| 特徴     | 職場を`離れ`て行う研修           |
| 学習内容 | `業界知識`や`特定のスキル`を学ぶ |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>集合研修</p>

| 項目     | 内容                                                             |
| -------- | ---------------------------------------------------------------- |
| 参加者   | `新入社員全員`                                                   |
| 学習内容 | 社会人として必要な`基礎知識`、`ビジネスマナー`、`企業文化`の習得 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>メンター制度</p>

| 項目           | 内容                                                                                   |
| -------------- | -------------------------------------------------------------------------------------- |
| 役割           | `経験豊富な社員`（メンター）が、若手社員や新入社員（メンティー）に`助言`・`指導`を行う |
| 効果・メリット | 安心して`相談できる`環境が整い、`業務理解が早まる`                                     |

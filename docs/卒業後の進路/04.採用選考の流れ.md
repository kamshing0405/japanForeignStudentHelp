採用選考とは、企業が求人に応募した候補者の中から、職務に適した人材を見つけ出すプロセスで、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">書類選考、筆記試験、面接、グループワーク、適性検査</span>など多様な手法を組み合わせて、応募者の能力・適性・意欲を評価します

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">エントリー</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 4 年生の春</span>

「エントリー」とは、企業側への「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">興味がある</span>」という意思表示です

企業の選考に参加するには、通常、エントリーが<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70
">必要</span>となります

就職活動のエントリーは、「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">プレエントリー</span>」 と 「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">本エントリー</span>」 の 2 段階に分けられるため、その違いを確認しておきましょう。

<div class="flex relative">

<img src='../../images/text_entrance.png' class='absolute w-46 right-15 bottom-0 opacity-25'>

::: tip プレエントリー

- 求人サイトや、企業の採用ページから`氏名や大学名を登録`します
- 企業専用の「`マイページ`」が作られ、説明会の案内や選考スケジュールが届くようになります

本エントリー

- `実際に採用選考`（面接など）に参加するための申し込みです
- エントリーシート（ES）の`提出`、適性検査（SPI など）の`受検`、履歴書のアップロード
- `書類選考`の土俵に上がることができます

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">会社説明会</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 4 年生の春</span>

<div class="flex justify-center">
  <img src='../../images/seminar.png' class='rounded-2xl w-80'>
</div>
<br>

会社説明会は、企業の HP やパンフレットだけでは分からない「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">社内の雰囲気</span>」や「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">具体的な仕事内容</span>」を肌で感じるための貴重な機会です。

<br>

<br>

<div class="custom-title">会社説明会の種類</div>

会社説明会は以下の２つに分類できます。
<br>

<div class='flex  gap-2 h-60'>
<div class='flex-1 h-full'>

::: tip 会社説明会（単独型）

- <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">1 つ</span>の企業だけで開催する会社説明会
- <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">企業</span>理解を深められる
- <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">事前の予約</span>が必要になる

:::

</div>

<div class='flex-1'>

::: tip 合同企業説明会

- ひとつの会場に<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">たくさん</span>の企業が集る
- <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">業界</span>の知識を深められる
- 企業ごとに<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">比較</span>できる
  :::

</div>

  <img src='../../images/online_school_girl.png' class='rounded-2xl w-80'>

<br>

最近は「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">オンライン</span>」の パターンがありますが、それぞれの特徴とチェックポイントをまとめました。

<br>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>オンデマンド型</p>

- 事前に収録した動画を配信して行う会社説明会
- 自分の都合のいいタイミングで視聴することができる

<br>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>ライブ配信</p>

- 決まった日時に行われる会社説明会を Web 上で配信します
- リアルタイムで参加でき配信中にチャット機能などを使って企業側に質問をすることができる

<br>

<div class="custom-title">会社説明会の準備</div>

1. 参加する企業について予習する。
2. Wi-Fi など安定した通信環境を用意
3. 積極的に質問して、今までの企業研究でわからなかった疑問を解消する。
4. 遅刻をしない

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">提出書類作成</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 4 年生の春</span>

<div class="flex justify-center">
  <img src='../../images/cv.png' class='rounded-2xl'>
</div>

<br>

`エントリーシート` や `履歴書`などは、応募書類は企業が応募者の特性や情報を把握するための書類です

<br>

<div class="custom-title">エントリーシート（ES）と履歴書の違い</div>

|          | エントリーシート（ES）             | 履歴書                         |
| -------- | ---------------------------------- | ------------------------------ |
| 目的     | 選考の`合否`判断                   | 公的な身分`証明`・事実`確認`　 |
|          | `マッチング`確認                   | 　 `人事データ`に登録するため  |
| 記入項目 | 住所、氏名などの`基本情報`         | 　 住所、氏名などの`基本情報`  |
|          | `自己 PR` や`志望動機`、`ガクチカ` | 　`自己 PR` や`志望動機`       |
|          | 企業独自の`質問`への`回答`         | 　                             |
| 形式     | 企業`独自`の設問が`多い`           | 　 `定型`（JIS 規格など共通）  |

<br>

履歴書は、いわばビジネスにおける「身分証明書」です。企業側がまず確認するのは、あなたのスペックそのものよりも、「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">情報の正確さ</span>」と「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">誠実さ</span>」です。

<br>

一方で、エントリーシート（ES）は「自分という商品をどう活かせるか」を伝える「提案書」です。履歴書のスペックだけでは見えない、あなたの「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">思考のプロセス</span>」や「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">熱意の温度感</span>」が評価の対象になります。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">筆記試験</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 4 年生の春</span>

筆記試験は、就職・転職活動において応募者の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">基礎学力（言語・非言語）、一般常識、論理的思考力、専門知識、性格（適性・夏）</span>などを測るための試験。

近年は、学生の利便性や評価の客観性を高めるために、「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">オンライン受験</span>」や「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">適性検査（性格診断＋能力検査）</span>」が主流となっています。これらのテストは、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自宅</span>や<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">テストセンター</span>など、さまざまな場所で実施されています。さらに、企業や企業が<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">指定する場所</span>で、紙の試験用紙を使って適性検査が実施されることもあります。

<br>

<div class="custom-title">筆記試験の種類</div>

| 試験種別     | 内容                                                 | 目的                                            |
| ------------ | ---------------------------------------------------- | ----------------------------------------------- |
| 適性検査     | `言語`（国語）、`非言語`（数学・論理）、`英語`など   | 思考の速さ、正確さ、論理的思考力の確認　        |
|              | `価値観`や`行動パターン`に関する質問                 | 　 社風への適性、ストレス耐性、職種適性の確認   |
| 一般常識     | `時事`問題、`社会`、`理科`、`国語`、`ビジネスマナ`ー | 　 社会人としての最低限の知識・関心の広さを確認 |
| 小論文・作文 | 与えられた`テーマ`に対して自身の考えを書く           | 　文章構成力、論理力、価値観の深掘り            |

<br>

<div class="custom-title">代表的な外部テスト</div>

| 検査名称                                                          | 特徴                                                                                  | 問題構成                      |
| ----------------------------------------------------------------- | ------------------------------------------------------------------------------------- | ----------------------------- |
| [SPI3](https://www.spi.recruit.co.jp)                             | 日本で`最も普及`。性格と能力（言語・非言語）をバランスよく測定。                      | 言語・非言語・性格検査        |
| [玉手箱 Ⅲ](https://www.shl.co.jp/service/assessment/tamatebako3/) | 1 問あたりの制限`時間が非常に短い`。スピードと正確性を重視。                          | 　 計数・言語・英語・OPQ      |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">面接</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 4 年生の春・夏</span>

面接は、企業と求職者が相互理解を深め、スキル・意欲・適性を見極める場。

日本の面接試験は、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 回以上</span>実施されるのが一般的です。応募者の性格や思考などをさまざまな立場の人に見てもらい、その企業にとって良い人材かどうかを見極めるためです。

また、現在は<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">オンラインと対面の併用</span>が当たり前になり、単なる「質問と回答」の場から、「企業と学生がお互いのマッチングを確かめ合う対話」へと進化しています。

<br>

<div class="custom-title">面接の種類</div>

| 種類                     | 特徴                                   | メリット                                                                    |
| ------------------------ | -------------------------------------- | --------------------------------------------------------------------------- |
| 個人面接                 | 面接官 1〜3 名に対し、`学生 1 名`。    | 最も一般的。一人を深く掘り下げ、本音や価値観を確認できる。                  |
| 集団面接                 | 面接官 1〜3 名に対し、`学生複数名`。   | 　一度に多くの学生を評価できる。他者との比較や、端的に話す力がわかる。      |
| グループディスカッション | 特定のテーマに対し、`学生同士で議論`。 | 　 リーダーシップ、協調性、論理的思考力など「集団内での振る舞い」が見える。 |

日本的垃圾處理規則，即使從全球角度來看也屬於<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">非常細緻</span>的類型，且隨地域不同，規則也完全不同。

如果丟棄方式錯誤，可能會與鄰居或房東產生糾紛，或導致垃圾未被回收而被遺留在現場，請務必<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">多加注意</span>。

#### <div class="custom-title">基本的垃圾分類</div>

一般而言會分為下列 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">4〜6 種</span>。但各區域的分類方式可能有所差異。

<br>

| 分類                             | 主要範例                     | 排放方式・注意點                    |
| :------------------------------- | :--------------------------- | :---------------------------------- |
| `可燃垃圾`<br>（燃えるゴミ）     | 廚餘、髒污紙張、衣類、木製品 | 須瀝乾水分                          |
| `不可燃垃圾`<br>（燃えないゴミ） | 金屬、玻璃、陶瓷器、小型家電 | 易碎品須包裹並標示「危險 (キケン)」 |
| `資源垃圾`                       | 寶特瓶、罐類、玻璃瓶、廢紙   | 須清洗並晾乾                        |
| `塑膠類垃圾`                     | 食品托盤、包裝袋、杯狀容器   | 確認塑膠標誌、輕微沖洗              |
| `大型垃圾`                       | 家具、自行車、棉被           | 多為事前申請並收費                  |
| `有害・危險垃圾`                 | 電池、螢光燈管、噴霧罐       | 使用指定回收方式                    |

#### <div class="custom-title">垃圾排放的「三大鐵則」</div>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>使用「指定垃圾袋」</p>

<img src='../../../images/i-img1200x620-1678874508903ha1dl8.jpg' class=' rounded-2xl'>

在許多地方自治體，若不使用在超市或便利商店販售的 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">付費指定垃圾袋</span>，清潔隊將不予回收。有些地區可以使用普通塑膠袋，但有些地區則嚴格禁止。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>遵守「日期」與「時間」</p>

<br>

<img src='../../../images/gomisuteba.jpeg' class='rounded-2xl w-90 mx-auto'>

<br>

每種垃圾都有固定的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">清理日期</span>。各類別皆有明確規定，例如「星期一為可燃垃圾」。

一般建議在<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">當天早上 8 點</span>前排出。許多地方<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">禁止</span>在前一天晚上丟棄，以免遭到烏鴉破壞或引發火災。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>丟在「正確地點」</p>

<img src='../../../images/20190423-114141.png' class='rounded-2xl'>

<br>

請將垃圾拿到指定的「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">集積所</span>」。應放在公寓專用的垃圾場，或鄰近居民共同使用的場所（垃圾站）。以「隔壁公寓比較近」為由將垃圾丟在他人土地是犯罪行為（<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">非法棄置</span>）。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>罰則與糾紛</p>

<br>

<img src='../../../images/51PRZY3lCEL._AC_SX679_.jpg' class='rounded-2xl w-90 mx-auto'>

<br>

非法棄置設有罰則。法律嚴禁將垃圾隨意丟棄在山林、道路或他人土地，「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">可處 5 年以下有期徒刑</span>」或「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">1,000 萬日圓以下罰金</span>」，刑責非常沉重。

此外，作為公寓租約糾紛的案例，若多次不遵守垃圾排放規則，在收到鄰居投訴後，也可能發生<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">公寓租約被解除</span>並被要求搬離的情況。

<br>

#### <div class="custom-title">正確的處理方式（重要）</div>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>資源垃圾的「3R」規則</p>

資源垃圾（瓶、罐等）<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">不可以直接</span>丟棄，必須經過以下步驟：

<br>

| Action            | 說明                                                                 |
| :---------------- | :------------------------------------------------------------------- |
| Rinse<br>(沖洗)   | 喝光內容物後，用清水輕輕沖洗（為防止`髒污`與`異味`）。               |
| Remove<br>(拆除)  | 拆除寶特瓶的`瓶蓋`與`塑膠標籤`。<br>將其分類為塑膠資源。             |
| Recycle<br>(分類) | 將`瓶身`、`瓶蓋`、`標籤`分別放入指定的籃子或袋子。                   |
| 注意              | `無法去除髒污`的塑膠（如沾滿油漬的容器）通常作為「`可燃垃圾`」處理。 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>處理危險物品</p>

這是為了<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">防止垃圾車或處理廠發生火災、受傷</span>而制定的極重要規則。

<br>

| 種類           | 排放方式・注意點                                                               |
| :------------- | :----------------------------------------------------------------------------- |
| 噴霧罐・瓦斯罐 | 務必`將內容物用盡`後再排出。請搖晃確認是否還有聲音。                           |
| 電池・蓄電池   | `絕對禁止`混入普通垃圾，這會引發火災。請投入`專用回收箱`（如家電賣場或超市）。 |
| 刀具・碎玻璃   | 用厚紙或報紙牢牢`包裹`，並在袋上大字標註`危險 (キケン)`。                      |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>紙類的捆綁方式</p>

雜誌、報紙、紙箱的基本做法是不裝袋，而是「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">用繩子十字綑綁</span>」後排出。

<br>

| 種類             | 排放方式・注意點                                                   |
| :--------------- | :----------------------------------------------------------------- |
| 紙箱             | 務必先壓至`扁平`後，再用繩子綑綁。                                 |
| 小型紙張（雜紙） | `信封`或`傳單`等可裝入`紙袋`內，或夾在雜誌中間一起綑綁，以免散落。 |

<br>

#### <div class="custom-title">大型垃圾（家具・家電）該如何處理？</div>

家具或家電若只是放在平時的垃圾場，絕對<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">不會被回收</span>。在日本丟棄大型垃圾需要「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">預約</span>」並支付「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">費用（手續費）</span>」。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>家具與小型家電</p>

凡任一邊超過 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">30cm〜50cm</span> 者（如棉被、書桌、椅子、自行車、微波爐等）皆為對象。

| 步驟         | 內容                                                           |
| :----------- | :------------------------------------------------------------- |
| 申請（預約） | 撥打電話或透過網路向各地的「`大型垃圾受理中心`」申請。         |
| 告知內容     | 說明要丟棄物品的`種類`、`尺寸`、`數量`。                       |
| 確認資訊     | 服務人員會告知`清運日期`與`手續費`（金額）。                   |
| 購買處理券   | 前往便利商店或超市購買指定金額的「`大型垃圾處理券`（貼紙）」。 |
| 黏貼貼紙     | 在貼紙上寫下`姓名`或`受理編號`，貼在垃圾顯眼處。               |
| 排出         | 在`指定日`當天早上，放在門口或指定地點（不提供入室搬運）。     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>無法隨意丟棄的家電</p>

以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">4 種</span>家電無法作為一般大型垃圾處理，法律規定必須進行資源回收。

`電冰箱・冷凍庫`、`洗衣機・烘衣機`、`電視`、`冷氣`

<br>

| 情況           | 方式・費用                                                         |
| :------------- | :----------------------------------------------------------------- |
| 購買新品換新時 | 委託購買新產品的店家進行舊家電的`回收`。                           |
| 純粹丟棄時     | 委託原購買店家或居住地的「家電回收協力店」`處理`。                 |
| 費用           | 通常包含「回收費」＋「運送費」，總計約 `3,000 日圓〜10,000 日圓`。 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 '>免費處理掉的方法</p>

| 方法                | 內容                                                              |
| :------------------ | :---------------------------------------------------------------- |
| 學校佈告欄          | 張貼「因回國欲`贈送`冰箱」等訊息。                                |
| 二手商店            | 若品項良好，可能可以獲得`收購`（部分店家提供到府估價）。          |
| Jimoty (ジモティー) | 日本知名的「物品贈送・交換」`App`。若能免費贈出則無需支付處理費。 |

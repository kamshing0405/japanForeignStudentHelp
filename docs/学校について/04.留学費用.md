日本留学を計画する際は、費用を「年度の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">学費</span>（入学金を含む）」と「毎月の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">生活費</span>」の 2 つに分けて考えることをおすすめします。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">費用</div>

#### <div class="custom-title">年度の学費（平均学費）</div>

日本では、年度の学費は主に<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">入学金（初年度のみ）・授業料・施設設備費</span>などで構成されています。
以下は、学校の種類別に見た初年度学費の平均的な目安です。

| 学校の種類             | 入学金（平均）          | 授業料（年額）           | 施設費・その他 | 初年度納入金合計（平均） |
| ---------------------- | ----------------------- | ------------------------ | -------------- | ------------------------ |
| 国立大学               | 282,000 円              | 535,800 円               | ―              | 817,800 円               |
| 公立大学               | 392,391 円              | 538,294 円               | 7,540 円       | 938,225 円               |
| 私立大学（文系）       | 225,386 円              | 813,244 円               | 150,361 円     | 1,188,991 円             |
| 私立大学（理系）       | 251,029 円              | 1,116,880 円             | 198,353 円     | 1,566,262 円             |
| 私立大学（医・歯学系） | 1,073,083 円            | 2,865,067 円             | 952,389 円     | 4,890,539 円             |
| 専門学校               | 約 160,000 円           | 約 710,000 円            | 約 370,000 円  | 約 1,240,000 円          |
| 日本語教育機関         | 約 50,000 ～ 100,000 円 | 約 600,000 ～ 750,000 円 | ―              | 約 700,000 ～ 850,000 円 |

#### <div class="custom-title">毎月の生活費</div>

最新の調査によると、日本で生活する留学生の月平均支出は約 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">148,000 円</span>とされています。

| 項目                                   | 月額目安      |
| -------------------------------------- | ------------- |
| 食費                                   | 約 28,000 円  |
| 住居費（学生寮・シェア）               | 約 38,000 円  |
| 交通費                                 | 約 5,000 円   |
| 水道・光熱費（電気・ガス）             | 約 7,000 円   |
| 保険・医療費                           | 約 3,000 円   |
| 娯楽費・雑費（書籍・衣類・通信費など） | 約 67,000 円  |
| 合計                                   | 約 148,000 円 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>地域による費用の違い</p>

生活費は都市の規模によって大きく異なります。

東京および関東圏は最も<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">高く</span>、東北地方や九州地方は比較的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">安い</span>傾向があります。

| 地域           | 月平均生活費               |
| -------------- | -------------------------- |
| 東京周辺       | 約 `160,000 円`以上        |
| 九州・東北地域 | 約 `110,000 ～ 130,000 円` |

#### <div class="custom-title">収入源：アルバイト</div>

多くの留学生は、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">アルバイト</span>によって生活費の一部を補っています。

| 項目           | 内容                                                                       |
| -------------- | -------------------------------------------------------------------------- |
| 許可申請       | 出入国在留管理局で「`資格外活動許可`」を取得する必要あり                   |
| 労働時間の制限 | `週 28 時間`以内（長期休暇中は 1 日 8 時間以内）                           |
| 想定収入       | 時給 約 `900 ～ 1,200 円`（例：時給 1,100 円 → 月約 123,200 円）           |
| 注意点         | 生活費の補助が目的。学費を全額まかなうのは`困難`なため、過度な依存は`不可` |

<br>

#### <div class="custom-title">節約のポイント</div>

| 節約・支援方法 | 内容                                                                  |
| -------------- | --------------------------------------------------------------------- |
| 学費減免制度   | 国立大学・一部私立大学で、授業料の `30％・50％`、場合により 100％免除 |
| スーパーの割引 | 夕方〜夜に弁当・総菜が「`半額`」になる時間帯があり、食費節約に有効    |
| 学生割引       | 学生証提示で、`電車の定期券`・`映画チケット`などが割引価格に          |

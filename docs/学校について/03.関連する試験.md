日本の高等教育機関へ進学するために、留学生は通常、いくつかの<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">重要な試験</span>を受験する必要があります。これらの試験では、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日本語能力</span>だけでなく、基礎的な<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">学力</span>も評価されます。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">主な試験</div>

#### <div class="custom-title">日本留学試験（EJU）</div>

[日本留学試験](https://www.jasso.go.jp/ryugaku/eju/index.html)は、外国人留学生として、日本の大学（学部）等に入学を希望する者について、日本の大学等で必要とする<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日本語力及び基礎学力</span>の評価を行うことを目的に実施する試験です。

| 項目     | 内容                                         |
| -------- | -------------------------------------------- |
| 実施時期 | 毎年 `6月・11月`                             |
| 日本語   | `読解・聴解（聴読解）・記述`                 |
| 理科     | 物理・化学・生物から `2科目選択`             |
| 総合科目 | `政治・経済・地理・歴史`                     |
| 数学     | 文系用（コース 1）と `理系用（コース2）`     |
| 重要性   | 多くの大学が `合否判定の重要資料` として活用 |

<br>

#### <div class="custom-title">日本語能力試験（JLPT）</div>

世界最大規模の[日本語能力認定試験](https://www.jlpt.jp)で、N1 から N5 までの 5 段階に分かれています。

| 項目         | 内容                                                     |
| ------------ | -------------------------------------------------------- |
| 実施時期     | 毎年 `7月・12月`                                         |
| 進学での利用 | 専門学校は N2 以上、大学院は `N1` が求められることが多い |
| 就職での利用 | 多くの企業で `N1` の取得が望ましい                       |
| メリット     | 有効期限がなく、`国際的に通用する資格`                   |

<br>

#### <div class="custom-title">英語能力試験</div>

有名大学の学部（早稲田・慶應、旧帝国大学など）や、大学院（特に理工系・経済学・医学系）を受験する場合、英語試験の成績提出を求められることが多いです。

| 試験名                                          | 特徴                                                        |
| ----------------------------------------------- | ----------------------------------------------------------- |
| [TOEFL](https://www.toefl-ibt.jp)               | 認知度が高く、`ほぼすべての大学で利用可`                    |
| [TOEIC](https://www.iibc-global.org/toeic.html) | 日本国内で評価が高く、`商学・実務系分野` で有効な場合がある |
| [IELTS](https://www.eiken.or.jp/ielts/)         | 認知度は上昇中だが、日本では `利用範囲がやや限定的`         |

<br>

#### <div class="custom-title">大学別試験（校内考）</div>

書類審査や EJU 選考を通過した後、各大学が独自に実施する試験です。

| 項目     | 内容                                            |
| -------- | ----------------------------------------------- |
| 筆記試験 | 小論文や `専門科目` により論理力・表現力を評価  |
| 面接     | 留学動機・学習計画・`日本語での会話能力` を重視 |
| 重要性   | 合否を左右する `最終選考` として特に面接が重要  |

<br>

#### <div class="custom-title">その他の特別な試験</div>

| 試験名             | 内容                                                 |
| ------------------ | ---------------------------------------------------- |
| 大学入学共通テスト | 多くの留学生は `受験不要` だが、医学系など一部で必要 |
| 芸術系の実技試験   | 美大・音大で `デッサンや演奏` などの実技を実施       |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">受験に関する注意点</div>

日本で受験（特に大学・大学院・専門学校）をする際には、日本独自の習慣やルールがいくつかあります。特に留学生や海外在住の方が間違いやすいポイントをまとめました。

#### <div class="custom-title">書類準備</div>

| 注意点         | 具体内容                                                                       |
| -------------- | ------------------------------------------------------------------------------ |
| 原本の提出     | 卒業証明書・成績証明書は原則「`原本`」を提出。コピー不可の場合が多い           |
| 翻訳文の添付   | 日本語・英語以外の書類には公的な翻訳（`公証`）が必要                           |
| 有効期限の確認 | EJU・TOEFL・IELTS は原則 `2 年間有効`。発行から `3 か月以内指定の書類`にも注意 |
| 名前表記の統一 | パスポート・証明書・成績表で名前の綴りを`一致させる`                           |

<br>

#### <div class="custom-title">出願締切・郵送</div>

| 注意点         | 具体内容                                                         |
| -------------- | ---------------------------------------------------------------- |
| 必着・消印有効 | `必着`：締切日までに大学到着、`消印有効`：締切日の消印があれば可 |
| 郵送方法       | `書留`・`レターパック`など、追跡可能な方法が指定されることが多い |

<br>

#### <div class="custom-title">試験の出願時間</div>

| 時期        | 試験・手続き   | 補足                                                        |
| ----------- | -------------- | ----------------------------------------------------------- |
| 3 月と 9 月 | EJU 出願       | 6 月・11 月実施分                                           |
| 3 月と 8 月 | JLPT 申込み    | 7 月・12 月実施分                                           |
| 通年        | TOEFL<br>TOEIC | 成績証明書の郵送に約 `1 か月`かかるため早めの受験がおすすめ |

<br>

#### <div class="custom-title">EJU 科目選択</div>

| 注意点           | 具体内容                                                       |
| ---------------- | -------------------------------------------------------------- |
| 文系・理系の選択 | 総合科目（`文系`）か理科（`理系`）かを志望校指定どおりに選ぶ   |
| 出題言語         | `日本語`・`英語`の選択を確認（日本語学科は日本語指定が一般的） |

<br>

#### <div class="custom-title">面接マナー</div>

| 注意点   | 具体内容                                                       |
| -------- | -------------------------------------------------------------- |
| 服装     | `スーツ`など`清潔感`のある服装。ジーンズ・サンダルは禁止       |
| 時間厳守 | 会場・オンラインともに `15 分前到着`が基本                     |
| 志望理由 | 大学・学部を`選んだ理由`を具体的かつ論理的に説明できるよう準備 |

<br>

#### <div class="custom-title">受験料支払い</div>

| 注意点     | 具体内容                                         |
| ---------- | ------------------------------------------------ |
| 支払い証明 | 振込証明書・収納証明書を出願書類に`貼り忘れない` |

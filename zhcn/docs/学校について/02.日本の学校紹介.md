<h1 style="text-align: center;">学校选择</h1>

为了让留学取得成功，事前的计划制定非常重要。你是否已经为自己的留学计划做好充分准备了呢？

首先最需要考虑的是<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">教育机关的类别选择</span>。日本的高等教育机关根据“想学什么”以及“未来想朝什么样的职涯发展”，其角色与特征有很大的不同。

因此，在明确留学目的后，选择适合自己的教育机关至关重要。

由于日本的入学考试制度与签证（在留资格）申请手续非常耗时，建议从<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">一年前</span>就开始行动，这样可以增加选择空间并防止失败。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">大学与研究生 (大学院)</div>

#### <div class="custom-title">大学（学部）・短期大学</div>

日本的大学不仅是习得高度<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">专业知识</span>的地方，更是培养自主研究能力与广泛<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">通识教养</span>的高等教育核心机构。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基本概要与修业年限</p>
</div>

| 项目               | 内容                                                            |
| ------------------ | --------------------------------------------------------------- |
| 标准修业年限       | 大多数学部、学科为 `4 年制`                                     |
| 修业年限较长的学部 | 医学、齿学、兽医学，以及部分药学部（药剂师养成课程）为 `6 年制` |
| 学位               | 毕业后将授予`学士`学位                                          |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学资格</p>
</div>

要进入日本的大学就读，必须符合以下任一项<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">学历条件</span>。

| 项目         | 内容                                                                                                 |
| ------------ | ---------------------------------------------------------------------------------------------------- |
| 标准学历     | 在海外完成 `12 年常规学校教育`（高中毕业或具同等学历）                                               |
| 学力认定考试 | 通过被认定与完成 `12 年教育同等学力的考试`（如大学预科课程等）                                       |
| 国际资格     | 拥有国际学士学位（IB）、德国大学入学资格（Abitur）、法国大学入学资格（Baccalauréat）等国际`认可资格` |
| 补充说明     | 若本国学校教育年限`未满 12 年`，原则上需修满指定的“`准备教育课程`”                                   |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学选拔・考试方法</p>
</div>

日本的大学提供多样化的选拔方式，且针对<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">留学生的制度</span>也十分完善。

| 项目                     | 内容                                                                             |
| ------------------------ | -------------------------------------------------------------------------------- |
| 日本留学试验（EJU）      | 许多大学要求留学生提交 `EJU 成绩`，用以评估日语能力及基础学力                    |
| 日语能力试验（JLPT）     | 视大学或学部而定，有时可以提交 `N1 或 N2` 合格证书来代替 EJU 的日语科目          |
| 全英文学位课程（SGU 等） | 提交 `TOEFL・IELTS` 分数、`SAT・ACT`・高考成绩及志愿理由书等，无需在日本参加笔试 |
| 校内选考                 | 参加各大学自行办理的`笔试`（小论文・专业科目）或`面试`                           |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">毕业要求与学分制度</p>
</div>

| 项目           | 内容                                       |
| -------------- | ------------------------------------------ |
| 学分制         | 学生在学期间必须修满大学规定的`必要学分数` |
| 一般学部       | 4 年制大学通常需要修满 `124 学分`以上      |
| 医学・兽医学系 | 6 年制课程通常要求修满 `182 学分`以上      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">短期大学</p>
</div>

对于想要在短时间内习得<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">专业知识</span>与技能，并<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">希望尽快步入社会的人</span>来说，短期大学是一个有力的选择。

| 项目           | 内容                                                                       |
| -------------- | -------------------------------------------------------------------------- |
| 修业年限与目的 | 一般为 `2 年制`（部分医疗领域为 3 年制），重视培养与职业直接相关的实践能力 |
| 学位           | 毕业后将授予`短期大学士`学位                                               |
| 升学之路       | 毕业后除了就业，也可以通过“编入学考试”进入 `4 年制`大学的 `3 年级`就读     |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">6</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">如何选择大学</p>
</div>

| 项目         | 内容                                                                       |
| ------------ | -------------------------------------------------------------------------- |
| 观察大学性质 | 国公立大学学费相对便宜且研究环境充实；私立大学则学部与学科的`选择非常丰富` |
| 考虑授课语言 | 衡量要用日语学习，或是挑战`全英文授课课程`                                 |
| 确认数据资料 | 可利用 `JPSS` 搜索网站确认过去的`合格基准`                                 |

#### <div class="custom-title">研究所 (大学院)</div>

日本的研究所是进行<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">高度学术研究</span>并培养高专业人才的教育机关。根据研究目的与取得学位的不同，分为硕士课程、博士课程及专业职学位课程。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">课程内容与修业年限</p>
</div>

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>硕士课程</p>

| 项目     | 内容                                           |
| -------- | ---------------------------------------------- |
| 修业年限 | 通常为 `2 年`                                  |
| 学位     | 修满学分并通过硕士论文审查后，授予“`硕士`”学位 |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>博士课程</p>

| 项目     | 内容                                               |
| -------- | -------------------------------------------------- |
| 修业年限 | 通常为 `5 年`                                      |
| 构成     | 一般分为前期课程（`2 年`）与后期课程（`3 年`）     |
| 补充     | 医学・齿学・兽医学等 6 年制学部毕业后通常为 `4 年` |
| 学位     | 修满课程后授予“`博士`”学位                         |

<br>

<p class=' text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>专业职学位课程</p>

| 项目     | 内容                                                       |
| -------- | ---------------------------------------------------------- |
| 目的     | 培养如法科大学院、教职大学院、MBA 等具备高`实务能力的人才` |
| 修业年限 | 原则上为 `2 年`（依领域不同可能有所变动）                  |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学资格</p>
</div>

要报考日本的研究所，通常需要符合以下条件：

| 对象课程             | 申请要件                                                                   |
| -------------------- | -------------------------------------------------------------------------- |
| 硕士・专业职学位课程 | 在外国完成 `16 年学校教育`并取得学士学位                                   |
| 博士课程（后期）     | 取得`硕士`学位，或被认定具备同等学力                                       |
| 特殊情况             | 若学制为 `15 年`等情况，可能需要通过个别入学资格审查，或先以研究生身份在校 |
| 语言能力证明         | 提交 `JLPT`、`TOEFL` 或 `IELTS` 等语言考试成绩                             |
| 推荐信               | 提交 1 ～ 2 封由指导教授或院长等撰写的`推荐信`（依大学、研究科而异）       |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">关于“研究生”制度</p>
</div>

日本所称的“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">研究生</span>”与中文里的“硕士班学生”意义不同。日本大学或研究所的“研究生”是指不以取得学位（学士或硕士等）为目的的“非正规生”身份。

| 项目     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 定义     | 不取得学位、`不属于正规课程`的学生制度                       |
| 目的     | 作为进入硕、博士课程前的准备期，主目的通常是应对`入学考试`等 |
| 选考方法 | 多数情况仅通过`书面审查`或与指导教授`面谈`即可获得许可       |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">修了要求与取得学分</p>
</div>

| 课程区分       | 修了条件                                                       |
| -------------- | -------------------------------------------------------------- |
| 硕士课程       | 需修满 `30 学分`以上，完成硕士论文并`通过最终审查与考试`       |
| 专业职学位课程 | 通常需修满 `30 学分`以上，依领域不同可能要求提交实践性课题报告 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">5</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">授课语言</p>
</div>

| 授课形态             | 内容                                                       |
| -------------------- | ---------------------------------------------------------- |
| `日语`授课           | 许多研究科要求具备 N1 等级的日语能力，且需具备学术写作能力 |
| `英语`授课（SGU 等） | 在理工、经济、国际关系等领域设有全英文学位学程             |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">专门学校与专修学校</div>

#### <div class="custom-title">高等专门学校 (高专)</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">对象为初中毕业生</span>

高等专门学校（简称“高专”）是日本<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">独特</span>的高等教育机关。其目的是从<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">早期阶段</span>开始进行专业教育，培养出能应对<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">尖端技术</span>的实践型技术人才。
<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基本概要与定位</p>
</div>

| 项目     | 内容                                                  |
| -------- | ----------------------------------------------------- |
| 教育目标 | 以`工业领域`为中心，重视`实验・实习・实作`的专业教育  |
| 教育特色 | 采行 `5 年`一贯制教育课程，兼顾`专业知识`与`通识`教养 |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">修业年限与留学生升学路径</p>
</div>

| 项目         | 内容                                                       |
| ------------ | ---------------------------------------------------------- |
| 标准修业年限 | 对象为初中毕业生，修业年限通常为 `5 年`（商船系为 5.5 年） |
| 留学生编入学 | 留学生通常以第 `3 学年编入`方式入学，修业年限通常为 `3 年` |
| 专攻科       | 本科毕业后，可进入为期 `2 年`的专攻科进行高度应用研究      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">授予称号与进路</p>
</div>

| 项目     | 内容                                                                |
| -------- | ------------------------------------------------------------------- |
| 准学士   | 完成 `5 年`课程（或留学生 3 年课程）后，授予准学士称号              |
| 学士     | 再完成 `2 年`专攻科课程并通过审查，可取得学士学位                   |
| 主要进路 | 就业评价极高且就业率近 100%，亦可通过编入学进入 ~4 年制大学 3 年级~ |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">入学资格（编入学）</p>
</div>

| 项目     | 内容                                                        |
| -------- | ----------------------------------------------------------- |
| 学历条件 | 在外国完成相当于 ~11 年学校教育~的课程（针对第 3 学年编入） |
| 年龄条件 | 原则上报名时需`满 17 岁`以上                                |
| 日语能力 | 为了修习专业科目，需要具备充分的`日语能力`                  |

#### <div class="custom-title">专修学校 (专门学校)</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">对象为高中毕业生</span>

专修学校在日本高等教育制度中具有<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">显著特征</span>。在专门学校可以学到与<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">职业及实际生活</span>直接挂钩的知识与技能，是培养“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">职人 (专家)</span>”及高技术人才的场所。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">基本概要与定位</p>
</div>

日本的专门学校涵盖广泛领域，整体分为 8 大类。

| 主要领域           | 主要专攻范例                       |
| ------------------ | ---------------------------------- |
| 工业领域           | `IT、汽车维修、建筑、电子`         |
| 农业领域           | `园艺、生物技术、动物照护`         |
| 医疗领域           | `护理、物理治疗、放射线技术`       |
| 卫生领域           | `营养、烹饪、美容、理发`           |
| 教育・社会福利领域 | `幼儿教育、社会工作师`             |
| 商业实务领域       | `酒店、观光、会计、秘书`           |
| 服饰・家政领域     | `服装设计、室内设计`               |

#### <div class="custom-title">日语教育机关</div>

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>

</div>

| 项目         | 内容                                           |
| ------------ | ---------------------------------------------- |
| 定位         | 由学校法人或企业等设立，`最常见`的日语教育机关 |
| 课程特征     | 提供`升学`、`商务`、`文化体验`等多样化课程     |
| 学生在留资格 | 在日本以“`留学`”在留资格就读                   |
| 优点         | 升学辅导`体制完善`，如 EJU 对策与面试指导      |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>

| 项目     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 定位     | 附设于私立大学或短期大学内的`日语预备教育课程`               |
| 课程特征 | 以进入该大学或研究所为目标，教育内容以`日语及基础科目`为中心 |

<div class="flex items-center text-center gap-2 h-6 mt-6">

<div class="flex items-center text-center gap-2 h-6 mt-6">

#### <div class="custom-title">短期课程</div>

“短期留学”是指不以取得学位为目的，以“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">非正规生</span>”身份进行大约<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">数周至一个学年</span>的留学。

学程形式多样，有些设有专为短期留学生开设的特别班级，有些则是修习一般学部或学科内的课程。

短期留学依其特征可分为以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">5 类</span>：

| 区分                 | 概要                                                     | 学费支付对象                         |
| -------------------- | -------------------------------------------------------- | ------------------------------------ |
| 交换留学             | 在签署学生交流协定的大学间留学的制度                     | 原所属学校（留学大学的学费通常免除） |
| “交换留学”以外的课程 | 交流协定外的留学制度，也有个别接收学生的课程             | 留学大学                             |
| 研究生               | 对象为大学毕业生，以研究生身份在校进行一定期间研究的制度 | 留学大学                             |
| 听讲生               | 以“听讲”课程科目为目的的制度                             | 留学大学                             |
| 科目等履修生         | 修习课程科目并以取得学分为目的的制度                     | 留学大学                             |

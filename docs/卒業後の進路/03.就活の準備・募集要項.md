<h2 class=' text-center'> 就活準備</h2>

就活準備の時期は、大学生にとって重要な段階です。一般的に、就活（就職活動）は<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">大学 3 年生の春</span>から始まることが多いですが、準備はそれより前から始めるのが理想です。

ここでは、就活準備を意識し始めた時に取り組んでおきたいことを紹介します。

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">自己分析</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：早めに始めるのが理想</span>

`自己分析`の目的は、自分自身の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">性格や価値観、強み・弱み</span>を客観的に理解することにあります。

これにより、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自分に合った仕事や職場環境を明確</span>にし、将来の方向性を考えるための土台を築くことができます。

また、就職活動においては、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">志望動機や自己 PR </span>を具体的かつ説得力のある内容にするためにも、自己分析は欠かせません。自分の経験を根拠として語ることで、企業に対して<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自分の魅力</span>を的確に伝えることが可能になります。

<br>

 <div class="custom-title">自己分析の方向</div>

| 方向            | 内容例・説明                           | 分析方法                         |
| --------------- | -------------------------------------- | -------------------------------- |
| `長所`          | 自分の強み、得意なこと                 | パーソナル SWOT 分析（Strength） |
| `短所`          | 自分の弱み、改善したい点               | パーソナル SWOT 分析（Weakness） |
| `好き`　`嫌い ` | 好きなこと、苦手なこと                 | 過去の自分史・MBTI               |
| `専門能力`      | 専門知識やスキル、言語能力             | 過去の自分史・RIASEC             |
| `興味`          | 興味・関心のある分野やテーマ           | MBTI                             |
| `将来の夢`      | 将来の目標やなりたい姿                 | Ikigai・未来の自分史             |
| `価値観`        | 仕事や生活で大切にしている考え方や信念 | Ikigai・未来の自分史             |
| `人生観`        | 人生や働き方に対する考え方や方向性     | Ikigai・未来の自分史             |

<br>

::: tip 自己分析の方法

- [SWOT](https://offerbox.jp/columns/42821.html)

(個人の成長のために、自分自身の強み、弱み、機会、脅威を把握する方法です)

- [MBTI](https://www.mbti.or.jp/what/)

（カール・ユングの心理学理論を基にした、性格を理解するための自己分析ツール）

- [RIASEC](https://www.sifiji.org/riasec自己分析ツールを徹底解説就活生が必ずやる/)

（パーソナリティ（性格）と職業（環境）の 6 つのタイプ」分類理論で、自分の興味や強みを知り、適職を見つけるための自己分析ツール）

- [過去の自分史](https://job.rikunabi.com/contents/howto/18838/)

（自分のこれまでの人生を振り返り、年表のように時系列で書き出したものです）

- [生きがい（Ikigai）](https://www.keiro.org/jp/fact-sheet/ikigai-j)

（生きる喜び、生きるはりあい、人生の意味や目的を指す日本語の概念）

- [未来の自分史](https://woman.nikkei.com/atcl/doors/column/19/040300173/040300003/)

（将来どうなっていたいか、どんな人生を歩んでいたいかを想像して書く 自分史です）

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">業界研究</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 3 年生の春</span>

希望する企業選びのための<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">第一歩</span>が「`業界研究`」です

業界研究とは、さまざまな業界の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">特徴や動向</span>を理解し、志望する業界を絞り込むためのプロセスです

まずどのような業界があるのか<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">全体像</span>を把握し、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自己分析を通して、自分の興味・関心が明確</span>になったら、どのような業界が<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自分に合う</span>か調べましょう

<br>

 <div class="custom-title">業界分類</div>

| 業界                   | 主な内容                                   | 役割                                                                     |
| ---------------------- | ------------------------------------------ | ------------------------------------------------------------------------ |
| `メーカー（製造業）`   | 食品、自動車、電子機器、化学、医薬品、建設 | 　モノをつくる仕事                                                       |
| `商社`                 | 総合商社 、専門商社                        | モノ・サービス・情報を国内外でつなぎ、取引を成立させる仕事               |
| `金融`　               | 銀行、証券、保険、クレジット、信用組合     | お金を集める・貸す・増やす・動かす・守る仕事                             |
| `サービス`             | 旅行、ホテル、飲食、不動産、コンサル・運輸 | 売り買いしたあとにモノが残らず、効用や満足などを提供するカタチのない仕事 |
| `流通・小売`           | 百貨店、スーパー、コンビニ、EC（通販）     | 消費者にモノを届ける仕事                                                 |
| `IT・情報・通信`       | ソフトウェア、通信キャリア、Web サービス   | 情報インフラを支える仕事                                                 |
| `広告・出版・マスコミ` | テレビ、新聞、出版、広告                   | 情報を伝える仕事                                                         |
| `官公庁・団体`         | 公務員、非営利団体、インフラ（電気・ガス） | 社会の基盤を支える                                                       |

<br>

::: tip 業界研究に役立つサイト

1. [業種・職種・業態分析](https://www.theport.jp/portcareer/article/79266/#⑤広告・出版・マスコミ)
2. [業界研究のやり方](https://www.onecareer.jp/articles/790)

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">企業研究</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 3 年生の春</span>

企業の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">事業内容・戦略・強み・社風</span>などを調べ、自分との適性や志望理由を明確にするための分析です。

 <div class="custom-title">研究視点</div>

| 視点       | 項目                 | 主な内容                                                |
| ---------- | -------------------- | ------------------------------------------------------- |
| `経営視点` | 経営・理念           | 　企業理念、ビジョン、ミッション、経営方針、社会的役割  |
|            | 事業戦略・業績       | 主力事業、成長戦略、市場での位置づけ、売上・利益の推移  |
| 　         | 経営資源（リソース） | 人材、資金力、技術力、ブランド力、海外展開、IT・DX 基盤 |
| `職場視点` | 人材・組織           | 人材育成制度、研修内容、組織体制、社風、企業文化        |
|            | 仕事                 | 新入社員の業務内容、配属、業務の流れ、裁量の大きさ      |
|            | 評価・処遇           | 評価制度、昇進・昇格、給与・福利厚生、働き方の柔軟性    |

<br>

::: tip 企業研究を行う方法

1. 会社の`公式 SNS`・`ニュースサイト`を調べる
2. 会社に関する`ニュース`を確認する
3. 業界内での`位置`づけを確認する（同業他社との比較）
4. `会社説明会`に参加する
5. `OB・OG 訪問`で生の情報を収集する
6. 採用ページ・`社員インタビュー`を読む
7. `インターンシップ`に参加する

:::

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">職種研究</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 3 年生の春</span>

業界や企業という「場所」ではなく、あなた自身が毎日行う「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">仕事の中身</span>」に焦点を当てる作業。

<br>

<div class="custom-title">職種紹介</div>

| 職種                     | 主な仕事内容                                                   | 求められるスキル・適性                                           |
| ------------------------ | -------------------------------------------------------------- | ---------------------------------------------------------------- |
| `事務・管理系`           | 書類作成、データ入力、経理、人事、総務                         | 正確性、事務処理能力、コミュニケーション力、スケジュール管理能力 |
| `企画系`                 | 商品企画、マーケティング、経営企画、営業企画                   | 分析力、発想力、プレゼン力、課題解決能力                         |
| `技術・研究系`           | 研究開発、製品設計、実験・分析、技術サポート                   | 専門知識、論理的思考力、問題解決力、継続学習力                   |
| `営業・販売・サービス系` | 法人営業、個人営業、販売、接客、カスタマーサポート             | コミュニケーション力、提案力、交渉力、柔軟性                     |
| `医療・福祉系`           | 医師、看護師、介護、リハビリ、福祉サービス                     | 専門知識、対人スキル、責任感、協調性                             |
| `言語系`                 | 翻訳、通訳、語学教師、編集・ライティング                       | 語学力、文章力、正確性、異文化理解                               |
| `金融系`                 | 銀行業務、証券、保険、資産運用、財務分析                       | 数字力、分析力、リスク管理能力、コミュニケーション力             |
| `クリエイティブ系`       | デザイン、映像制作、広告、Web 制作、編集                       | 発想力、表現力、デザインソフト技能、企画力                       |
| `情報処理・通信技術系`   | プログラミング、システム開発、ネットワーク管理、AI・データ分析 | IT スキル、論理的思考力、問題解決能力、チーム協働力              |

<br>

::: tip 職種研究を行う方法

1. https://www.stephouse-recruit.com/article/job-research
2. https://job.mynavi.jp/conts/2027/complete_guide/shokushu/

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">インターンシップ</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活動時期：大学 3 年生から（長期休暇）</span>

学生や求職者が、企業や団体で一定期間実際の仕事を<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">体験する</span>、就活準備に役立つプログラムです

学生が、自分にその仕事に就く<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">能力が備わっているか</span>を見極めることを目的として、自らの専攻や関心分野、将来のキャリアに関連した就業体験を行う活動。

<br>

::: tip インターンシップに参加するメリット

1. 実務`経験`・スキル習得
2. 社会人`マナー`や`職場環境`の理解
3. `ネットワーク形成`（社員や他学生との交流）
4. 自己 PR や志望動機の`材料`

:::

<br>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>インターンシップの種類</p>

| タイプ             | 期間             | 主な内容                                                                     |
| ------------------ | ---------------- | ---------------------------------------------------------------------------- |
| `短期`             | 1 日〜5 日程度   | 会社説明、グループワーク、職場見学など。                                     |
| `中期`             | 1 週間〜2 週間   | 実際の業務の一部を体験したり、課題解決の提案をしたりする。                   |
| `長期`             | 数ヶ月〜1 年以上 | 社員に近い立場で実務を行う（有給の場合が多い）。ベンチャー企業などで一般的。 |
| <br>               |                  |                                                                              |
| インターン         | 開催時期         | 特徴                                                                         |
| `サマーインターン` | 8 月 〜 9 月     | 最も参加者が多い。業界研究の第一歩。                                         |
| `秋・冬インターン` | 10 月 〜 2 月    | 志望企業を絞る時期。選考に直結しやすい。                                     |
| `春インターン`     | 2 月 〜 3 月     | 本選考直前の最終確認。短期集中型が多い。                                     |

留學生在日本打工時，必須遵守「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">法律明定的規則</span>」。若違反這些規則，可能會導致無法更新簽證，甚至被強制遣返回國，請務必多加注意。

#### <div class="custom-title">取得工作的「許可」</div>

留學簽證（在留資格：留學）的核心目的是學習，而非工作。在開始打工前，請務必<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">確認</span>在留卡背面的「資格外活動許可」欄位。

<br>

</div>

<br>

| 項目           | 內容                                                                               |
| :------------- | :--------------------------------------------------------------------------------- |
| 資格外活動許可 | 請確認護照或在留卡的`背面`是否蓋有印章或註記。若無標示，須前往入國管理局進行申請。 |
| 學業優先       | 若學校`缺課過多`，即使持有資格外活動許可，也可能導致簽證難以更新。                 |

<br>

#### <div class="custom-title">絕對遵守「28 小時」限制</div>

法律對於留學生在日本的工作時間有著<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">嚴格的限制</span>。

<br>

| 項目             | 內容                                                                      |
| :--------------- | :------------------------------------------------------------------------ |
| 每週 28 小時以內 | 所有工作的總計時數，`一週不得超過 28 小時`。                              |
| 同時兼差         | 若同時做 `2 份`以上的工作，合計時數也必須在 `28 小時`以內。               |
| 違反風險         | 即使`超過 1 分鐘`也算違法（Overwork），未來可能面臨無法在日本就職的風險。 |
| 長假期間         | 僅限在學校規定的長假期間，可以工作到`每日 8 小時（每週 40 小時）`。       |

<br>

#### <div class="custom-title">禁止工作的場所（禁止從事風俗營業）</div>

法律明文<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">禁止</span>留學生在以下場所工作。即使只是負責清潔或洗碗也<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">絕對不可以</span>。

<br>

`柏青哥店`、`遊戲中心 (Game Center)`、`小酒館 (Snack)`、`酒店 (Cabaret)`、`男模俱樂部 (Host Club)`、`酒吧`（以提供酒類為主且須接待客人的情況）、`麻將店`等。

<br>

在這些場所工作，即使持有資格外活動許可也屬違法，且會面臨「`強制遣返（回國）`」等非常<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">嚴厲的處罰</span>。

<br>

#### <div class="custom-title">如何尋找打工機會</div>

大多數留學生會透過以下管道尋找工作：

<br>

| 尋找方式      | 內容                                                                                        |
| :------------ | :------------------------------------------------------------------------------------------ |
| 求職 APP      | 著名的有 `Townwork`、`Baitoru`、`Mynavi Baito` 等。<br>可使用「歡迎留學生」作為關鍵字搜尋。 |
| 店面張貼海報  | 查看`便利商店`或`餐廳`門口的「Staff 募集」海報，並`直接致電`詢問。                          |
| 學校介紹      | 大學的`職涯中心`或`事務局`有時會提供針對留學生的求職資訊。                                  |
| SNS・友人介紹 | 透過`學長姐`或`朋友`介紹，是相對安心的方法。                                                |

<br>

#### <div class="custom-title">錄取後應確認的事項</div>

<br>

| 確認要點 | 內容                                                   |
| :------- | :----------------------------------------------------- |
| 時薪     | 確認是否未低於所居住地區的「`最低工資`」。             |
| 交通費   | 確認是否會支付補貼`電車`或`巴士`的`交通費用`。         |
| 發薪日   | 確認`每月幾號`發薪，以及領取`方式`（通常為銀行轉帳）。 |

<h1 style="text-align: center;">公司雇佣制度</h1>

在求职活动中，与薪资及业务内容同样重要、甚至更值得关注的就是“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">公司雇佣制度</span>”。这不仅是公司的内部规则，更是塑造劳动者`生活稳定`、`技能成长`及`未来选择`的基石。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">雇佣形态</div>

雇佣形态是指企业与员工之间签署的劳动合同种类。

- 雇佣期限依雇佣形态而异。
- 在薪资、福利待遇等方面存在差异。
- 工作时间与工作方式也有所不同。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8 text-center'>概览说明</p>

| 雇佣形态   | 雇佣期限                | 社会信用・稳定性 | 薪资・待遇               | 责任范围       |
| ---------- | ----------------------- | ---------------- | ------------------------ | -------------- |
| 正式员工   | 无固定期限（至退休）    | ✓✓✓✓✓            | 高（有调薪・奖金）       | 广（责任较重） |
| 合同工     | 有固定期限（最长 3 年） | ✓✓✓              | 中等（多无奖金）         | 限定・明确     |
| 派遣员工   | 有固定期限（最长 3 年） | ✓✓               | 时薪制（由派遣公司支付） | 指示范围内     |
| 兼职／打工 | 有期／无期              | ✓                | 时薪制（排班制）         | 辅助・限定     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>正规雇佣（正式员工/正社员）</p>

| 项目               | 内容                       |
| ------------------ | -------------------------- |
| 雇佣形态           | `正式员工`（正规雇佣）     |
| 雇佣期限           | 无`特定期限`（无固定期限） |
| 工作时间           | 一般为`全职`（Full-time）  |
| 薪资・待遇         | 薪资、奖金、调薪`制度完善` |
| 社会保险・福利待遇 | 非常`完善`                 |
| 稳定性             | `高`                       |
| 责任・业务范围     | 相对`较广`                 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>合同工（契约社员）</p>

| 项目               | 内容                                     |
| ------------------ | ---------------------------------------- |
| 雇佣形态           | `合同工`（有期雇佣）                     |
| 雇佣期限           | `有期`（原则最长 `3 年`，不保证续约）    |
| 工作内容・职务范围 | `明确规定`                               |
| 薪资・待遇         | 通常比正式员工较为`受限`                 |
| 转正机会           | 根据`实绩`与`考核`，有`可能`转为正式员工 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>派遣员工</p>

| 项目               | 内容                                 |
| ------------------ | ------------------------------------ |
| 雇佣形态           | `派遣员工`                           |
| 雇佣合同           | 与`派遣公司`签署                     |
| 工作地点           | 实际工作地点为`用工企业`（要派企业） |
| 雇佣期限           | 原则最长 `3 年`                      |
| 工作时间・业务内容 | 听从`用工企业`的指示                 |
| 薪资               | 由`派遣公司`支付                     |
| 担当业务           | 多为`专业性`或`辅助性`业务           |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>兼职 / 打工 (Part-time)</p>

| 项目               | 内容                               |
| ------------------ | ---------------------------------- |
| 雇佣形态           | 兼职、打工（非正规雇佣）           |
| 工作形态           | `短时间`工作、每周劳动天数较少     |
| 工作目的           | 多为`兼顾`家庭或学业               |
| 雇佣合同           | 多为`有期合同`                     |
| 福利待遇・社会保险 | 依`工作时数`或`天数`有适用条件限制 |
| 责任・业务范围     | 相对较为`限定`                     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>不同雇佣形态的福利、责任、调薪与稳定性对照</p>

| 项目       | 正式员工                   | 合同工               | 派遣员工                     | 兼职员工                   |
| ---------- | -------------------------- | -------------------- | ---------------------------- | -------------------------- |
| 福利       | 社会保险・福利待遇`完善`   | 有`部分`福利待遇     | `视派遣公司而定`，多为`受限` | 依条件部分适用             |
| 责任       | 对整体业务与成果的`责任大` | 对所担负`业务负责`   | 业务范围`受限`，责任`较轻`   | 业务范围`受限`，责任`较轻` |
| 晋升・调薪 | 制度`完善`                 | `有限`，有转正可能性 | 基本`无`                     | 基本`无`                   |
| 雇佣稳定性 | `高`                       | `中等`               | `低`                         | `低`                       |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">招聘形态（招募类别）</div>

招聘形态是指招募时考虑到职种及未来职业发展路径的招募框架。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>综合职</p>

| 项目     | 内容                                                     |
| -------- | -------------------------------------------------------- |
| 招募前提 | 以未来担任`管理职`或经历`广泛职种`为前提进行招募         |
| 定岗单位 | 入职时`不固定`部门，通常会经历业务、企划、管理等多个部门 |
| 职业路径 | 较易成为升迁、晋级对象，职业发展路径`明确`               |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>一般职</p>

| 项目           | 内容                                     |
| -------------- | ---------------------------------------- |
| 招募前提       | 以专门从事`特定业务`或职种为前提进行招募 |
| 定岗・业务范围 | `相对固定`，转调或部门异动的可能性较少   |
| 职业路径       | 升迁、晋级范围较为`受限`，不如综合职广泛 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>技术职</p>

| 项目           | 内容                                                 |
| -------------- | ---------------------------------------------------- |
| 招募前提       | 发挥`专业知识`或`技能`进行业务的招募形态             |
| 定岗・业务范围 | 多限定于`研究`、`开发`、`设计`、`IT・工程`等特定领域 |
| 升迁・调薪     | 多根据`技术能力`或成果进行评定                       |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">培训（员工进修）</div>

培训是指企业为了让从新入职员工到管理岗位的各阶层员工，有系统且高效率地习得业务所需知识与技能，而实施的组织性教育计划。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>OJT（On the Job Training，在职培训）</p>

| 项目       | 内容                                         |
| ---------- | -------------------------------------------- |
| 特征       | 在`工作现场`通过实际业务进行的实践性教育训练 |
| 指导方式   | 由`资深员工`或`上司`直接指导，习得技能与知识 |
| 目的・效果 | 习得与`实务直接挂钩`的技能与问题解决能力     |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>Off-JT（Off-the-Job Training，脱产培训）</p>

| 项目     | 内容                           |
| -------- | ------------------------------ |
| 特征     | `离开`工作岗位进行的培训       |
| 学习内容 | 学习`行业知识`或`特定专业技能` |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>集合培训（集训）</p>

| 项目     | 内容                                                     |
| -------- | -------------------------------------------------------- |
| 参加者   | `全体新入职员工`                                         |
| 学习内容 | 习得作为职场人士所需的`基础知识`、`商务礼仪`及`企业文化` |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>导师制度 (Mentor)</p>

| 项目       | 内容                                                             |
| ---------- | ---------------------------------------------------------------- |
| 角色       | 由`资深员工`（导师）对年轻员工或新员工（学员）进行`建议`与`指导` |
| 效果・优点 | 建立安心`可咨询`的环境，`加速对业务的理解`                       |

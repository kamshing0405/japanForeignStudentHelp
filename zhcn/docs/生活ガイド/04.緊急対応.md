「紧急应对」是指针对<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">火灾、自然灾害、事故、急病</span>或重大商务障碍等威胁生命财产安全、且「刻不容缓」的事态，采取迅速且适当行动的过程。

#### <div class="custom-title">紧急求助电话（24 小时・免费）</div>

首先请务必记住这 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">3 个</span>号码。

| 号码  | 内容                     | 使用方法・备注                                   |
| :---- | :----------------------- | :----------------------------------------------- |
| 110   | 警察（刑事案件・事故）   | 遭遇`犯罪`、`遭小偷`、发生`交通事故`等。         |
| 119   | 消防・急救（火灾・急病） | 接通后会询问是`火灾 (Kaji)`还是`救急 (Kyukyu)`。 |
| #7119 | 急救咨询中心             | 犹豫`是否该叫救护车`时可以拨打咨询。             |

#### <div class="custom-title">灾害（地震・台风）准备</div>

日本是一个<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地震与台风</span>频繁的国家。请事先做好以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">3 点</span>准备。

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">1</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">确认避难场所</p>
</div>

<br>

<img src='../../../images/hinansyo.jpg' class='rounded-2xl w-136'>

请事先透过防灾地图 (Hazard Map) 确认住家或学校附近的「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">避难所</span>」。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">2</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">防灾避难包</p>
</div>

<br>

<img src='../../../images/bag.avif' class='rounded-2xl w-136'>

请在后背包内准备好`饮用水`、`干粮`（3 天份）、`移动电源`、`在留卡影本`、`现金`、`常备药`等物品。

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">3</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">资讯收集 APP</p>
</div>

除了准备防灾物资，下载防灾 APP 以便随时<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">收集最新资讯</span>也是非常重要的。

| 服务名称                                                                  | 特征                                           |
| :------------------------------------------------------------------------ | :--------------------------------------------- |
| [Safety Tips](https://www.jnto.go.jp/safety-tips/eng/app.html)            | 观光厅监修，提供多语系的地震速报与避难建议通知 |
| [NHK WORLD-JAPAN](https://www3.nhk.or.jp/nhkworld/en/multilingual_links/) | 提供英文等各国语言的灾害新闻转播               |
| [Yahoo! 防灾速报](https://emg.yahoo.co.jp)                                | 可透过推播接收地震、台风、豪雨等灾害资讯       |
| [JMA（气象厅）APP](https://www.data.jma.go.jp/suishin/info/keitai.html)   | 日本气象厅官方提供，包含地震、海啸、台风资讯   |

<br>

<div class="flex items-center text-center gap-2 h-6 mt-6">
    <span class="flex items-center justify-center w-6 h-6 bg-[#00873c] text-white rounded-full text-sm font-bold shrink-0">4</span>
    <p class="text-[#00873c] font-bold text-lg pl-0.5">灾害用留言专线（171）</p>
</div>

当发生大规模灾害导致电话难以接通时，可用来向<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">家人或友人</span>告知平安。

- 录音：拨打 `171` → 按 `1` → 输入`自己的电话号码` → 录制留言。
- 播放：拨打 `171` → 按 `2` → 输入`对方的电话号码` → 听取留言。

<br>

#### <div class="custom-title">夜间・假日的急病处理</div>

若在夜间或假日突发急病，首先请务必记住这 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 个</span>咨询号码。

| 号码  | 咨询窗口         | 内容・角色                                     |
| :---- | :--------------- | :--------------------------------------------- |
| #7119 | 急救安心中心     | 由护理师针对是否需要就医及应对方法提供建议。   |
| #8000 | 儿童急救电话咨询 | 由小儿科医师或护理师针对儿童突发疾病提供咨询。 |

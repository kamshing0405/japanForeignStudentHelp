日本へ留学するためには、自国政府が発行するパスポートと、日本政府（大使館・領事館）が発給する<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">ビザ</span>の 2 つが必要です。
留学生が取得する在留資格は、通常 「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">留学</span>」 です。留学の場合、観光とは異なり「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">在留資格認定証明書（COE）</span>」の取得が必須となります。

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">在留資格</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">合計で 2 ヶ月〜4 ヶ月程度</span>

留学ビザを申請する一般的な手続きの流れは、次のとおりです。

| 手続きの流れ   | 内容                                                                                  |
| -------------- | ------------------------------------------------------------------------------------- |
| 入学許可の取得 | 日本の教育機関から`合格通知`（入学許可書）を受け取る                                  |
| COE 申請       | 学校が`代理`で入管へ`在留資格認定証明書`を申請 <br>（審査に 1〜3 ヶ月程度かかります） |
| COE 交付       | 審査後、`COE` が発行され本人に郵送される                                              |
| ビザ申請       | COE など必要書類をそろえ、`在外公館`で申請                                            |
| 来日           | ビザ取得後、COE `有効期間`内に入国                                                    |

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>COE（在留資格認定証明書）の重要性</p>

| 項目     | 内容                                                                                                                                         |
| -------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| 役割     | 申請者が`学習計画・経済能力`など入国条件を満たしていることを証明する書類。COE を提出してビザ申請を行うことで、審査期間が大幅に`短縮される`。 |
| 有効期間 | ビザ発給後、原則として`3 か月以内に入国`する必要がある。期限を過ぎると COE が`無効`になる場合がある。                                        |

<br>

#### <div class="custom-title">在留カードの交付</div>

| 項目       | 内容                                                                                         |
| ---------- | -------------------------------------------------------------------------------------------- |
| 上陸許可   | `空港`での入国審査時に、パスポートへ「`上陸許可`」のシールが貼付される。                     |
| 在留カード | 新千歳、成田、羽田、中部、関西、広島、福岡などの主要空港では、入国時にその場で`交付される`。 |
| 重要事項   | 在留カードは日本での身分を証明する`唯一の公的書類`。常に携帯する`義務`がある。               |

<br>

#### <div class="custom-title">入国後に必要な届出</div>

| 項目         | 内容                                                                                                                 |
| ------------ | -------------------------------------------------------------------------------------------------------------------- |
| 住所登録     | 住居が決まり次第、`14 日以内`に在留カードを持参し、居住地の`市区町村役場（区役所・市役所）`で住所登録を行う。        |
| マイナンバー | 住所登録後、後日「`マイナンバー通知書`」が郵送される。`銀行口座開設`や`アルバイト`の`税務手続`きなどで`必要`となる。 |

<br>

#### <div class="custom-title">資格外活動許可（アルバイト許可）</div>

| 項目     | 内容                                                                                               |
| -------- | -------------------------------------------------------------------------------------------------- |
| 原則     | 在留資格「`留学`」では、`原則として就労は禁止`されている。                                         |
| 申請     | `アルバイト`を行う場合は、`資格外活動許可`の申請が必要。                                           |
| 時間制限 | `週 28 時間以内`<br>夏休みなどの長期休暇中は`1日 8 時間以内`。                                     |
| ポイント | 入国時の上陸審査で申請書を提出すれば、`その場で許可を受けられる`ため、後日入管へ行く手間を省ける。 |

为了进入日本的高等教育机关进修，留学生通常需要参加几项<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">重要考试</span>。这些考试不仅评估<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日语能力</span>，也会评估基础的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">学力</span>。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本留学试验（EJU）</div>

#### <div class="custom-title">日本留学试验（EJU）</div>

[日本留学试验](https://www.jasso.go.jp/ryugaku/eju/index.html)是以希望作为外国留学生进入日本大学（学部）等就读的人为对象，旨在评估其是否具备日本大学等机关所要求的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">日语能力及基础学力</span>。

| 项目     | 内容                                       |
| -------- | ------------------------------------------ |
| 实施时期 | 每年 `6 月・11 月`                         |
| 日语     | `读解`、`听解`（听读解）、`记述`（作文）   |
| 理科     | 从物理、化学、生物中选择 `2 科目`          |
| 综合科目 | `政治、经济、地理、历史`                   |
| 数学     | 分为`文科用`（课程 1）与`理科用`（课程 2） |
| 重要性   | 许多大学将其活用为`判定合格与否的重要资料` |

<br>

#### <div class="custom-title">日语能力试验（JLPT）</div>

这是世界规模最大的[日本语能力认定试验](https://www.jlpt.jp)。分为从 N1 到 N5 的 5 个等级。

| 项目     | 内容                                            |
| -------- | ----------------------------------------------- |
| 实施时期 | 每年 `7 月・12 月`                              |
| 升学用途 | 专门学校通常要求 `N2 以上`，研究所则多要求 `N1` |
| 就业用途 | 许多企业希望应征者取得 `N1` 等级                |
| 优点     | 证照`没有有效期限`，是国际通用资格              |

<br>

#### <div class="custom-title">英语能力考试</div>

若要报考著名大学的学部（如早稻田、庆应、旧帝国大学等）或研究所（特别是理工、经济、医学系），通常会被要求提交英语考试成绩。

| 考试名称                                        | 特征                                         |
| ----------------------------------------------- | -------------------------------------------- |
| [TOEFL](https://www.toefl-ibt.jp)               | 知名度最高，几乎`所有大学皆可采计`           |
| [TOEIC](https://www.iibc-global.org/toeic.html) | 在日本国内评价高，在 `商学、实务领域` 较有效 |
| [IELTS](https://www.eiken.or.jp/ielts/)         | 知名度上升中，但在日本 采计范围`相对有限`    |

<br>

#### <div class="custom-title">大学校内考（校内考）</div>

通过书面审查或 EJU 选拔后，由各大学独自实施的考试。

| 项目   | 内容                                         |
| ------ | -------------------------------------------- |
| 笔试   | 透过小论文或`专业科目`评估逻辑能力与表达能力 |
| 面试   | 重视`留学动机`、`学习计划`及`日语会话能力`   |
| 重要性 | 作为决定录取与否的`最终选拔`，面试尤其重要   |

<br>

#### <div class="custom-title">其他特殊考试</div>

| 考试名称         | 内容                                             |
| ---------------- | ------------------------------------------------ |
| 大学入学共通考试 | 多数留学生`无需参加`，但医学系等部分学科可能需要 |
| 艺术类术科测验   | 美大、音大会进行`素描`、`演奏`等术科检定         |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">考试相关注意事项</div>

在日本参加考试（特别是大学、研究所、专门学校）时，有一些日本特有的习惯与规则。我们总结了留学生或海外居住者容易出错的重点。

#### <div class="custom-title">文件准备</div>

| 注意事项     | 具体内容                                                                       |
| ------------ | ------------------------------------------------------------------------------ |
| 提交正本     | 毕业证书、成绩单原则上需提交“`正本`”。多数情况不接受影本                       |
| 附上翻译本   | 若文件非日文或英文，需附上官方翻译（`公证`）                                   |
| 确认有效期限 | EJU、TOEFL、IELTS 原则上 `2 年有效`。也要注意需提交发行 `3 个月内的文件之规定` |
| 统一姓名拼音 | 确保护照、证照、成绩单上的姓名拼写`完全一致`                                   |

<br>

#### <div class="custom-title">出愿截止・邮寄</div>

| 注意事项       | 具体内容                                                             |
| -------------- | -------------------------------------------------------------------- |
| 必着・消印有效 | `必着`：需于截止日前送达大学；`消印有效`：只要有截止日当天的邮戳即可 |
| 邮寄方法       | 通常指定使用挂号（`书留`）、`Letter Pack` 等可追踪进度的邮寄方式     |

<br>

#### <div class="custom-title">考试报名时间</div>

| 时期        | 考试・手续  | 补充说明                                  |
| ----------- | ----------- | ----------------------------------------- |
| 3 月与 9 月 | EJU 报名    | 针对 `6 月、11 月`实施之考试              |
| 3 月与 8 月 | JLPT 报名   | 针对 `7 月、12 月`实施之考试              |
| 全年        | TOEFL/TOEIC | 成绩单邮寄约`需 1 个月`，建议尽早参加考试 |

<br>

#### <div class="custom-title">EJU 科目选择</div>

| 注意事项   | 具体内容                                                 |
| ---------- | -------------------------------------------------------- |
| 文理科选择 | 依照志愿学校的要求选择综合科目`文科`或`理科`             |
| 出题语言   | 确认选择`日语`或`英语`（日语学科一般指定必须选日语出题） |

<br>

#### <div class="custom-title">面试礼仪</div>

| 注意事项 | 具体内容                                            |
| -------- | --------------------------------------------------- |
| 服装     | 穿着`西装`等具备`整洁感的服装`。严禁牛仔裤、凉鞋    |
| 守时     | 无论现场或线上，基本要求为 `15 分钟前`抵达          |
| 志愿理由 | 准备好能具体且具逻辑性地说明`选择该大学/学部的理由` |

<br>

#### <div class="custom-title">报名费支付</div>

| 注意事项 | 具体内容                                       |
| -------- | ---------------------------------------------- |
| 支付证明 | 切勿忘记将汇款证明或缴费证明`粘贴于报名文件上` |

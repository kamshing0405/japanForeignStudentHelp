<h2 class=' text-center'> 求职准备</h2>

求职准备时期对大学生来说是至关重要的阶段。一般而言，求职活动（就活）多从<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">大学三年级的春季</span>开始，但理想的情况是比这更早便展开准备。

在此介绍当您开始意识到需要准备求职时，应该着手进行的事项。

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">自我分析</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：理想是尽早开始</span>

“自我分析”的目的是客观地理解自己的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">性格、价值观、优势与劣势</span>。

通过这项工作，能<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">明确适合自己的工作与职场环境</span>，并为思考未来的发展方向奠定基础。

此外，在求职活动中，为了让<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">应聘动机与自我推荐（Self PR）</span>的内容更具体且具说服力，自我分析是不可或缺的。通过以自身经验作为根据来陈述，才能精准地向企业传达<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">自己的魅力</span>。

<br>

 <div class="custom-title">自我分析的方向</div>

| 方向                     | 内容范例・说明               | 分析方法                       |
| ------------------------ | ---------------------------- | ------------------------------ |
| `长处`                   | 自己的优势、擅长的事         | 个人 SWOT 分析（Strength）     |
| `短处`                   | 自己的弱点、想改善的点       | 个人 SWOT 分析（Weakness）     |
| `喜欢的事物・讨厌的事物` | 喜欢的事、不擅长的事         | 过去的“自我史”・MBTI           |
| `专业能力`               | 专业知识、技能、语言能力     | 过去的“自我史”・RIASEC         |
| `兴趣`                   | 感兴趣的领域或主题           | MBTI                           |
| `未来梦想`               | 未来的目标或理想模样         | Ikigai（生存意义）・未来自我史 |
| `价值观`                 | 工作或生活中重视的思想与信念 | Ikigai・未来自我史             |
| `人生观`                 | 对人生或工作方式的想法与方向 | Ikigai・未来自我史             |

<br>

::: tip 自我分析的方法

- [SWOT](https://offerbox.jp/columns/42821.html)
  （掌握自身的优势、劣势、机会与威胁，促进个人成长的方法）

- [MBTI](https://www.mbti.or.jp/what/)
  （基于荣格心理学理论，用以理解性格的自我分析工具）

- [RIASEC](https://www.sifiji.org/riasec自己分析ツールを徹底解説就活生が必ずやる/)
  （“人格与职业环境的 6 种类型”分类理论，寻找适合职业路径的工具）

- [过去的自我史](https://job.rikunabi.com/contents/howto/18838/)
  （回想至今的人生，按时间顺序记录下来的个人年表）

- [生存意义 (Ikigai)](https://www.keiro.org/jp/fact-sheet/ikigai-j)
  （指生存的喜悦、动力、人生的意义或目的之日式概念）

- [未来自我史](https://woman.nikkei.com/atcl/doors/column/19/040300173/040300003/)
  （想象将来想变成什么样子、过什么样的人生而写下的剧本）

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">行业研究（产业研究）</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学三年级春季</span>

挑选心仪企业的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">第一步</span>就是“行业研究”。

行业研究是指理解各种产业的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">特征与趋势</span>，进而缩小目标行业范围的过程。

首先应掌握产业有哪些种类的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">全貌</span>，并在<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">通过自我分析明确兴趣</span>后，再调查哪种行业<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">适合自己</span>。

<br>

 <div class="custom-title">行业分类</div>
 
| 产业 | 主要内容 | 角色 |
|------|----------|------|
| `制造业` | 食品、汽车、电子设备、化学、医药、建设 | 制作物品的工作 |
| `商社` | 综合商社、专业商社 | 在国内外串联物品、服务、信息，促成交易的工作 |
| `金融` | 银行、证券、保险、信用卡、信用合作社 | 筹集、贷款、增值及保护资金的工作 |
| `服务业` | 旅游、酒店、餐饮、房地产、顾问、运输 | 提供效用或满意度等无形服务的工作 |
| `流通・零售` | 百货公司、超市、便利店、电商 (EC) | 将物品送达消费者手中的工作 |
| `IT・信息・通信` | 软件、通信运营商、Web 服务 | 支撑信息基础设施的工作 |
| `广告・出版・媒体` | 电视、报纸、出版、广告 | 传达信息的工作 |
| `政府部门・团体` | 公务员、非营利团体、基础设施（电、气） | 支撑社会基石的工作 |

<br>

::: tip 有助于行业研究的网站

1. [业种・职种・业态分析](https://www.theport.jp/portcareer/article/79266/#⑤广告・出版・マスコミ)
2. [行业研究的做法](https://www.onecareer.jp/articles/790)

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">企业研究</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学三年级春季</span>

针对企业的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">业务内容、战略、优势、公司风气</span>等进行调查，是为了明确自己与企业的契合度及应聘理由。

 <div class="custom-title">研究视角</div>

| 视角       | 项目           | 主要内容                                            |
| ---------- | -------------- | --------------------------------------------------- |
| `经营视角` | 经营・理念     | 企业理念、愿景、使命、经营方针、社会角色            |
|            | 业务战略・业绩 | 主力业务、成长战略、市场定位、营收・利润推移        |
|            | 经营资源       | 人才、资金力、技术力、品牌力、海外拓展、IT・DX 基础 |
| `职场视角` | 人才・组织     | 人才培养制度、研修内容、组织体制、社风、企业文化    |
|            | 工作内容       | 新员工的业务内容、分配、业务流程、权限大小          |
|            | 考核・待遇     | 评价制度、晋升・职级、薪资・福利、工作灵活性        |

<br>

::: tip 进行企业研究的方法

1. 调查公司的`官方 SNS` 与`新闻网站`
2. 确认与公司相关的`新闻报道`
3. 确认在行业内的`定位`（与同行企业比较）
4. 参加`公司说明会`
5. 通过 OB・OG 访问 搜集`一手信息`
6. 阅读`招聘页面`与`员工访谈`
7. 参加`实习` (Internship)

:::

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">职种研究（岗位研究）</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学三年级春季</span>

这项工作并非聚焦于行业或企业等“场所”，而是专注于您每天实际执行的“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">工作内容</span>”。

<br>

<div class="custom-title">职种介绍</div>

| 职种                   | 主要工作内容                               | 要求的技能・契合度                            |
| ---------------------- | ------------------------------------------ | --------------------------------------------- |
| `事务・管理类`         | 文件制作、资料录入、会计、人事、总务       | 精确性、事务处理能力、沟通能力、进度管理能力  |
| `企划类`               | 商品企划、市场营销、经营企划、营业企划     | 分析力、构思力、演示力、问题解决能力          |
| `技术・研究类`         | 研究开发、产品设计、实验・分析、技术支持   | 专业知识、逻辑思考力、问题解决力、持续学习力  |
| `营业・销售・服务类`   | 法人营业、个人营业、销售、接待、客户支持   | 沟通能力、提案力、谈判力、灵活性              |
| `医疗・福祉类`         | 医师、护士、护理、康复、福祉服务           | 专业知识、人际技能、责任感、协调性            |
| `语言类`               | 翻译、口译、语言教师、编辑・写作           | 语言能力、写作能力、精确性、异文化理解        |
| `金融类`               | 银行业务、证券、保险、资产运用、财务分析   | 数字敏感度、分析力、风险管理能力、沟通能力    |
| `创意类`               | 设计、影像制作、广告、Web 制作、编辑       | 构思力、表现力、设计软件技能、企划力          |
| `信息处理・通信技术类` | 程序开发、系统开发、网络管理、AI・数据分析 | IT 技能、逻辑思考力、问题解决能力、团队协作力 |

<br>

::: tip 进行职种研究的方法

1. [职种研究的做法](https://www.stephouse-recruit.com/article/job-research)
2. [职种研究 彻底指南](https://job.mynavi.jp/conts/2027/complete_guide/shokushu/)

:::

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">实习 (Internship)</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学三年级起（长假期间）</span>

这是让学生在企业或团体中，在一定期间内<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">体验实际工作</span>，对求职准备非常有帮助的计划。

目的在于让学生评估自己是否具备从事该工作所需的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">能力</span>，并进行与自身专业、兴趣或未来职业相关的就业体验活动。

<br>

::: tip 参加实习的优点

1. 获得`实务经验`与`技能`
2. 理解职场人士的`礼仪`与`工作环境`
3. 建立`人脉`（与员工或其他学生交流）
4. 作为自我推荐与应聘动机的`素材`

:::

<br>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>实习的种类</p>

| 类型       | 期间            | 主要内容                                                     |
| ---------- | --------------- | ------------------------------------------------------------ |
| `短期`     | 约 1 至 5 天    | 公司介绍、小组讨论、职场参观等。                             |
| `中期`     | 1 至 2 周       | 体验部分实际业务，或针对课题提出解决方案。                   |
| `长期`     | 数月至 1 年以上 | 以接近员工的身份执行实务（通常有薪）。在创业公司中较为常见。 |
| ------     | ------          | ----------                                                   |
| `夏`季实习 | 8 月 〜 9 月    | 参加者最多。行业研究的第一步。                               |
| `秋`冬实习 | 10 月 〜 2 月   | 缩小目标企业范围的时期。较容易与正式选考挂钩。               |
| `春`季实习 | 2 月 〜 3 月    | 正式选考前的最终确认。多为短期集中型。                       |

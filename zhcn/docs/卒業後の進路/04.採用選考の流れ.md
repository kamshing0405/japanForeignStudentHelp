采用选考是企业从应聘候选人中找出适合该职位人才的过程。通过结合<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">材料审核、笔试、面试、小组讨论（Group Work）、适性检查</span>等多元手段，评估应聘者的能力、契合度与意愿。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">Entry（意向登记/报名）</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学四年级春季</span>

“Entry”是指向企业表达“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">我有兴趣</span>”的意愿表示。

若要参加企业的选考，通常<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">必须</span>先完成 Entry。

就职活动中的 Entry 分为“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">Pre-Entry（初步登记）</span>”与“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">Entry（正式报名）</span>”两个阶段，请务必确认两者的差异。

<div class="flex relative">

<img src='../../../images/text_entrance.png' class='absolute w-46 right-25 top-5 opacity-20'>

::: tip Pre-Entry（初步登记）

- 在求职网站或企业招聘页面注册`姓名`与`大学名称`。
- 企业会为您建立专属的“`My Page（个人主页）`”，并发送说明会通知与选考日程表。

<b>Entry（正式报名）</b>

- 为了`实际参加采用选考`（面试等）而进行的申请。

- 须提交`报名表`（Entry Sheet, ES）、`参加适性检查`（如 SPI 等）、上传`个人简历`。

- 完成后才能进入`材料审核`阶段。

:::

</div>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">公司说明会</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学四年级春季</span>

<div class="flex justify-center">
  <img src='../../../images/seminar.png' class='rounded-2xl w-80'>
</div>
<br>

公司说明会是亲自感受“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">公司内部氛围</span>”与“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">具体工作内容</span>”的宝贵机会，这些是仅看企业官网或手册无法获知的细节。

<br>

<br>

<div class="custom-title">公司说明会的种类</div>

公司说明会主要分为以下两类：
<br>

<div class='flex  gap-2 h-60'>
<div class='flex-1 h-full'>

::: tip 公司说明会（单独型）

- 由<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">单一</span>企业自行举办
- 能深度了解该<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">企业</span>
- 通常需要<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">事先预约</span>

:::

</div>

<div class='flex-1'>

::: tip 联合企业说明会

- <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">众多</span>企业聚集在同一个会场
- 能增加对该<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">产业</span>的知识
- 可以在不同企业间进行<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">比较</span>
  :::

</div>
</div>

<div class="flex justify-center">
  <img src='../../../images/online_school_girl.png' class='rounded-2xl w-80'>
</div>
<br>

最近出现了许多“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">线上</span>”模式，以下总结了各类型的特征与检查重点。

<br>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>录播型 (On-demand)</p>

- 播放事先录制好的影片的公司说明会
- 可以在自己方便的时间收看

<br>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>直播型 (Live)</p>

- 在网络同步直播于预定时间举行的说明会。
- 能即时参加，并在直播期间利用聊天室功能向企业提问。

<br>

<div class="custom-title">公司说明会的准备</div>

1. 事前预习要参加的企业信息
2. 准备稳定的通信环境（如 Wi-Fi）
3. 积极提问，解决在产业研究中未弄清楚的疑问
4. 绝对不可迟到

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">募集要项（招聘简章）</div>

募集要项是企业在进行招聘时，所提示的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">招聘岗位内容、工作条件、待遇</span>等必要信息的总结。

<div class='flex flex-col justify-center items-center '>

<p class='text-dark-white font-bold underline decoration-red-800  decoration-4 underline-offset-8'>募集要项内容范例</p>

| 项目             | 内容                                    |
| ---------------- | --------------------------------------- |
| 雇用形态         | 正式员工、契约员工、派遣员工等工作类型  |
| 职种・工作内容   | 招聘的职称及入职后的工作细节            |
| 所在地・工作地点 | 工作场所或总部位置                      |
| 试用期间         | 正式录取前的考核期                      |
| 给与（薪资）     | 基本工资、加班费等薪水信息              |
| 手当（津贴）     | 交通补贴、住房补贴等                    |
| 升给（调薪）     | 薪资上调的制度                          |
| 赏与（奖金）     | 分红或年终奖金                          |
| 休日・休暇       | 休假日、年假（有薪假）、育儿假等        |
| 福利厚生         | 社会保险、公积金/退休金等公司的支援制度 |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">制作提交文件</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学四年级春季</span>

<div class="flex justify-center">
  <img src='../../../images/cv.png' class='rounded-2xl'>
</div>

<br>

`Entry Sheet (ES)` 与 `个人简历` 等应聘文件，是企业用来掌握应聘者特质与个人信息的文件。

<br>

<div class="custom-title">Entry Sheet (ES) 与个人简历的差异</div>

|          | Entry Sheet (ES)                                     | 个人简历                          |
| -------- | ---------------------------------------------------- | --------------------------------- |
| 目的     | 判断选考`合格与否`                                   | 身份`证明`、`事实确认`            |
|          | 确认`匹配度` (Matching)                              | 注册至`人事数据库`用              |
| 填写项目 | 住所、姓名等`基本信息`                               | 住所、姓名等`基本信息`            |
|          | `自我推荐` (Self PR)、应聘动机、大学事迹 (Gakuchika) | `自我推荐`与`应聘动机`            |
|          | 回答企业独自设计的`问题`                             |                                   |
| 形式     | 企业`自定题目`较多                                   | `定型化`（如 JIS 规格等通用格式） |

<br>

个人简历可以说是职场上的“身份证”。企业首先确认的并非你本身的能力，而是“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">信息的准确性</span>”与“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">诚实度</span>”。
<br>

相对地，Entry Sheet (ES) 是一份传达“如何发挥我的个人能力”的“提案书”。仅凭简历无法看出的“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">思考过程</span>”与“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">热情的温度</span>”将会是评估重点。

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">笔试 (笔记试验)</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学四年级春季</span>

笔试是求职过程中用来测量应聘者<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">基础学力（语文・数理逻辑）、一般常识、逻辑思考能力、专业知识、性格（适性）</span>等的考试。

近年来，为了提高学生便利性与评估的客观性，“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">线上受测</span>”与“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">适性检查（性格诊断+能力检查）</span>”已成为主流。这些测试可在<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">家中</span>或<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">测试中心 (Test Center)</span> 等多种地点进行。此外，部分企业也会在公司或<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">指定场所</span>使用纸质试卷进行适性检查。
<br>

<div class="custom-title">笔试的种类</div>

| 考试类别     | 内容                                               | 目的                                 |
| ------------ | -------------------------------------------------- | ------------------------------------ |
| 适性检查     | `语文`（日文）、`数理`逻辑（数学・推理）、`英语`等 | 确认思考速度、准确度与逻辑思考力     |
|              | 有关`价值观`与`行动模式`的问题                     | 确认对社风的适应性、抗压性、职务适性 |
| 一般常识     | `时事`问题、`社会`、`理化`、`国文`、`商务礼仪`     | 确认作为社会人士的基本知识与关注广度 |
| 小论文・作文 | 针对给定的`主题`撰写自己的见解                     | 考察文章构成力、逻辑力与价值观深度   |

<br>

<div class="custom-title">代表性的外部测试</div>

| 检查名称                                                          | 特征                                                 | 问题构成                   |
| ----------------------------------------------------------------- | ---------------------------------------------------- | -------------------------- |
| [SPI3](https://www.spi.recruit.co.jp)                             | 日本`普及率最高`。平衡测量性格与能力（语文・数理）   | 语文・数理・性格检查       |
| [玉手箱 Ⅲ](https://www.shl.co.jp/service/assessment/tamatebako3/) | 每题限制`时间极短`。重视速度与准确性。               | 　计算・语文・英语・OPQ    |
| [TG-WEB](https://tg-web.humanage.co.jp)                           | 与其他测试相比`难度较高`，考验语文与数理的逻辑思考。 | 语文・计数・英语・性格检查 |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">面试</div>

<span class="bg-blue-500 w-48 px-3 py-1 rounded-2xl text-4 text-sm text-white">活动时期：大学四年级春、夏季</span>

面试是企业与求职者深化相互理解、确认技能、热情与适性的场合。

在日本，面试通常会进行<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">两次以上</span>。这是为了让不同阶层的人观察应聘者的性格与思维，判断其是否为适合该企业的人才。

此外，现在“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">线上与线下并用</span>”已成为常态，面试已从单纯的“问答”进化为“企业与学生互相确认匹配度的对话”。

<br>

<div class="custom-title">面试的种类</div>

| 种类          | 特征                         | 优点                                                 |
| ------------- | ---------------------------- | ---------------------------------------------------- |
| 个人面试      | 面试官 1〜3 名对`学生 1 名`  | 最常见。能深度挖掘个人特质，确认真心话与价值观       |
| 集体面试      | 面试官 1〜3 名对`学生多名`   | 能一次评估多名学生。可看出与他人的比较及简要表达能力 |
| 小组讨论 (GD) | 针对特定主题，`学生彼此讨论` | 可看见领导力、协调性、逻辑力等“在群体中的行为”       |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">奨学金</div>

#### <div class="custom-title">奨学金の種類</div>

支給機関によって、日本の留学生向け奨学金は主に次の <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">4 種類</span>に分けられます。

| 奨学金の種類           | 概要                                                                             |
| ---------------------- | -------------------------------------------------------------------------------- |
| 日本政府（MEXT）奨学金 | `文部科学省`が支給。授業料免除、毎月の生活費、往復航空券付き。最も支給額が大きい |
| JASSO 奨学金           | `私費外国人留学生向け`。毎月一定額の生活費補助                                   |
| 地方自治体・民間奨学金 | `自治体`・`企業`・`財団`が実施。専攻分野や出身国が限定される場合あり             |
| 大学独自の奨学金・減免 | `入学金免除`、授業料 `30 ～ 100％減免`、学内限定奨学金など                       |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請時期</p>

奨学金の申請時期は一律ではなく、主に次の <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 つ</span>の段階に分かれます。

| 区分         | 奨学金の種類                 | 内容・特徴                                            |
| ------------ | ---------------------------- | ----------------------------------------------------- |
| 渡日前に申請 | MEXT 奨学金                  | 渡日の約 `1 年前から準備`。大使館推薦・大学推薦で申請 |
|              | JASSO 予約制度               | `EJU 成績優秀者`は渡日前に学習奨励費の予約が可能      |
|              | 民間奨学金（一部）           | `入国前に書類審査のみ`で申請できる場合あり            |
| 入国後に申請 | 民間奨学金・地方自治体奨学金 | 入学後、`在籍大学`を通じて申請するケースが多い        |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>選考方法</p>

奨学金の選考は成績だけで決まるわけではなく、一般的に次のような<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">審査</span>が行われます。

| 選考方法 | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 書類審査 | `成績`、`経済状況`、`研究計画書`・`志望理由書`などを総合評価 |
| 筆記試験 | 一部奨学金（例：MEXT）で`基礎学力`・`日本語試験`を実施       |
| 面接     | `コミュニケーション力`、`学習意欲`、`将来目標`を重視         |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請時の重要なポイント</p>

| 注意点         | 内容                                                  |
| -------------- | ----------------------------------------------------- |
| 情報収集の時期 | 募集期間が`短く`定員も`少ない`ため、早めの確認が重要  |
| 日本語能力     | 多くの民間奨学金で `N2` 以上の日本語力が求められる    |
| 併給不可       | MEXT など高額奨学金は他奨学金との`同時受給不可`が多い |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本政府奨学金</div>

日本政府（文部科学省）奨学金（通称 MEXT 奨学金）は、1954 年に設立され、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">優秀な外国人学生</span>を日本に受け入れることで、日本と海外諸国との友好関係を深めることを目的としています。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>選支給内容</p>

MEXT 奨学金は全額給付型の奨学金で、主に次の <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">3 つ</span>の支援が含まれます。

| 支援内容 | 詳細                                           |
| -------- | ---------------------------------------------- |
| 学費     | 入学試験料・入学金・授業料が`全額免除`         |
| 生活費   | 課程に応じて月額約 `117,000 ～ 145,000` 円支給 |
| 航空券   | 渡日・帰国時の国際`往復航空券を支給 `          |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請ルート</p>

MEXT 奨学金の主な公式申請ルートは、次の <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 つ</span>です。

| 推薦方式     | 選考機関                           | 募集・申請時期（目安）           | 主な選考内容                       | 特徴                             |
| ------------ | ---------------------------------- | -------------------------------- | ---------------------------------- | -------------------------------- |
| `大使館推薦` | 日本の在外公館（大使館・総領事館） | 3 ～ 5 月                        | 書類審査・筆記試験・面接           | 募集人数が多く、一般的な応募方法 |
| `大学推薦`   | 日本の大学 → 文部科学省            | 12 ～翌 1 月（大学により異なる） | 書類審査・面接（試験なしの場合も） | 大学間交流協定による推薦が中心   |

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>重要な注意事項</p>

| 注意点       | 内容                                                  |
| ------------ | ----------------------------------------------------- |
| `日本語学習` | 多くの区分で、進学前に約 6 か月の予備日本語教育が必要 |
| `併給不可`   | MEXT 受給者は JASSO・民間奨学金との同時受給不可       |
| `毎年の審査` | 成績・出席状況により、奨学金が停止される場合あり      |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本学生支援機構（JASSO）奨学金</div>

MEXT 奨学金（政府直接給付）と比べると、JASSO 奨学金は対象者の<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">範囲が広く</span>、日本の私費留学生が最も多く利用している、実用性の高い奨学金制度です。

JASSO が提供する奨学金は主に私費留学生向けの<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">給付型奨学金</span>（返済不要）であり、大きく次の 2 種類に分かれます。

日本留学試験（EJU）で優秀な成績を収めた学生は、日本に来る前にこの奨学金を「<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">予約</span>」できます。EJU 申込時に申請チェックを入れるのを忘れないでください。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>文部科学省外国人留学生学習奨励費</p>

この奨学金は、学業成績が優秀で、かつ経済的に支援が必要な私費留学生を対象としています。

| 項目       | 内容                                                                                                                         |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------- |
| 対象者     | `大学院・大学`（学部）・`短期大学`・`高等専門学校`（3 年次以上）・`専修学校`・`準備教育課程`・`日本語教育機関`に在籍する学生 |
| 支給額     | 月額 `48,000 円`（日本語教育機関の学生も同額）                                                                               |
| 支給期間   | 原則 `12 か月`                                                                                                               |
| 学業条件   | `学業成績が優秀`で、GPA が所定基準を満たしていること                                                                         |
| 経済条件 ① | 校外からの仕送り月平均額が `120,000 円以下`（学費・入学金を除く）                                                            |
| 経済条件 ② | 日本国内に扶養者がいる場合、その年収が`500 万円未満  `                                                                       |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>海外留学支援制度（協定受入）奨学金</p>

この奨学金は、大学間交流協定に基づく交換留学生を対象とした制度です。

| 項目     | 内容                                                                              |
| -------- | --------------------------------------------------------------------------------- |
| 対象者   | 日本の教育機関と海外大学の交流協定により、`8 日以上 1 年以内`の短期留学を行う学生 |
| 支給額   | 月額 `80,000 円 `                                                                 |
| 申請方法 | 学生が`直接申請不可`。母国の所属大学を通じて、日本の受入校へ申請                  |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申請に関する基本条件</p>

| 応募条件・注意点 | 内容                                              |
| ---------------- | ------------------------------------------------- |
| 在留資格         | 「`留学`」の在留資格を有していることが必須        |
| 併給不可         | MEXT・自国政府奨学金との`同時受給は不可`          |
| 成績基準         | 前年度成績が JASSO 基準で `2.30 以上`（満点 3.0） |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">その他の奨学金</div>

日本には、日本政府（MEXT）奨学金や JASSO 奨学金のほかにも、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地方自治体、民間団体、大学</span>が独自に設けている奨学金が多数あります。

これらの奨学金は種類が多く、<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地域性・専門性・学校限定</span>といった条件があるのが特徴です。

<br>

#### <div class="custom-title">地方自治体・国際交流協会の奨学金</div>

| 項目     | 内容                                                     |
| -------- | -------------------------------------------------------- |
| 対象者   | `当該地域`に居住、またはその地域の大学に在学している学生 |
| 実施団体 | 都道府県・市町村などの`地方自治体`、`国際交流協会`       |
| 特徴 ①   | 明確な`地域制限`あり（例：東京都在住者限定など）         |
| 特徴 ②   | `現金給付`のほか、`住宅補助`・`生活用品支給`など多様     |

<br>

#### <div class="custom-title">民間団体奨学金（Private Foundation Scholarships）</div>

| 項目   | 内容                                                               |
| ------ | ------------------------------------------------------------------ |
| 対象者 | `民間企業`・`奨学財団`が設立。企業の関心分野や創設者の理念に基づく |
| 特徴 ① | `専門分野特化型`が多い（理工系・芸術・特定経済分野など）           |
| 特徴 ② | 国籍条件がある場合あり（中国・東南アジア出身者対象など）           |
| 特徴 ③ | 交流会・セミナー参加が求められ、人脈形成につながる                 |

<br>

#### <div class="custom-title">大学独自の奨学金・学費減免制度</div>

| 項目         | 内容                                                                |
| ------------ | ------------------------------------------------------------------- |
| 対象者       | `当該大学`に正式在籍する成績優秀な留学生                            |
| 学費減免制度 | 授業料の `30％・50％`、場合により `100％免除`（特に私立大学で多い） |
| 学内奨学金   | 大学基金による`月額生活補助`、または一時金支給                      |
| 入学奨励金   | `入試成績優秀者`に対して支給される奨励金                            |

<br>

#### <div class="custom-title">支給額の目安</div>

| 奨学金の種類               | 支給内容・注意点                                             |
| -------------------------- | ------------------------------------------------------------ |
| 地方自治体・民間団体奨学金 | 月額 約 `30,000 円～ 200,000 円`                             |
| 大学独自の支援             | `金額`・`内容`に大きな差あり。志望校の公式サイトで確認が必要 |

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">奖学金</div>

#### <div class="custom-title">奖学金的种类</div>

根据发放机构的不同，针对日本留学生的奖学金主要可分为以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">4 种类型</span>。

| 奖学金种类             | 概要                                                               |
| :--------------------- | :----------------------------------------------------------------- |
| 日本政府 (MEXT) 奖学金 | 由 `文部科学省` 发放。包含学费全免、每月生活费、来回机票。额度最高 |
| JASSO 奖学金           | 针对 `自费外国留学生`。提供每月固定额度的生活费补助                |
| 地方自治体・民间奖学金 | 由 `地方政府`、`企业`、`财团` 实施。可能限定专业领域或出身国家     |
| 大学独自奖学金・减免   | 包含 `入学金免除`、学费 `30 ～ 100％ 减免`、校内限定奖学金等       |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申请时期</p>

奖学金的申请时期各异，主要分为以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 个</span>阶段。

| 区分         | 奖学金种类             | 内容・特征                                            |
| :----------- | :--------------------- | :---------------------------------------------------- |
| 赴日以前申请 | MEXT 奖学金            | 须在赴日前约 `1 年开始准备`。透过大使馆或大学推荐申请 |
|              | JASSO 预约制度         | `EJU 成绩优秀者` 可在赴日前预约学习奖励费             |
|              | 民间奖学金 (部分)      | 有些可透过 `入境前书面审查` 即可申请                  |
| 入境以后申请 | 民间・地方自治体奖学金 | 多数在入学后，透过 `就读大学` 进行申请                |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>选拔方法</p>

奖学金的选拔不单取决于成绩，通常会进行以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">审查</span>。

| 选拔方法 | 内容                                                       |
| :------- | :--------------------------------------------------------- |
| 书面审查 | 综合评估 `成绩`、`经济状况`、`研究计划书`、`志愿理由书` 等 |
| 笔试     | 部分奖学金 (如 MEXT) 会进行 `基础学力` 或 `日语测验`       |
| 面试     | 重视 `沟通能力`、`学习意欲`、`未来目标`                    |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申请时的重要关键</p>

| 注意事项     | 内容                                               |
| :----------- | :------------------------------------------------- |
| 信息搜集时期 | 招募期间 `短` 且名额 `少`，尽早确认至关重要        |
| 日语能力     | 许多民间奖学金要求 `N2` 以上的日语能力             |
| 不可兼领     | MEXT 等高额奖学金多规定 `不可与其他奖学金同时领取` |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本政府奖学金</div>

日本政府（文部科学省）奖学金（通称 MEXT 奖学金）设立于 1954 年，旨在通过接收<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">优秀的外籍学生</span>，深化日本与海外各国的友好关系。

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>支給内容</p>

MEXT 奖学金为全额给付型，主要包含以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">3 项</span>支援。

| 支援内容 | 详细                                       |
| :------- | :----------------------------------------- |
| 学费     | 入学考试费、入学金、学费 `全额免除`        |
| 生活费   | 依课程每月发放约 `117,000 ～ 145,000` 日元 |
| 机票     | 补助赴日及归国时的国际 `往复机票`          |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申请渠道</p>

MEXT 奖学金的主要官方申请渠道有以下 <span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">2 个</span>。

| 推荐方式     | 选拔机构                | 招募・申请时期（基准） | 主要选拔内容            | 特征                       |
| :----------- | :---------------------- | :--------------------- | :---------------------- | :------------------------- |
| `大使馆推荐` | 日本驻外公馆 (大使馆等) | 3 ～ 5 月              | 书面审查・笔试・面试    | 名额较多，最常见的申请方式 |
| `大学推荐`   | 日本大学 → 文部科学省   | 12 ～ 隔年 1 月        | 书面审查・面试 (或免试) | 以大学间交流协定推荐为主   |

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>重要注意事项</p>

| 注意事项   | 内容                                                  |
| :--------- | :---------------------------------------------------- |
| `日语学习` | 许多类别规定在正式入学前需接受约 6 个月的预备日语教育 |
| `不可兼领` | 领取 MEXT 者不可同时领取 JASSO 或其他民间奖学金       |
| `每年审查` | 若成绩或出席状况不佳，奖学金可能会被 `停止发放`       |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">日本学生支援机构（JASSO）奖学金</div>

与 MEXT 奖学金相比，JASSO 奖学金的受众<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">范围更广</span>，是日本自费留学生利用率最高、最实用的奖学金制度。

JASSO 提供的奖学金主要是针对自费留学生的<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">给付型奖学金</span>（不需偿还），主要分为以下两类。

在日本留学试验（EJU）中取得优秀成绩的学生，可以在来日本之前“<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">预约</span>”此奖学金。请务必在报名 EJU 时勾选申请选项。

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>文部科学省外国留学生学习奖励费</p>

此奖学金针对学业优秀且需要经济援助的自费留学生。

| 项目       | 内容                                                                                                 |
| :--------- | :--------------------------------------------------------------------------------------------------- |
| 对象者     | 就读于 `研究所・大学`・`短期大学`・`高等专门学校`・`专修学校`・`预备教育课程`・`日语教育机构` 的学生 |
| 支給额     | 每月 `48,000 日元` (日语教育机构学生同额)                                                            |
| 支給期间   | 原则上为 `12 个月`                                                                                   |
| 学业条件   | `学业成绩优秀`，GPA 达到规定基准                                                                     |
| 经济条件 ① | 国外汇款每月平均额在 `120,000 日元以下` (不含学费入学金)                                             |
| 经济条件 ② | 在日扶养人（若有）年收入须 `低于 500 万日元`                                                         |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>海外留学支援制度（协定受入）奖学金</p>

此奖学金对象为基于大学间交流协定的交换留学生。

| 项目     | 内容                                                    |
| :------- | :------------------------------------------------------ |
| 对象者   | 基于交流协定，进行 `8 天以上 1 年以内` 短期留学的学生   |
| 支給额   | 每月 `80,000 日元`                                      |
| 申请方式 | 学生 `不可直接申请`。须透过母校向日本的接收大学提出申请 |

<br>

<p class='text-dark-white font-bold underline decoration-red-800 decoration-4 underline-offset-8'>申请相关基本条件</p>

| 应募条件・注意点 | 内容                                                  |
| :--------------- | :---------------------------------------------------- |
| 在留资格         | 必须持有“`留学`”在留资格                              |
| 不可兼领         | `不可同时领取` MEXT 或母国政府奖学金                  |
| 成绩基准         | 前一年度成绩依 JASSO 换算须达 `2.30 以上`（满分 3.0） |

<br>

### <div class="relative font-bold text-2xl pb-3 after:absolute after:left-0 after:bottom-0 after:w-full after:border-b-4 after:border-dotted after:border-blue-600">其他奖学金</div>

除了政府与 JASSO 之外，还有许多由<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地方自治体、民间团体、大学</span>独立设置的奖学金。

这些奖学金种类繁多，特征在于通常带有<span class="font-bold bg-linear-to-t from-yellow-400/70 from-10% to-transparent to-50% dark:from-green-400/70">地域性、专业性、学校限制</span>等条件。

<br>

#### <div class="custom-title">地方自治体・国际交流协会奖学金</div>

| 项目     | 内容                                                          |
| :------- | :------------------------------------------------------------ |
| 对象者   | 居住于 `该地区` 或就读于该地区大学的学生                      |
| 实施团体 | 都道府县・市町村等 `地方政府`、`国际交流协会`                 |
| 特征 ①   | 有明确的 `地域限制`（例如：仅限东京都居住者）                 |
| 特征 ②   | 除了 `现金给付`，也有 `住宅补助` 或 `生活用品发放` 等多元方式 |

<br>

#### <div class="custom-title">民间团体奖学金（Private Foundation Scholarships）</div>

| 项目   | 内容                                                       |
| :----- | :--------------------------------------------------------- |
| 对象者 | 由 `民间企业` 或 `奖学财团` 设立，基于企业兴趣或创始人理念 |
| 特征 ① | 多为 `专业领域特定型`（如理工、艺术、特定经济领域等）      |
| 特征 ② | 可能有`国籍限制`（如仅限中国、东南亚出身者）               |
| 特征 ③ | 通常要`求参加交流会或研讨会`，有助于建立人脉               |

<br>

#### <div class="custom-title">大学独自奖学金・学费减免制度</div>

| 项目         | 内容                                                      |
| :----------- | :-------------------------------------------------------- |
| 对象者       | 在 `该大学` 正式就读且成绩优秀的外籍留学生                |
| 学费减免制度 | 学费 `30％、50％` 甚至 `100％ 免除`（在私立大学尤为常见） |
| 校内奖学金   | 来自大学基金的 `每月生活补助` 或一次性给付                |
| 入学奖励金   | 针对 `入学考试成绩优秀者` 发放的奖励金                    |

<br>

#### <div class="custom-title">发放额度基准</div>

| 奖学金种类                 | 支給内容・注意点                                |
| :------------------------- | :---------------------------------------------- |
| 地方自治体・民间团体奖学金 | 每月约 `30,000 日元 ～ 200,000 日元`            |
| 大学独自支援               | `金额` 与 `内容` 差异极大。须在志愿学校官网确认 |
